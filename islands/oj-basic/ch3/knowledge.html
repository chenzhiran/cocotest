二、布尔类型：只有“对”与“错”的世界 (bool)

在 C++ 中，有一种特殊的箱子，里面只能装两个值：真 (true) 或者 假 (false)。

这种类型叫做 布尔类型 (bool)。

🧙‍♂️ 魔法小贴士：
在计算机内部：
  • true (真) 通常用数字 1 表示。
  • false (假) 通常用数字 0 表示。
  • 其实，任何非 0 的数都代表真，只有 0 代表假。
bool isHappy = true;   // 定义一个布尔变量，值为真
bool isSad = false;    // 定义一个布尔变量，值为假

cout << isHappy << endl; // 输出 1
cout << isSad << endl;   // 输出 0

三、单分支结构：如果满足条件就行动 (if)

生活中我们经常说：“如果下雨，我就带伞”。这就是一种选择。

if ( 条件 ) { 语句... } 👉 如果条件成立（为真），就执行大括号里的语句。
🌧️ 情景模拟

如果分数大于 90 分，妈妈奖励一个冰淇淋。

int score;
cin >> score;

// 如果分数大于90
if (score > 90) {
    cout << "奖励冰淇淋！" << endl;
}
// 如果分数不大于90，什么都不发生，直接往下走

四、双分支结构：非黑即白的两难选择 (if-else)

如果条件不满足，我们也想做点别的事情，怎么办？比如：“如果下雨带伞，否则（没下雨）戴太阳帽”。

if ( 条件 ) { 语句1 } 👉 条件成立时，做这件事
else { 语句2 } 👉 否则（条件不成立），做这件事 
if (score >= 60) {
    cout << "及格啦！" << endl;
} else {
    cout << "不及格，要加油！" << endl;
}

五、应用案例：判断奇数与偶数

这是编程中最经典的判断题！

🧠 逻辑分析：
  • 偶数：能被 2 整除的数（除以 2 余数为 0）。例如：2, 4, 100。
  • 奇数：不能被 2 整除的数（除以 2 余数为 1）。例如：1, 3, 99。
  • 核心算式：num % 2 == 0
int num;
cin >> num;

// 如果除以2的余数是0
if (num % 2 == 0) {
    cout << "Even (偶数)" << endl;
} else {
    cout << "Odd (奇数)" << endl;
}
