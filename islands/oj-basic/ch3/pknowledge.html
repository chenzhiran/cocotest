第三章：选择结构——if 单双分支

定位：教会孩子“让程序做决定”，把数学判断题 变成 程序判断题。

一、课堂导入：地牢岔路口 & 会思考的程序

示例：

“前两章我们教电脑会‘算’、会‘比较大小’，但电脑现在还有点笨——它不会自己做决定。”

“想象一下：英雄走到地牢岔路口，左边写着‘有宝箱’，右边写着‘有怪物’。
如果血量 > 50，就可以去打怪；
如果血量 ≤ 50，就先去拿宝箱补给。
这时候，就需要程序来帮英雄做选择了！”

“今天我们要学的关键角色叫 if 语句，它就像一个路牌：如果……就……。有时候还会配一个弟弟 else：要么这样，要么那样。”

板书关键词：

  • if：如果……
  • else：否则……
  • 条件：会得到 True / False 的表达式
  • 缩进：谁是 if 管的孩子，就往里缩一点

二、if 语句基础：单分支——“如果…就执行”

1. 单分支结构：只有一个“如果”

例子 1：温度计简单版——大于 0 就提示“可以前行”

# 读入当前温度 temp = int(input()) # 如果温度大于 0，就提示可以前行 if temp > 0: print("Yes")

逐行讲解：

  • temp = int(input())：从键盘读入一个整数。
  • if temp > 0:：if 就是“如果”。temp > 0 是条件。结尾的冒号 : 非常重要，表示“接下来是 if 控制的代码块”。
  • print("Yes")：前面有缩进（一般 4 个空格）。表示：只有在 temp > 0 时，才会执行这一行。

2. 缩进的重要性：只要缩进内才算 if 的“孩子”

对比例子：

temp = int(input()) if temp > 0: print("Yes") print("End")

当输入 10 时，输出：Yes End
当输入 -5 时，输出：End

解释：print("End") 没缩进 → 不受 if 控制，总是会执行。

三、if-else 双分支：要么这样，要么那样

1. 标准形式

if 条件: # 条件为 True 时执行这里 ... else: # 条件为 False 时执行这里 ...

类比生活：“如果考试 ≥ 60 分，就‘及格’，否则就是‘不及格’。”

2. 例子：判断成绩是否及格

score = int(input()) # 读入一个整数，表示成绩 if score >= 60: print("Pass") # 成绩大于等于 60，输出 Pass else: print("Fail") # 否则输出 Fail

四、条件表达式写法：把数学条件翻译成代码

这一部分重点：把我们数学上的条件：奇偶、整除、大于、小于，翻译成 a % 2 == 0、x > 0 等表达式。

1. 奇偶判断：用 % 2

n = int(input()) if n % 2 == 0: # n 能被 2 整除吗？ print("even") else: print("odd")

2. 整除判断：用 % b == 0

例子 4：判断 b 是否是 a 的约数

a, b = map(int, input().split()) if a % b == 0: print("Yes") else: print("No")

五、程序调试初步：打印中间结果查逻辑

很多学生：“明明逻辑是对的呀，怎么总是 WA？”
我们让他们养成一个好习惯：——在关键地方加 print，把中间结果打印出来看看。

year = int(input()) # 调试输出：让电脑把每个小条件都说出来 print("year % 4 == 0:", year % 4 == 0) print("year % 100 != 0:", year % 100 != 0) if (year % 4 == 0 and year % 100 != 0) or year % 400 == 0: print("Yes") else: print("No")

六、典型竞赛题型与 if 的应用

1. 两数中较大值

a, b = map(int, input().split()) if a > b: print(a) else: print(b)

2. 闰年判断（规则 + 代码）

数学规则：能被 4 整除但不能被 100 整除；或者能被 400 整除。

year = int(input()) if (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0): print("Yes") else: print("No")

七、易错点提醒与纠正方法 ⚠️

  • = 和 == 混淆：
    错误：if x = 0:
    正确：if x == 0: （一个等号是赋值，两个是比较）
  • 忘记冒号 :
    if 后面要带一个小尾巴——冒号。
  • 缩进错误：
    建议统一用 4 个空格。
  • 条件写反：
    想要“非正数”，却写成 x > 0。
  • 输出大小写：
    OJ 是“严格老师”，Yes 和 YES 不一样。

八、课堂小结：本节知识清单 ✔

  • if 单分支：条件为 True 才执行缩进内的代码。
  • if-else 双分支：一定会走一条路。
  • 条件表达式：比较 (>, ==) 和 逻辑 (and, or)。
  • 缩进 & 冒号：冒号标记头，缩进表示“被 if 管”。
  • 程序调试：用 print 查逻辑。

九、课后练习题讲解（Python 视角）

1. CSP-J1301 智能温度计

题意：t 是正数输出 "Yes"，否则 "No"。

t = int(input()) if t > 0: print("Yes") else: print("No")
2. CSP-J1302 偶数座位

题意：n 是偶数输出 "Yes"，否则 "No"。

n = int(input()) if n % 2 == 0: print("Yes") else: print("No")
3. CSP-J1303 约数座位

题意：判断 b 是否是 a 的约数（即 a 能被 b 整除）。

a, b = map(int, input().split()) if a % b == 0: print("Yes") else: print("No")
5. CSP-J1305 萤火之光

题意：如果数字末尾是 0 输出 "Yes"。核心：n % 10 == 0。

n = int(input()) if n % 10 == 0: print("Yes") else: print("No")
