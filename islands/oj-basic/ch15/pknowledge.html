一、什么是字符串：文字的魔法箱

在 Python 中，任何被单引号 ' ' 或 双引号 " " 包裹起来的内容，都是字符串。

name = "Alice"
message = 'Hello World'
number_str = "666" # 这不是数字，是文字哦！

二、输入输出：不挑食的 Input

在 C++ 中，输入带空格的句子很麻烦。但在 Python 中，input() 函数非常强大，它是个“大胃王”，会把整行内容（包括空格）一口吃掉！

🍽️ Python 的胃口

输入：Hello World

  • C++ (cin): 只吃到 "Hello" ❌
  • Python (input): 吃到 "Hello World" ✅
print("请输入一句口号：")
s = input()  # 就算有空格也能全部读进来
print("你输入的是：", s)

四、基础技能：长度与下标

字符串就像一串珍珠项链，我们可以数它有多少颗，也可以拿出其中的某一颗。

s = "Python"
print(len(s))  # 长度：6
print(s[0])    # 第1个字符：P
print(s[-1])   # 倒数第1个：n (Python特技！)

🚨 易错点：
字符串是不可变的！你不能写 s[0] = 'J' 来修改它。想修改必须创建一个新字符串。

五、切片忍者：提取关键信息 (Slicing)

如果你想提取字符串的一段（比如 "Hello World" 中的 "World"），可以使用 Python 的切片功能，就像忍者挥刀一样精准！

⚔️ 忍术口诀： s[开始 : 结束]
注意：包含开始，不包含结束（包头不包尾）。
s = "0123456789"
s[2:5] 👉 "234" (下标2,3,4)
s[:3]  👉 "012" (从头开始)
s[5:]  👉 "56789" (一直到尾) 
date = "2023-10-01"
year = date[0:4]  # 2023
month = date[5:7] # 10
day = date[8:]    # 01
print(year, month, day)
