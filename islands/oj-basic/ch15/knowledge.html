二、字符数组：存储文字的容器 (C风格)

在早期的 C 语言中，我们是用 char 类型的数组来存文字的。

但是！ 电脑怎么知道这句话在哪里结束呢？它需要一个“红灯”。

🧙‍♂️ 老师划重点：
这个 \0 (空字符) 非常重要！它告诉电脑“字符串到这里结束了”。
所以，存 "Hello" (5个字母)，至少需要申请 6 个格子的数组！ 
char s[10] = "Hello";
// 实际上存的是：'H','e','l','l','o','\0',...

三、String类：C++给予的现代化馈赠

字符数组太麻烦了？容易越界？忘了写 \0？
别担心，C++ 发明了超级好用的 string 类型！它就像一个可以自动伸缩的魔法盒子。

🚀 开启魔法：
1. 必须引入头文件：#include <string>
2. 像用 int 一样用它！ 
#include <string>
using namespace std;

string name = "Kouda";
string hobby;
cin >> hobby;

四、输入输出：遇到空格怎么办？(cin vs getline)

这是新手最容易掉进去的坑！如果你输入 "Hello World"，cin 只能读到 "Hello"。

cin >> s
Hello World
❌ 遇到空格就停止 (挑食)
getline
Hello World
✅ 读取整行 (不挑食)
string s;
// 如果输入包含空格，请用这个：
getline(cin, s);

五、基本操作：连接、比较与长度计算

String 类型让这些操作变得像搭积木一样简单。

1. 字符串连接 (+)

想把 "Super" 和 "Man" 拼起来？直接用加号！

string a = "Super";
string b = "Man";
string c = a + b; // c 就是 "SuperMan"

2. 获取长度 (.length() 或 .size())

cout << a.length(); // 输出 5
cout << a.size();   // 也是输出 5，完全一样

3. 比较大小 (==, <, >)

字符串比较是按照字典序（Dictionary Order）。就像查字典一样，a 在 b 前面，所以 "apple" < "banana"。

if (s1 == s2) cout << "一样大";
if (s1 < s2) cout << "s1 排在 s2 前面";
