一、复习回顾：While循环的执行流程

上节课我们学习了 while 循环，它像一个严格的门卫。

[Image of while loop flow chart]

特点： 先判断，后执行。如果一开始条件就是假的，那么一次都不执行。

二、模拟 Do-While：先斩后奏的逻辑

生活中有时候我们需要重复做某件事，比如：“先玩一局游戏（至少玩一次），觉得好玩再继续玩”。

虽然 Python 没有 do...while 语句，但我们可以用 while True 加上 break 来完美实现！

🧙‍♂️ 老师小贴士：
Python 的设计理念是“只有一种最好的方法”。它认为 while True 这种写法更灵活、更强大，所以就没有专门设计 do-while 语法。我们叫它“无限循环+紧急出口”模式。 
while True:
    print("先跑一圈再说！")
    
    # 检查是否要停止
    if 跑不动了:
        break  # 🛑 紧急刹车，跳出循环

三、区别对比：两种循环模式

它们长得很像，但性格完全不同！

特性 普通 While 模拟 Do-While (While True)
绰号 严格的门卫 热情的推销员
执行顺序 先判断，后执行 先执行，后判断
最少执行次数 0 次 (一开始条件就不满足) 1 次 (不管条件如何)
🧪 实验验证

假设一开始 i = 10，我们要打印小于 5 的数。

普通 While:

Check: 10 < 5? (假)
结果：一次都不跑！

Do-While 模式:

Run: 打印 10 (先跑一次)
Check: 10 >= 5? (真，break)
结果：跑了1次！

四、魔法刹车：Break & Continue

有时候我们想中途让循环停下来，或者跳过某一圈。这时候就需要两个魔法指令。

🛑 Break (紧急刹车)

作用： 直接跳出整个循环，不玩了！

场景： 在跑步时腿受伤了，立刻停止比赛。

while True:
    print("跑")
    if 受伤:
        break
⏭️ Continue (跳过本轮)

作用： 这一圈剩下的不做了，直接进入下一圈。

场景： 老师点名，喊到请假同学的名字，跳过他，继续喊下一个。

i = 0
while i < 5:
    i += 1
    if i == 3:
        continue
    print(i) # 输出1 2 4 5

五、应用案例：设计一个交互式菜单

while True 最经典的应用场景就是菜单。因为我们总希望先把菜单展示给用户看一次，用户选择退出时才结束。

while True:
    # 1. 先展示菜单
    print("=== 游戏大厅 ===")
    print("1. 开始游戏")
    print("2. 查看战绩")
    print("0. 退出")
    
    # 2. 用户输入
    choice = input("请选择: ")

    # 3. 处理选择
    if choice == '1':
        print("Loading...")
    elif choice == '2':
        print("Rank: SSS")
    elif choice == '0':
        print("游戏结束，拜拜！")
        break # 🛑 只有选0才退出
    else:
        print("输入错误，请重选")
    
    print() # 空一行
