一、复习回顾：列表查找与最值

在之前的课程中，我们学会了如何在列表中找到特定的数（in 或循环），以及如何找到最大的数（max() 或擂台法）。

💡 魔法捷径

Python 提供了超级好用的魔法指令：

  • 查找是否存在：if x in nums:
  • 找最大值：max(nums)
  • 找最小值：min(nums)

二、元素移动：乾坤大挪移 (插入与删除)

有时候我们需要在一个排好序的队伍里，强行插进一个人，或者把一个人拉出来。这会导致后面所有人都要挪动位置！

1. 插入元素 (Insert)

场景： 在排好的队伍中，让新同学插进第 k 个位置。

原理： 就像挤沙发，新同学坐下来之前，坐在 k 位置及后面的人都要往后挪。

# Python 的魔法方法：insert
names = ["Alice", "Bob", "Charlie"]
names.insert(1, "David") # 在下标1的位置插入David
print(names) 
# 结果：['Alice', 'David', 'Bob', 'Charlie']

2. 删除元素 (Delete)

场景： 第 k 个同学离开了。

操作： 后面的同学依次往前补位，把空缺填满。

# Python 的魔法方法：pop 或 remove
names.pop(2)      # 删除下标为2的元素 (Bob)
# 或者
names.remove("Charlie") # 删除第一个叫Charlie的元素

三、桶排序思想：统计数字出现的次数

如果我们要统计全班同学每个人投了多少票（编号 1-10），怎么做最快？

我们可以准备 10 个贴了标签的桶，拿到一张票（比如 3 号），就往 3 号桶里丢一颗石子（计数+1）。

💡 核心思想：
利用列表的下标来代表具体的数字，列表的值代表出现的次数。
counts[x] += 1 => 数字 x 出现了一次。 
# 准备11个桶(下标0-10)，初始化为0
counts = [0] * 11 

votes = [1, 3, 1, 2, 3, 3]
for x in votes:
    counts[x] += 1 # 关键：对应编号的桶，计数+1

print(counts)
# 结果：[0, 2, 1, 3, 0, 0, ...] (1号2次, 2号1次, 3号3次)

四、开关灯问题：模拟状态的批量切换

有 n 盏灯，开始都是关着的。我们要进行多次操作，每次按下某些灯的开关（开变关，关变开）。

我们可以用列表 lights[i] 来表示第 i 盏灯的状态：

  • 0：关灯 ⚫
  • 1：开灯 🟡

0 (Off)

➡️ 按开关 ➡️

1 (On)

# 切换状态的魔法公式
# 方法1：用 if
if lights[i] == 0:
    lights[i] = 1
else:
    lights[i] = 0

# 方法2：数学魔法 (1变0, 0变1)
lights[i] = 1 - lights[i] 

# 方法3：逻辑非 (适合bool类型)
lights[i] = not lights[i]
