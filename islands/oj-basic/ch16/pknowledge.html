一、复习回顾：ASCII码与魔法函数

在 Python 中，虽然字符串很智能，但底层依然是数字在支撑。

'A'
ord('A') ⬇
65
chr(65) ⬆
+ 1 =
66
'B'

Python 特有神器：

  • ord(字符)：获取字符的身份证号（ASCII码）。
  • chr(数字)：根据身份证号找到字符。
  • s.upper() / s.lower()：大小写转换魔法。

二、大小写转换：字母的变身术

在 Python 中，你不需要像 C++ 那样去加减 32，我们有直接的魔法指令！

"Hello"
➡️
.upper()
➡️
"HELLO"
"World"
➡️
.lower()
➡️
"world"
s = "Hello Python"

# 1. 全部变大写
print(s.upper())  # HELLO PYTHON

# 2. 全部变小写
print(s.lower())  # hello python

# 3. 大小写互换
print(s.swapcase()) # hELLO pYTHON
⚠️ 注意： 字符串是不可变的！s.upper() 不会改变 s 本身，而是产生一个新的字符串。

三、回文串判断：正读反读都一样

回文串 (Palindrome)：正着读和反着读完全一样的字符串。例如："level", "12321"。

在 Python 中，我们不需要像 C++ 那样写循环比较，我们有切片魔法！

s = input()

# 魔法切片 [::-1] 意思是：从头到尾，倒着走
if s == s[::-1]:
    print("Yes, it is a palindrome!")
else:
    print("No")

四、单词统计：文章里有多少个词？

给定一句话 "I love coding"，怎么算出有 3 个单词？

C++ 需要一个字符一个字符去数空格。但 Python 有个超级好用的工具：`split()` (切割)。

"I love coding"
⬇️ s.split()
"I"
"love"
"coding"

变成了一个列表，里面有 3 个元素！

s = input() # 输入：Hello world python
words = s.split() # 默认按空格切割

print(words)      # ['Hello', 'world', 'python']
print(len(words)) # 3 (单词数量)

五、简单加密：凯撒密码的实现

凯撒密码：把每个字母按照字母表往后移 k 位。例如后移 1 位：A -> B, Z -> A。

这需要用到 ord() 和 chr() 配合数学计算。

s = "abc"
k = 1
result = ""

for char in s:
    # 1. 拿到 'a' 的编号 97
    base = ord('a') 
    
    # 2. 计算当前字符是第几个 (0-25)
    # ord('a') -> 97, current -> 0
    current = ord(char) - base
    
    # 3. 移位并取模 (转圈圈)
    new_char_code = (current + k) % 26
    
    # 4. 变回字符并拼接到结果
    result += chr(base + new_char_code)

print(result) # bcd
