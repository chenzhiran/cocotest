关键知识点： 字符在电脑里其实是数字。大写字母比小写字母小 32。

String 常用咒语：

  • s.length() 或 s.size()：获取长度
  • s[i]：访问第 i 个字符
  • s1 + s2：连接两个字符串
  • getline(cin, s)：读取带空格的一整行

二、大小写转换：字母的变身术

有时候我们需要把 "Hello" 变成 "hello" 或者 "HELLO"。利用 ASCII 码的特性，这很容易实现。

1. 小写变大写 (To Upper)

原理：小写字母的值比较大，减去 32 就变回大写了。

string s = "Hello";
for (int i = 0; i < s.length(); i++) {
    // 如果是小写字母 (a-z)
    if (s[i] >= 'a' && s[i] <= 'z') {
        s[i] = s[i] - 32; // 变身！
    }
}
cout << s; // 输出 HELLO

2. 大写变小写 (To Lower)

原理：大写字母的值比较小，加上 32 就变成小写了。

if (s[i] >= 'A' && s[i] <= 'Z') {
    s[i] = s[i] + 32;
}

三、回文串判断：正读反读都一样

回文串 (Palindrome)：正着读和反着读完全一样的字符串。例如："level", "12321", "aba"。

算法思路： 双指针法。一个指头（i）指开头，一个指头（j）指结尾，一起往中间走，看是否每次都相等。

string s;
cin >> s;
bool isPalindrome = true;
int len = s.length();

for (int i = 0; i < len / 2; i++) {
    // s[i] 是前面的字符
    // s[len-1-i] 是对应的后面的字符
    if (s[i] != s[len - 1 - i]) {
        isPalindrome = false;
        break; // 发现不一样，肯定不是回文，停止！
    }
}

四、单词统计：文章里有多少个词？

给定一句话，比如 "I love coding"，怎么算出有 3 个单词？

我们不能简单地数空格，因为可能有两个连续的空格。比较好的方法是：数有多少次“从空格变成了非空格”，或者简单点，数有多少个空格，然后加 1（假设首尾没空格且单词间只有一个空格）。

I_love_coding

遇到空格，说明一个单词可能结束了。

string s;
getline(cin, s); // 读入整行
int count = 0;

for (int i = 0; i < s.length(); i++) {
    // 如果当前是空格
    if (s[i] == ' ') {
        count++;
    }
}
cout << count + 1; // 单词数 = 空格数 + 1 (简单版)
注意： 上面是简单版。严谨的题目可能会有连续空格，那时候需要判断 if (s[i] != ' ' && s[i-1] == ' ')。

五、简单加密：凯撒密码的实现

凯撒密码 (Caesar Cipher) 是一种古老的加密方法。它的原理是把每个字母按照字母表往后移几位。

例如：后移 1 位，A -> B, B -> C, ... Z -> A。

💡 难点： 'z' 往后移 1 位是谁？是 '{' 吗？不，应该是 'a'！这需要用到循环（取模）或者判断。 
// 假设只处理小写字母，后移 k 位
for (int i = 0; i < s.length(); i++) {
    if (s[i] >= 'a' && s[i] <= 'z') {
        // 1. 变成 0-25 的数字
        int num = s[i] - 'a';
        // 2. 移位并取模 (转圈圈)
        num = (num + k) % 26;
        // 3. 变回字母
        s[i] = num + 'a';
    }
}
