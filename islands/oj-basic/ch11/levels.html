3303离散数字 (离散化)

思路分析

离散化的目的是把数值很大但数量有限的数据，映射到较小的区间（排名）。
方法：
1. 创建结构体存储数值和原始下标。
2. 按数值从小到大排序。
3. 排序后，第 i 个元素的排名就是 i+1。
4. 将排名存回原始位置数组，最后按顺序输出。

n = int(input()) a = list(map(int, input().split()))
输入
struct Node { int val, id; } a[10005]; int rank[10005];
# 列表存 (值, 原始下标) d = [] for i in range(n): d.append((a[i], i))
记录ID
for(int i=0; i<n; i++) { cin >> a[i].val; a[i].id = i; }
d.sort(key=lambda x: x[0])
按值排序
bool cmp(Node x, Node y){ return x.val < y.val; } sort(a, a+n, cmp);
ans = [0] * n for r, (val, original_idx) in enumerate(d): ans[original_idx] = r + 1
映射排名
for(int i=0; i<n; i++) rank[a[i].id] = i + 1;
print(*ans)
输出
for(int i=0; i<n; i++) cout << rank[i] << " ";
