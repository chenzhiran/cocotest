二、算术运算：计算机的加减乘除与取模

计算机不仅能存数据，最厉害的是它能做运算！C++里的运算顺序和数学一样：从左到右，先乘除模，后加减。

1. 常用运算符

符号 含义 例子 结果
+ 加法 1 + 2 3
- 减法 5 - 1 4
* 乘法 2 * 3 6
/ 除法 3 / 2
3.0 / 2		 1 (整数除法去尾)
1.5 (小数除法)
% 取余(模) 10 % 3 1 (10除以3余1)
⚠️ 老师划重点：整数除法
两个整数相除，结果只保留整数部分！ 
int a;
float b;
a = 3 / 2;    // a 的结果是 1
b = 3.0 / 2;  // b 的结果是 1.5

2. 神奇的自增与自减

这是一个让变量自己变大变小的魔法符号。

  • ++ (自增)：让变量的值 +1
  • -- (自减)：让变量的值 -1
int a = 3;
a++;  // 此时 a 变成了 4

难点辨析：放在前面还是后面？

int a = 3, k;
k = a++;   // 先把3给k，a再变4。结果：k=3, a=4

a = 3;
k = ++a;   // a先变4，再把4给k。结果：k=4, a=4

三、赋值运算：变量之间的魔法交换

1. 普通赋值 (=)

把右边的值，装进左边的箱子里。

int a, b;
a = 1; // 把1放入a箱子
b = 2; // 把2放入b箱子

2. 变量交换 (重点！)

🧴 情景剧：酱油和醋怎么换瓶？

变量 a (酱油) 和 变量 b (醋) 想要互换身体里的东西。直接倒会混在一起，怎么办？

办法： 找一个空瓶子 (变量 c)！

int a = 1, b = 2, c;
c = a;  // 1. 把酱油(a)倒进空瓶(c)
a = b;  // 2. 把醋(b)倒进酱油瓶(a)
b = c;  // 3. 把空瓶里的酱油(c)倒进醋瓶(b)

3. 复合赋值 (偷懒的写法)

程序员很喜欢偷懒，所以发明了简写：

写法 等价于 含义
x += 10 x = x + 10 x自己增加10
x -= 10 x = x - 10 x自己减少10
x *= 10 x = x * 10 x自己乘以10
x /= 10 x = x / 10 x自己除以10
x %= 10 x = x % 10 x自己取余10

四、比较运算：是真还是假？

比较运算的结果只有两种：true (真，输出1) 和 false (假，输出0)。

关系运算符 描述 例子 (5 vs 3) 结果
== 等于 (注意是两个等号！) 5 == 3 false (0)
!= 不等于 5 != 3 true (1)
> 大于 5 > 3 true (1)
< 小于 5 < 3 false (0)
⚠️ 易错点警示
  • 判断相等要用 ==，不能用 = (这是赋值)！
  • 5 > 3 是真的，程序里会输出 1。

五、数学魔法：常用数学库函数 (cmath)

基本的加减乘除不够用怎么办？比如求平方根、绝对值？这时候我们需要召唤 C++ 强大的数学库！

召唤咒语： 在程序最开头写上 #include <cmath>

常用数学函数表

函数名 功能描述 数学符号 代码示例 结果
pow(a, b) 计算 a 的 b 次方 $a^b$ pow(2, 3) 8 (即 2*2*2)
sqrt(x) 计算 x 的平方根 $\sqrt{x}$ sqrt(9) 3
abs(x) 计算 x 的绝对值 (整数) $|x|$ abs(-5) 5
ceil(x) 向上取整 (天花板) $\lceil x \rceil$ ceil(3.1) 4
floor(x) 向下取整 (地板) $\lfloor x \rfloor$ floor(3.9) 3 
#include <iostream>
#include <cmath> // 别忘了引入数学库头文件！
using namespace std;

int main() {
    double a = 3.0, b = 4.0;
    double c;
    
    // 勾股定理：c = sqrt(a*a + b*b)
    c = sqrt(pow(a, 2) + pow(b, 2)); 
    
    cout << "斜边长度: " << c << endl; // 输出 5
    return 0;
}
