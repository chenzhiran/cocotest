二、数据类型初步：数字 VS 字符串

1. 三个小角色：int / float / str

a = 10        # int：整数，像课本里的自然数，如 3, 0, -5
b = 3.14      # float：浮点数，带小数点的数，如 0.5, 2.0
c = "10"      # str：字符串，被引号包围的文字，不能直接计算

2. 类型转换：int() / float() / str()

input() 读进来的默认都是字符串，要做数学题，必须先变身！

x = input()           # 假设输入 5，此时 x 是字符串 "5"
y = int(x)            # 变身！把 "5" 变成整数 5
print(y + 1)          # 现在可以计算了，输出 6

三、算术运算深化：会分糖、会取余、会平方

1. 复习基本四则运算

加(+)、减(-)、乘(*)、除(/) 大家都很熟悉啦，注意 Python 里的除法 / 结果永远是小数哦。

a, b = map(int, input().split())  # 一次输入两个数
print(a + b)   # 加法：求和
print(a - b)   # 减法：求差
print(a * b)   # 乘法：求积
print(a / b)   # 除法：结果是浮点数，例如 4/2=2.0

2. 整除 // 和 取余 % —— 分糖果故事

🍬 类比：分糖果

有 17 颗糖，3 个小朋友，每人分一样多。

  • // 整除： 每人分到几颗？（只取整数部分）
  • % 取余： 最后剩下几颗？（余数）
candies = 17
kids = 3
 
each = candies // kids   # 整除：每人分到 5 颗
left = candies % kids    # 取余：剩下 2 颗

3. 幂运算 **：平方、立方

想要算某数的几次方，用两个乘号 **。

x = int(input())
square = x ** 2   # 计算 x 的 2 次方（平方）
cube = x ** 3     # 计算 x 的 3 次方（立方）

四、比较运算与布尔值：让电脑知道“谁更大”

运算符 含义 例子 (a=10, b=5)
>大于a > b (True)
<小于a < b (False)
>=大于等于a >= 10 (True)
<=小于等于b <= 5 (True)
==相等a == b (False)
!=不等a != b (True)
a, b = map(int, input().split())
print(a > b)    # 比较大小，输出 True(真) 或 False(假)
print(a == b)   # 判断是否相等，注意是两个等号哦！

五、逻辑运算：同时做作业才能玩游戏

and (并且)

都要满足。
“数学写完 并且 语文写完，才能玩。”

or (或者)

满足一个就行。
“吃冰淇淋 或者 吃蛋糕，都开心。”

not (不是)

唱反调。
“今天 不是 周一。”

# 例子：and 逻辑
math_done = True        # 数学做完了
chinese_done = False    # 语文没做完
can_play = math_done and chinese_done
print(can_play)         # 输出 False，因为有一项没完成就不能玩

# 例子：综合条件（正偶数）
a = int(input())
# 大于0 并且 余数为0
is_positive_even = (a > 0) and (a % 2 == 0)
print(is_positive_even)

六、数学魔法：math 库的秘密武器

有时候，基本的加减乘除不够用，比如要算“圆周率 π”或者“开根号”。这时候我们需要召唤 Python 自带的数学工具箱 —— math 库！

召唤咒语： 使用 math 库之前，必须在代码第一行写上：
import math
函数 含义 例子 结果
math.ceil() 向上取整（天花板） math.ceil(3.1) 4
math.floor() 向下取整（地板） math.floor(3.9) 3
math.sqrt() 开平方根 math.sqrt(9) 3.0
math.pow() 幂运算 (和 ** 类似) math.pow(2, 3) 8.0
math.fabs() 绝对值 (变为正数) math.fabs(-5) 5.0 
import math

# 1. 使用圆周率 pi
r = 5
area = math.pi * r * r   # 面积 = 3.14159... * 5 * 5
print(area)

# 2. 开根号（勾股定理 c = sqrt(a*a + b*b)）
a = 3
b = 4
c = math.sqrt(a**2 + b**2)
print(c)  # 输出 5.0

七、典型竞赛题型训练思路

1. 判断奇偶

n = int(input())
# 如果除以2余数为0，就是偶数，输出 True
print(n % 2 == 0)

2. 三数求和、平均数

a, b, c = map(int, input().split())
total = a + b + c   # 求和
avg = total / 3     # 求平均数（注意除法结果是小数）
print(total)
print(avg)

3. 鸡兔同笼：枚举验证思想

故事引入：笼子里一共有 10 个脑袋、28 只脚。鸡 2 条腿，兔 4 条腿。鸡和兔各多少只？

# 这是一个预告式代码，感受一下计算机的枚举逻辑
heads = 10
legs = 28
for chickens in range(heads + 1):       # 假设鸡是从0只到10只
    rabbits = heads - chickens          # 剩下就是兔子
    if 2 * chickens + 4 * rabbits == legs:  # 检查腿数对不对
        print(chickens, rabbits)        # 对了就输出

八、易错点提醒与纠正方法 ⚠️

  1. == 和 = 混淆：
    ❌ if a = b: (这是赋值，错！)
    ✅ if a == b: (这是判断相等，对！)
    口诀：一个等号是“给东西”，两个等号是“比一比”。
  2. 除法混乱：/ 结果是小数，// 结果是整数。
  3. 取余理解错误：% 不是百分号，是“余数”。
  4. 类型没转换：input() 拿到的是字符串，拿着字符串做加法会报错或拼接（"1"+"1"="11"）！记得用 int()。

九、课堂小结：本节知识清单 ✔

  • 数据类型：int (整), float (小), str (文)。做题前记得转换！
  • 算术运算：// 和 % 是好兄弟（分糖果）。** 是变强（平方）。
  • 数学魔法：用 import math 召唤 sqrt 和 pi。
  • 比较运算：结果只有 True 或 False。
  • 逻辑运算：and (且), or (或), not (非)。
  • 加油：只要这些“算一算、比一比”练熟了，你就已经迈进信息学竞赛的大门啦！
