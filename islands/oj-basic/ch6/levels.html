CSP-J1603

宠物动能 (平方计算)

思路分析

动能和速度的平方有关。

我们需要输出 1 到 100 每个数字的平方。

  • 速度 1 -> 1 (1x1)
  • 速度 2 -> 4 (2x2)
  • 速度 3 -> 9 (3x3)
  • ...

循环遍历 1 到 100，每次计算 i * i 即可。

i = 1
1. 初始化速度
int i = 1;
while i <= 100:
2. 循环范围
while (i <= 100) {
  print(i * i, end=" ")
3. 计算平方并输出
  cout << i * i << " ";
  i = i + 1
4. 下一个速度
  i++; }
CSP-J1604

对称花园 (偶数求和)

思路分析

我们要算 1 到 n 之间所有偶数的和。

这就像是一个存钱罐 (变量 s)。

我们可以从 2 开始（第一个偶数），每次往存钱罐里加钱，然后跳 2 步找下一个偶数。

s = 0 (空罐子)
s = s + 2
s = s + 4
s = s + 6...
n = int(input())
输入 n
int n; cin >> n;
s = 0
i = 2
准备求和变量 s
偶数从 2 开始
long long s = 0;
int i = 2;
while i <= n:
只要没超过 n
while (i <= n) {
  s = s + i
累加当前偶数
  s += i;
  i = i + 2
找下一个偶数
  i += 2; }
print(s)
输出总和
cout << s;
CSP-J1605

魔法数 (条件计数)

思路分析

我们要找一种特殊的数字：

  1. 末尾是 6 -> i % 10 == 6
  2. 能被 8 整除 -> i % 8 == 0

我们需要一个计数器 (cnt)，每次找到满足条件的数字，就摁一下计数器 (+1)。

n = int(input())
输入范围 n
int n; cin >> n;
cnt = 0
i = 1
计数器清零
从1开始找
int cnt = 0; int i = 1;
while i <= n:
遍历所有数字
while (i <= n) {
  if i % 10 == 6 and i % 8 == 0:
    cnt = cnt + 1
判断魔法条件
满足则计数+1
  if(i%10==6 && i%8==0)
    cnt++;
  i = i + 1
检查下一个数
  i++; }
print(cnt)
输出个数
cout << cnt;
CSP-J1606

山顶旗帜 (数字反转)

思路分析

输入一个三位数，比如 123，要变成 321。

输入 100，要变成 001。

数字分离法：

  • 个位：n % 10
  • 十位：(n / 10) % 10
  • 百位：n / 100

提示：题目要求输出001，所以不能算成数字1，要挨个输出数字。

n = int(input())
输入三位数
int n; cin >> n;
ge = n % 10
取个位
int ge = n % 10;
shi = (n // 10) % 10
取十位
先去尾再取个位
int shi = (n / 10) % 10;
bai = n // 100
取百位
int bai = n / 100;
print(ge, shi, bai, sep="")
倒序输出
不换行不加空格
cout << ge << shi << bai;
小贴士：如果是任意位数的数字反转，通常需要用到 while 循环配合 % 和 / 运算哦！
但这里明确是三位数，直接数学拆分最快。
