二、最值问题：三只小猪谁最重 (求最大/最小值)

问题： 三只小猪的体重分别是 a, b, c，请找出最重的那只。

很多同学会写很多个 if，比如 if(a>b && a>c)... 这样写不仅累，如果来了 100 只小猪怎么办？

我们来学习一个更聪明的方法：“擂台法” (King of the Ring)！

🥊 擂台法秘籍
  1. 先假设第一个人是擂主（最重的）。
  2. 第二个人上来挑战，如果比擂主重，换人！
  3. 第三个人上来挑战，如果比擂主重，换人！
  4. 最后站在台上的，就是最重的。
int a, b, c;
cin >> a >> b >> c;

// 1. 先让 a 当擂主
int max_val = a; 

// 2. b 上来挑战
if (b > max_val) {
    max_val = b; // b 赢了，b 成为新擂主
}

// 3. c 上来挑战
if (c > max_val) {
    max_val = c; // c 赢了，c 成为新擂主
}

cout << max_val;

三、几何判断：这里的木棍能围成三角形吗？

给你三根木棍，长度分别是 a, b, c。不是任意三根都能围成三角形的哦！

📐 数学原理：三角形任意两边之和大于第三边。
写成代码就是：
a + b > c 并且 a + c > b 并且 b + c > a 
int a, b, c;
cin >> a >> b >> c;

// 必须三个条件同时满足 (&&)
if (a + b > c && a + c > b && b + c > a) {
    cout << "Yes";
} else {
    cout << "No";
}

四、调试技巧：像侦探一样寻找逻辑漏洞

程序写完后不对怎么办？别急着举手问老师，先当一回“代码侦探”！

🕵️ 侦探工具箱
  • 边界测试法：专门测试那些“临界点”的数据。
    • 比如题目说 x < 10，那你一定要测 10 和 9。
    • 比如除法，一定要测分母是不是 0。
  • 间谍输出法：在程序中间加 cout，看看变量到底变成了多少。
    • cout << "debug: " << a << endl;

错误示范： 判断正数

int x;
cin >> x;
if (x > 0) cout << "正数";
else cout << "负数";

🔍 侦探发现漏洞：如果输入 0 怎么办？程序会输出“负数”，这不对！0 既不是正数也不是负数。
