第一章 顺序结构

会听、会说、会算的小程序

目标：让孩子写出会说话、会听话、会算数的小程序，为以后 if、循环、算法打好基础。

一、课堂导入：英雄闯地牢 & 会说话的电脑

教师口吻示例：

“同学们，你们玩过闯关游戏吗？打怪、开宝箱、开大门？”

“今天我们来玩一个‘英雄闯地牢’的游戏，不过英雄的武器不是剑，而是——Python 代码！”

“地牢里有很多石门，只有对电脑说出正确的密码，门才会打开。我们要学会让电脑：”
  • 会说：用 print 把信息显示出来；
  • 会听：用 input 从键盘读信息；
  • 会算：用表达式算结果。

板书关键词：

会说 → print

会听 → input

会算 → 表达式（+ - * / // % **）

二、走进 Python 世界：什么是程序？

1. 生活类比：电脑其实是个“超级听话的小孩”

引导提问：“你跟同桌说：‘把这本书递给我’，同桌会照做，那你对电脑说，它听得懂吗？”

“电脑听不懂人话，只听得懂程序——一条一条的指令。”

解释（面向孩子）：

  • 程序 = 给电脑写的“步骤说明书”。
  • Python = 一种很温柔、好懂的“程序语言”，专门用来告诉电脑要做什么。

对应 OJ 思维：每道 OJ 题都是：题目 = 任务说明书，我们用 Python 写程序 = 解题步骤。

小练习（口头）：

让同学口头说一个“程序”：比如 起床程序：

  1. 关闭闹钟
  2. 穿衣服
  3. 刷牙洗脸
  4. 吃早饭

强调：程序也是这样的“步骤”。

三、第一个程序：会说话的电脑（print）

1. 第一个程序：打印一行话

示例代码（Python 3）：

# 让电脑在屏幕上说一句话 print("Hello, OI!")

逐行讲解：

  • print：就像电脑的“嘴巴”，用来把东西说出来（显示在屏幕上）。
  • ()：括号里是要说的内容。
  • "Hello, OI!"：用双引号括起来的文字，叫字符串。

互动提问：“如果我想让电脑说：你好，世界！ 应该怎么改？”

预期学生答案：print("你好，世界！")

对应 OJ 题型：

只需要输出固定一句话的题目，例如：

  • 输出 Hello World！
  • 输出一两句古诗

→ 只需要写若干行 print("……") 即可。

四、OJ 平台简要介绍（竞赛味道养成）

可以配合实际 OJ 页面演示。要点说明（用很简单的话）：

  • 题目区：上面写着要做什么、输入输出格式。
  • 代码框：我们在里面写 Python 代码。
  • 运行 / 提交：
    • 运行：先自己测试一遍。
    • 提交：交给 OJ 老师，让电脑自动判分。
结果代码 含义 比喻
AC (Accepted) 答案正确 门开了！大大的 PASS！
WA (Wrong Answer) 答案不对 数字或者格式不符合，“再试一次”。
TLE 时间超限 太慢了，像做题超时没交卷。

五、基础输出：print 的各种玩法

1. 打印数字 & 字符串

# 打印一个数字 print(123) # 打印一段文字（字符串） print("今天的作业是写Python 程序")

说明：

  • 数字：直接写 123 这样的不用加引号。
  • 字符串：一定要用 " " 或 ' ' 包起来。

2. 多个内容一起输出：用逗号

name = "小明" age = 11 print("我的名字是", name, "，今年", age, "岁")

说明：多个内容之间用逗号 , 隔开。print 会自动在逗号之间加一个空格。

3. sep 和 end：控制中间的空格和结尾

# 默认情况：中间空格，最后换行 print(1, 2, 3) # 输出：1 2 3 # 用 sep 改变中间的连接符 print(1, 2, 3, sep="-") # 输出：1-2-3 # 用 end 改变结尾（不换行） print("Hello", end="") # 输出：Hello（光标停在后面） print("World") # 接着输出：World # → 整体：HelloWorld

六、基础输入：让电脑“长耳朵”——input

1. 读入一个字符串 / 数字

# 读入一行内容，存到变量 name 里（是字符串） name = input() # 读入一个整数 x = int(input())

讲解：

  • input()：像电脑的耳朵，从键盘听一行内容，结果是字符串。
  • int(...)：把字符串变成整数，比如 "123" → 123。

2. 一行读入两个整数

OJ 常见格式：“一行输入两个整数 a 和 b”。

a, b = map(int, input().split())

打比方解释：

  • input()：读到一整串，例如 "3 5"。
  • .split()：按照空格把字符串切成小块 → ["3", "5"]。
  • map(int, ...)：把每一块都从字符串变成整数。
  • a, b = ...：左边有两个盒子，右边有两个数，一一对应装进去。

七、表达式与运算顺序：会算账的小程序

1. 算术运算符

运算符 含义 示例
+ 加法 3 + 4
- 减法 10 - 2
* 乘法 5 * 6
/ 浮点除法 7 / 2 = 3.5
// 整数除法 7 // 2 = 3
% 取余数 7 % 2 = 1
** 次方（幂） 2 ** 3 = 8

2. 运算优先级：先乘除后加减，有括号先括号

生活类比：点套餐：先选主食（乘除），再加小菜（加减）；括号就像“提前下单”的订单，一定先做好。

print(3 + 4 * 2) # 输出：11 （先算 4 * 2 = 8，再 3 + 8） print((3 + 4) * 2) # 输出：14 （括号先算 3 + 4 = 7，再 7 * 2）

八、变量：给数据起名字

类比：变量 = 贴了标签的盒子

x = 5 # 把 5 放进盒子 x x = x + 1 # 先取出 x 的值 5，再加 1，结果 6 放回 x print(x) # 输出 6

十、易错点提醒 ⚠️

  1. 忘记引号：
    ❌ 错误：print(Hello)
    ✅ 正确：print("Hello")
  2. 中英文符号混用：英文双引号 " "，英文括号 ()。
  3. 忘记把输入转成整数：
    ❌ 错误：x = input(); print(x + 1)
    ✅ 正确：x = int(input()); print(x + 1)
  4. 多余空格 / 少空格：OJ 判结果比较“挑剔”，多一个空格就可能 WA。

十一、课堂小结：本节知识清单 ✔

  • 会说话：print(...) —— 输出数字、字符串；sep、end 控制格式。
  • 会听话：input() —— 记得用 int(input()) 变整数。
  • 会算账：表达式 —— 运算符 + - * / // % **。
  • 变量 —— 用 = 赋值，x = x + 1。
  • 竞赛意识 —— AC、WA、TLE，格式要严谨。
