二、走进 Python 世界：什么是程序？

程序就是：给电脑的一串“排好队”的命令

1. 生活类比：电脑其实是个“超级听话的小孩”

引导提问：“你跟同桌说：‘把这本书递给我’，同桌会照做，那你对电脑说，它听得懂吗？”

“电脑听不懂人话，只听得懂程序——一条一条的指令。”

解释（面向孩子）：
  • 程序 = 给电脑写的“步骤说明书”。
  • Python = 一种很温柔、好懂的“程序语言”，专门用来告诉电脑要做什么。

三、第一个程序：会说话的电脑 (print)

屏幕上显示出来了！

1. 基础用法：打印一句话

# 让电脑在屏幕上说一句话
print("Hello, OI!")

逐行讲解：

  • print：就像电脑的“嘴巴”，用来把东西说出来。
  • ()：括号里是要说的内容。
  • " "：双引号括起来的文字，叫字符串。

2. print 的进阶玩法

# 1. 打印数字（不用引号）
print(123)

# 2. 多个内容一起输出（用逗号隔开）
print("我是", "Python", "小高手")

# 3. 控制结尾不换行 (end)
print("Hello", end="")
print("World")
# 结果：HelloWorld

四、OJ 平台简要介绍

绿色 AC 代表通过，红色 WA 代表错误

结果代码 含义 比喻
AC (Accepted) 答案正确 门开了！大大的 PASS！✅

五、基础输入：让电脑“长耳朵” (input)

通过 input() 聆听键盘的声音

1. 读入一个字符串 / 数字

# 读入一行内容，存到变量 name 里
name = input()

# 读入一个整数（重点：要用 int 变身）
x = int(input())

2. 一行读入两个整数 (OJ常用)

a, b = map(int, input().split())

🔍 详细原理解析（假设输入是 "10 20"）：

1 input()
负责听。电脑听到了一串字："10 20"（注意这是一个字符串，带空格）。
2 .split()
负责切。像菜刀一样，按空格切开，变成了两个小字符串：["10", "20"]。
3 map(int, ...)
负责变身。给每一块都施加 int 魔法，把字符变成真正的数字：[10, 20]。
4 a, b =
负责入座。把做好的两个数字，分别放进变量盒子 a 和 b 里。

六、变量：给数据起名字

变量就是一个“标签盒子”

x = 5          # 把 5 放进盒子 x
x = x + 1      # 拿出 x 的值加 1，再放回去
print(x)       # 盒子现在是 6

七、易错点提醒 ⚠️

  1. 忘记引号：❌ print(Hello) → ✅ print("Hello")
  2. 中英文符号混用：代码里必须用英文标点 ( ) " " ,
  3. 忘记转换整数：输入算数时，别忘了 int()。
  4. 格式错误：OJ 对空格和换行很严格，多一个空格都可能 WA。

八、本章小结

  • 会说话：print(...) —— 输出数字、字符串。
  • 会听话：input() —— 记得用 int() 变整数。
  • 变量 —— 用 = 赋值，数据的小盒子。
