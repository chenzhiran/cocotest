CSP-J1103 精灵朋友

目标：输入名字，输出 "Hello + 名字！"

思路分析

步骤一：准备一个盒子 (变量)

每个精灵名字不同，我们需要一个贴着“名字”标签的盒子来装它。

name

步骤二：放入名字 (输入)

精灵说出名字，我们把它存进盒子里。

步骤三：打招呼 (拼接输出)

输出时要像拼积木一样：
"Hello " + 盒子里的名字 + "！"

输入输出对接

Python
动作连接
C++
# Python 自动创建变量
(无需声明类型)
1. 创建变量
准备一个存字符串的盒子
string name;
name = input()
2. 输入
把键盘输入的内容存入 name
cin >> name;
print("Hello", name + "！")
# 或者 print(f"Hello {name}！")
3. 拼接输出
固定词 + 变量 + 符号
cout << "Hello " << name << "！";
Python input():
这函数就像一只耳朵，专门听键盘输入了什么，然后变成字符串给左边的变量。
C++ string:
C++比较严谨，必须先告诉计算机 name 是个 string (字符串) 类型的盒子，才能往里装字。

CSP-J1104 迟日江山丽

目标：换行输出四句古诗

思路分析

英雄要背古诗，古诗是一行一行的。我们需要让程序也能一行一行地说话。

迟日江山丽，
春风花草香。
泥融飞燕子，
沙暖睡鸳鸯。

在编程里，“换行”很重要。

  • Python的 print 默认就会换行。
  • C++ 需要我们手动添加 endl (End Line) 或者 \n。

多行输出对照

print("迟日江山丽，")
第一句
cout << "迟日江山丽，" << endl;
print("春风花草香。")
第二句
cout << "春风花草香。" << endl;
print("泥融飞燕子，")
第三句
cout << "泥融飞燕子，" << endl;
print("沙暖睡鸳鸯。")
第四句
cout << "沙暖睡鸳鸯。" << endl;
* C++中的 endl 代表 "End Line" (结束这一行并回车)

CSP-J1105 鹦鹉学舌

目标：输入一句话，原封不动输出一句话

思路分析

鹦鹉不创造话，它只是搬运工。

英雄说 (Input) 变量 s
变量 s 鹦鹉说 (Output)

注意：这次没有 "Hello" 之类的额外文字，鹦鹉只重复听到的内容。

输入即输出

Python
解释
C++
# 自动类型推断
定义变量
string s;
s = input()
听 (Input)
内容存入 s
cin >> s;
print(s)
说 (Output)
打印 s 的内容
cout << s;
思考： 如果输入里有空格（比如 "I love coding"），Python的 input() 可以一次读完整行，但 C++ 的 cin 遇到空格会停止。这道题如果输入包含空格，C++可能需要使用 getline(cin, s);。不过对于基础题目，通常 cin >> s 足够处理无空格的字符串。
