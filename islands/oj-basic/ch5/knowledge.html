一、复习回顾：逻辑运算符与嵌套if

在进入新世界之前，我们先检查一下上节课的“魔法装备”。

🧠 知识闪回
  • 逻辑魔法： && (并且), || (或者), ! (非)。
  • 嵌套结构： 就像俄罗斯套娃，一个 if 里面藏着另一个 if。比如先要有钥匙，再要有体力，才能开门。

二、多分支结构：人生的多种可能性 (else if)

生活中的选择往往不止两个。比如：红灯停，绿灯行，黄灯要注意！这时候，双分支 if-else 就不够用了。

我们需要 多分支结构 (else if)，它就像一个多层的自动滑梯。

[Image of else if flowchart diagram] 
// 例子：根据年龄看电影
int age;
cin >> age;

if (age < 12) {
    cout << "看动画片";
} else if (age < 18) {
    cout << "看科幻片";
} else {
    cout << "看动作片";
}
🧙‍♂️ 老师划重点：
  • 计算机很“专一”，一旦发现某个条件满足了，执行完对应的语句后，就会直接跳出整个结构，不会再看后面的 else if。
  • 这就像你找到了丢失的钥匙，就不需要再继续翻箱倒柜了！

三、重点：多分支的“顺序陷阱”

这是多分支结构中最容易犯错的地方！顺序如果错了，结果就会大错特错。

❌ 错误的顺序
int score = 95;
if (score > 60) {
    cout << "及格";
} else if (score > 90) {
    cout << "优秀";
}

结果：输出“及格”。因为95大于60，条件1满足了，直接结束，根本没机会判断优秀！

✅ 正确的顺序
int score = 95;
if (score > 90) {
    cout << "优秀";
} else if (score > 60) {
    cout << "及格";
}

结果：输出“优秀”。先筛选最严格的条件。

💡 黄金法则：
  • 如果是判断 大于 (>) 某个数，要 从大到小 写。
  • 如果是判断 小于 (<) 某个数，要 从小到大 写。
  • 就像筛沙子，要先用孔最小的筛子，把最大的宝石留住！

四、应用案例：成绩等级评定 (A/B/C/D)

我们来做一个老师的小助手，自动把 0-100 的分数转换成等级。

  • 90-100分：A
  • 80-89分：B
  • 70-79分：C
  • 60-69分：D
  • 0-59分：E

利用我们刚才学的“黄金法则”，从大到小判断：

int score;
cin >> score;

if (score >= 90) {
    cout << "A"; 
} else if (score >= 80) {
    // 能走到这里，说明分数一定小于90
    cout << "B";
} else if (score >= 70) {
    // 能走到这里，说明分数一定小于80
    cout << "C";
} else if (score >= 60) {
    cout << "D";
} else {
    // 剩下的都是不及格
    cout << "E";
}
