CSP-J1501

算术计算器 (基础分支)

思路分析

我们要根据输入的符号（+, -, *, /）来决定做什么数学题。

这就好比一个四岔路口：

  • + 往左走做加法
  • - 往右走做减法
  • * 往前走做乘法

注意：Python没有switch语句，通常用if-elif；C++可以用switch。

a, b = map(int, input().split())
op = input()
输入数字和符号
int a, b; char op;
cin >> a >> b >> op;
if op == '+':
  print(a + b)
如果是加号
if(op == '+')
  cout << a + b;
elif op == '-':
  print(a - b)
如果是减号
else if(op == '-')
  cout << a - b;
elif op == '*':
  print(a * b)
如果是乘号
else if(op == '*')
  cout << a * b;
elif op == '/':
  print(a // b)
如果是除号
Python用//整除
else if(op == '/')
  cout << a / b;

CSP-J1503

数字成语 (字典 vs Switch)

思路分析

这道题是典型的“查表”问题。

Python方法 (字典):

就像查字典一样，1对应一帆风顺，2对应二龙腾飞... 把它们存到一个 `{key: value}` 的结构里，直接拿出来。

C++方法 (Switch):

像多路开关，如果 case 是 1，就输出一帆风顺，如果是 2...

n = int(input())
输入数字
int n; cin >> n;
data = {
 1: "一帆风顺",
 2: "二龙腾飞",
 3: "三阳开泰",
 ...
}
核心逻辑
左：定义字典
右：Switch结构
switch(n) {
 case 1: cout<<"一帆风顺"; break;
 case 2: cout<<"二龙腾飞"; break;
 case 3: cout<<"三阳开泰"; break;
 ...
}
print(data.get(n))

// C++在case中已输出

思路分析

1. 判断农产品种类（字符串比较）。

2. 根据种类乘以不同的税率。

  • wheat/corn: 5% (0.05)
  • vegetable/fruit: 3% (0.03)
  • livestock: 7% (0.07)
  • 其他: 2% (0.02)

保留小数：

Python用 "%.1f" 或 f-string。

C++用 fixed setprecision(1)。

kind, val = input().split()
val = int(val)
输入种类和数值
string kind; double val;
cin >> kind >> val;
rate = 0.02
默认税率(其他)
double rate = 0.02;
if kind in ["wheat","corn"]:
小麦或玉米
if(kind=="wheat"||kind=="corn")
  rate = 0.05
设为 5%
  rate = 0.05;
elif kind in ["vegetable","fruit"]:
蔬菜或水果
else if(kind=="vegetable"||...)
  rate = 0.03
设为 3%
  rate = 0.03;
elif kind == "livestock":
  rate = 0.07
牲畜 7%
else if(kind=="livestock")
  rate = 0.07;
print("{:.1f}".format(val*rate))
计算并输出
保留一位小数
printf("%.1f", val*rate);

CSP-J1506

三角花园 (海伦公式)

思路分析

1. 验真伪： 三角形任意两边之和必须大于第三边。

a+b>c 且 a+c>b 且 b+c>a

2. 算面积： 使用海伦公式。

半周长 p = (a+b+c)/2

面积 S = √[p(p-a)(p-b)(p-c)]

需要引入数学库:
Python: import math
C++: #include <cmath>
import math
引入数学库
#include <cmath>
a,b,c = map(float,input().split())
输入三边(实数)

if a+b>c and a+c>b and b+c>a:
判断是否为真地图
if(a+b>c && a+c>b && b+c>a)
  p = (a+b+c) / 2
计算半周长 p
  double p = (a+b+c)/2;
  s = math.sqrt(p*(p-a)*(p-b)*(p-c))
海伦公式
sqrt是开根号
  double s = sqrt(p*(p-a)...);

输出面积
  cout << s;
