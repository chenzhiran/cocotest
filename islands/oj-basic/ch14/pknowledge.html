一、复习回顾：查找最大值的方法

在学习排序之前，我们先回忆一下“擂台法”——如何在列表中找到最大的数。

口诀： 设擂主，逐个比，强则换，弱则离。排序其实就是多次查找最大值的过程！

二、排序概念：让混乱变得有序，稳定性

排序 (Sorting)：将一组杂乱无章的数据，按照从小到大（升序）或从大到小（降序）的顺序重新排列。

什么是“稳定性”？

想象一下，有一对双胞胎，哥哥叫大明，弟弟叫小明，他们考试都考了 100 分。

原始队伍： ... 大明(100), 小明(100) ...

定义： 如果两个相等的元素，在排序前后的相对位置保持不变，就是稳定的。

三、冒泡排序：像气泡一样浮起来

原理： 两个两个比较，把大的往后挪。就像水底的气泡，最大的泡泡会先浮到水面（列表末尾）。

# 冒泡排序 (Bubble Sort)
nums = [5, 2, 8, 1, 9]
n = len(nums)

for i in range(n - 1):       # 外层：控制轮数
    for j in range(n - 1 - i): # 内层：两两比较
        if nums[j] > nums[j+1]:
            # Python的交换魔法！不需要temp变量
            nums[j], nums[j+1] = nums[j+1], nums[j]

四、选择排序：每次挑出最好的

原理： 每次从剩下的数字里，找到最小的那个，把它放到队伍的最前面。

就像体育老师排队：先在全班找最矮的，站第一个；再在剩下的同学里找最矮的，站第二个……

# 选择排序 (Selection Sort)
for i in range(n - 1):
    min_idx = i                    # 假设 i 是最小的
    for j in range(i + 1, n):      # 在后面找更小的
        if nums[j] < nums[min_idx]:
            min_idx = j            # 记录最小值的下标
            
    # 把最小的换到前面
    nums[i], nums[min_idx] = nums[min_idx], nums[i]

五、桶排序：分类投递

原理： 就像之前学的“统计数字次数”。如果我们知道数字范围（比如 0-10 分），我们可以准备 11 个桶。

操作： 拿到一个 3 分，就往 3 号桶放。最后按顺序把桶里的东西倒出来：0 号桶、1 号桶……

# 计数排序/桶排序 (假设数字在0-10之间)
counts = [0] * 11
for x in nums:
    counts[x] += 1 # 投票

# 输出排序结果
for i in range(11):
    for j in range(counts[i]):
        print(i, end=" ")

六、Python Sort：超级魔法 (Built-in)

Python 发明者早就为我们准备了超级跑车。在实际写程序时，我们通常直接使用这两个方法。

🚀 魔法咒语：
1. nums.sort()：直接修改原来的列表（原地排序）。
2. new_list = sorted(nums)：不改变原列表，生成一个新的排好序的列表。 
a = [5, 2, 8, 1, 9]

a.sort()  # 默认升序
print(a)  # [1, 2, 5, 8, 9]

a.sort(reverse=True) # 降序
print(a)  # [9, 8, 5, 2, 1]
