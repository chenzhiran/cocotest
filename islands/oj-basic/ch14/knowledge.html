一、复习回顾：查找最大值的方法

在学习排序之前，我们先回忆一下“擂台法”——如何在数组中找到最大的数。

排序其实就是多次查找最大值（或最小值）的过程！

口诀： 设擂主，逐个比，强则换，弱则离。

二、排序概念：让混乱变得有序，稳定性

排序 (Sorting)：将一组杂乱无章的数据，按照从小到大（升序）或从大到小（降序）的顺序重新排列。

什么是“稳定性”？

想象一下，有一对双胞胎，哥哥叫大明，弟弟叫小明，他们考试都考了 100 分。

原始队伍： ... 大明(100), 小明(100) ...

定义： 如果两个相等的元素，在排序前后的相对位置保持不变，就是稳定的。

三、冒泡排序：像气泡一样浮起来

原理： 两个两个比较，把大的往后挪。就像水底的气泡，最大的泡泡会先浮到水面（数组末尾）。

// 冒泡排序 (Bubble Sort)
for (int i = 0; i < n-1; i++) {       // 外层：控制轮数
    for (int j = 0; j < n-1-i; j++) { // 内层：两两比较
        if (a[j] > a[j+1]) {          // 前面的比后面的大
            int temp = a[j];          // 交换
            a[j] = a[j+1];
            a[j+1] = temp;
        }
    }
}

四、选择排序：每次挑出最好的

原理： 每次从剩下的数字里，找到最小的那个，把它放到队伍的最前面。

就像体育老师排队：先在全班找最矮的，站第一个；再在剩下的同学里找最矮的，站第二个……

// 选择排序 (Selection Sort)
for (int i = 0; i < n-1; i++) {
    int k = i;                        // 假设 i 是最小的
    for (int j = i+1; j < n; j++) {   // 在后面找更小的
        if (a[j] < a[k]) {
            k = j;                    // 记录最小值的下标
        }
    }
    if (k != i) swap(a[i], a[k]);     // 把最小的换到前面
}

五、桶排序

原理： 就像之前学的“统计数字次数”。如果我们知道数字范围（比如 0-10 分），我们可以准备 11 个桶。

操作： 拿到一个 3 分，就往 3 号桶放。最后按顺序把桶里的东西倒出来：0 号桶、1 号桶……

⚠️ 局限性： 只能排整数，而且数字不能太大（比如不能排 1 亿）。

六、STL Sort：站在巨人的肩膀上 (Intro)

C++ 的发明者早就为我们准备了一个超级跑车——sort 函数。它比我们自己写的冒泡排序快得多！

🚀 魔法咒语：
1. 引入头文件：#include <algorithm>
2. 使用：sort(数组名, 数组名 + 长度); 
#include <algorithm> // 必须加这个！
using namespace std;

int a[100] = {5, 2, 8, 1, 9};
// 对前 5 个数排序
sort(a, a + 5); 

// 结果：1 2 5 8 9 (默认升序)
