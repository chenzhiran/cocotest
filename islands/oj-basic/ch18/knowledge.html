一、复习回顾：二维数组的遍历

在开始高难度操作前，我们先复习一下如何“走遍”二维数组的每一个格子。

// 外层循环控制行(i)，内层循环控制列(j)
for (int i = 0; i < n; i++) {
    for (int j = 0; j < m; j++) {
        cout << a[i][j] << " ";
    }
    cout << endl;
}

二、矩阵转置：行变成列的魔法

矩阵转置的意思是：把原来的“行”变成“列”，把原来的“列”变成“行”。

就像拿着一张纸，沿着对角线翻转了一下。

💡 魔法公式：
b[j][i] = a[i][j];
注意：原矩阵是 n 行 m 列，转置后变成 m 行 n 列，形状变了哦！

三、杨辉三角：数字的金字塔

这是一个神奇的数字三角形，每个数字都等于它头顶上两个数字之和。

// 1. 先把墙壁(两边)填成1
for(int i=0; i<n; i++) {
    a[i][0] = 1;
    a[i][i] = 1;
}
// 2. 中间的数 = 头顶两个数之和
for(int i=2; i<n; i++) {
    for(int j=1; j<i; j++) {
        a[i][j] = a[i-1][j-1] + a[i-1][j];
    }
}

四、蛇形填数：贪吃蛇的行走路径

有时候，我们要按照“S”型来填数字：第一行从左到右，第二行从右到左，第三行又从左到右……

规律：
偶数行 (0, 2...)：列 j 从 0 到 m-1 (正着走)。
奇数行 (1, 3...)：列 j 从 m-1 到 0 (倒着走)。

五、图像处理：像素点的简单操作 (翻转/旋转)

在电脑里，图片其实就是一个巨大的二维数组，每个格子代表一个像素点的颜色。

左右翻转 (Mirror)：其实就是每一行的数组进行“逆序”。

// 左右翻转：交换第 j 列和第 m-1-j 列
for(int i=0; i<n; i++) {
    for(int j=0; j<m/2; j++) {
        swap(a[i][j], a[i][m-1-j]);
    }
}
