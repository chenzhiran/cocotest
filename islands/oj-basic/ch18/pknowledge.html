一、复习回顾：二维列表的遍历

在开始高难度操作前，我们先复习一下如何“走遍”二维列表的每一个格子。在 Python 中，我们通常使用下标法来修改内容。

# 初始化一个 n 行 m 列的全 0 矩阵 (关键魔法！)
matrix = [[0] * m for _ in range(n)]

# 外层循环控制行(i)，内层循环控制列(j)
for i in range(n):
    for j in range(m):
        print(matrix[i][j], end=" ")
    print()
🚨 避坑指南：
千万不要写 [[0]*m]*n！这样会创建 n 个指向同一个列表的引用（改一个全都会变）。
正确写法： [[0]*m for _ in range(n)]。

二、矩阵转置：行变成列的魔法

矩阵转置的意思是：把原来的“行”变成“列”，把原来的“列”变成“行”。

就像拿着一张纸，沿着对角线翻转了一下。

# 原矩阵是 n 行 m 列，新矩阵 b 是 m 行 n 列
b = [[0] * n for _ in range(m)]

for i in range(n):
    for j in range(m):
        b[j][i] = a[i][j] # 行变列，列变行

三、杨辉三角：数字的金字塔

这是一个神奇的数字三角形，每个数字都等于它头顶上两个数字之和。

triangle = []
for i in range(n):
    # 1. 每一行先全是1，比如第3行就是 [1, 1, 1]
    row = [1] * (i + 1)
    
    # 2. 修改中间的数 (头尾不需要动)
    for j in range(1, i):
        # 上一行：triangle[i-1]
        row[j] = triangle[i-1][j-1] + triangle[i-1][j]
    
    triangle.append(row)

五、图像处理：极速翻转

在电脑里，图片其实就是一个巨大的二维列表。在 Python 中，实现图像的左右翻转简直易如反掌，因为我们有切片魔法！

[1, 2, 3]

↔️

[3, 2, 1]

row[::-1]

# 左右翻转：每一行都倒过来
for i in range(n):
    matrix[i] = matrix[i][::-1] # 魔法切片！一行代码搞定！
