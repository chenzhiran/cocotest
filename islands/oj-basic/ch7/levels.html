CSP-J1701

圣诞礼物 (数字拆分)

思路分析

我们要检查一个数里有没有恰好3个 '3'。

剥洋葱法：

  • 取个位：n % 10
  • 判断：是不是 3？如果是，计数器+1。
  • 切个位：n = n / 10 (变成整数)
  • 重复：直到 n 变成 0。

Python没有do-while
用while n > 0 即可
n = int(input())
cnt = 0
1. 输入与计数器
long long n; cin >> n;
int cnt = 0;
while n > 0:
2. 循环拆分
do {
  if n % 10 == 3:
    cnt += 1
3. 检查个位
  if (n % 10 == 3) cnt++;
  n //= 10
4. 砍掉个位
  n /= 10;
} while (n > 0);
if cnt == 3: print("YES")
else: print("NO")
5. 判断结果
if (cnt == 3) cout << "YES";
else cout << "NO";
CSP-J1702

翻转魔法 (数字反转)

思路分析

把 123 变成 321。

魔法口袋公式：

ans = ans * 10 + (n % 10)

比如 ans=3，下一个数字是2 -> ans变成 3*10+2 = 32。这样就接在屁股后面了。

*注意：负数怎么处理？C++取模负数会保留符号，逻辑通用。Python最好先存符号，取绝对值翻转后再乘回去。
n = int(input())
ans = 0
1. 初始化
int n; cin >> n;
int ans = 0;
while n != 0:
2. 只要不为0
C++用do-while处理0的情况
do {
  ans = ans * 10 + n % 10
3. 累加反转数
Python负数取模特殊，建议用abs处理
  ans = ans * 10 + n % 10;
  n = int(n / 10)
4. 移除个位
Python int()取整, C++自动取整
  n /= 10;
} while (n != 0);
print(ans)
5. 输出结果
cout << ans;
CSP-J1703

骷髅来袭 (Break运用)

思路分析

我们要凑够 m 个石头。

农民排队交石头，一旦总数够了，就不用收后面人的了。

Break 的作用：

就像装满水桶，水一满马上关水龙头（退出循环），防止溢出或浪费时间。

m, n = map(int, input().split())
stones = list(map(int, input().split()))
1. 读取输入
目标量, 人数, 石头数组
int m, n; cin >> m >> n;
int stones[100]; //输入循环略
total = 0
2. 当前总量
int total = 0;
for x in stones:
3. 遍历每个农民
for(int i=0; i<n; i++) {
  total += x
4. 收集石头
  total += stones[i];
  if total > m:
    break
5. 够了就停！
触发 Break 强制退出
  if(total > m) break;
}
if total > m: print("Yes")
else: print("No")
6. 判断是否足够
if(total > m) cout << "Yes";
else cout << "No";
CSP-J1704

光石奖励 (Continue运用)

思路分析

我们要计算正数的总和和平均值。

Continue 的作用：

如果遇到坏石头（负数或0），直接跳过，看下一个，不要计入总数。

虽然题目样例全是正数，但为了健壮性或题目隐藏要求，我们通常会过滤无效数据。

平均数 = 总和 / 人数(10.0)

arr = list(map(int, input().split()))
1. 输入10个数
int a; int sum = 0;
sum_val = 0
for x in arr:
2. 循环处理
for(int i=0; i<10; i++) {
  cin >> a;
  if x <= 0:
    continue
3. 负数跳过
进入下一次循环
  if(a <= 0) continue;
  sum_val += x
4. 累加
  sum += a;
}
print(sum_val)
print(sum_val / 10)
5. 输出
注意C++浮点除法
cout << sum << endl;
cout << sum / 10.0;
CSP-J1705

药店模拟器 (库存管理)

思路分析

安雅当店长，管理库存 m。

来一个客人，看看他要多少 need。

  • 如果库存够 (m >= need)：卖给他！库存减少，满意人数+1。
  • 如果库存不够：不卖，下一位。
m = int(input())
n = int(input())
1. 总库存与人数
int m; cin >> m;
int n; cin >> n;
reqs = list(map(int, input().split()))
cnt = 0
2. 需求列表与计数
int req; int cnt = 0;
for need in reqs:
3. 循环接待
for(int i=0; i<n; i++) {
  cin >> req;
  if m >= need:
    m -= need
    cnt += 1
4. 够就卖
扣库存，加人数
  if(m >= req) {
    m -= req; cnt++;
  }
}
print(cnt)
5. 满意人数
cout << cnt;
CSP-J1706

翻转魔法2 (Do-While)

思路分析

这次不仅要反转数字，还要记录它是几位数。

任务拆解：

  1. 反转数字 (同题1702)。
  2. 每剥离一位数字，计数器 len++。
  3. 最后输出反转数，再输出位数。

提示：C++同学必须用 do-while 循环哦！这能保证即使输入0，也能运行一次（位数1）。

n = int(input())
rev = 0; length = 0
1. 初始化
int n; cin >> n;
int rev=0; int len=0;
while True:
2. 循环结构
C++用do-while
do {
  rev = rev * 10 + n % 10
  n //= 10
3. 反转与截断
  rev = rev * 10 + n % 10;
  n /= 10;
  length += 1
  if n == 0: break
4. 记位与退出
  len++;
} while(n > 0);
print(f"{rev}{length}")
5. 拼接输出
cout << rev << len;
