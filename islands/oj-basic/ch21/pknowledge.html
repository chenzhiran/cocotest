一、复习回顾：基本数据类型的局限 🤔

以前我们要存一个同学的信息，可能需要定义好几个独立的变量：

name1 = "Xiaoming" # 名字
age1 = 10          # 年龄
score1 = 95.5      # 分数

如果有 100 个同学，就需要 300 个变量！这就像把所有人的书、笔、作业本都混在地上，太乱了！🤯 我们希望能有一个“大箱子”，把一个人的所有信息整整齐齐地打包在一起。

二、类定义：定制专属的“档案袋” (Class) 📂

在 Python 中，我们用 类 (Class) 来设计这个“大箱子”。它就像一个模具，用来生产一个个具体的“对象”。

🏗️ 类 vs 对象：模具与饼干
# 1. 定义图纸 (Class)
class Student:
    # 初始化函数：每次创建新学生，自动调用这里填表
    def __init__(self, name, sid, score):
        self.name = name   # 把传进来的名字记在自己身上
        self.sid = sid
        self.score = score

# 2. 制造实物 (创建对象)
s1 = Student("Xiaoming", 1001, 99.5)

# 3. 使用数据 (用 . 访问)
print(s1.name, "的分数是:", s1.score)
🧙‍♂️ 魔法解析：
__init__ 就像是新生报到处。self 就是“我这个学生自己”。
self.name = name 的意思是：“把我（self）的名字表格，填上参数里传来的名字”。

三、列表存储：存储全班同学的信息 📚

既然 s1 是一个变量，那我们完全可以把这些学生对象放进一个 列表 (List) 里！

这就像是一本厚厚的“班级花名册”，每一页都是一个同学的完整档案。

students = [] # 创建一个空的花名册

# 输入3个同学的信息
for i in range(3):
    print("请输入第", i+1, "个同学的 姓名 学号 分数:")
    # 假设输入：Alice 101 90
    line = input().split() 
    name = line[0]
    sid = int(line[1])
    score = float(line[2])
    
    # 创建对象并放入列表
    s = Student(name, sid, score)
    students.append(s)

四、排序魔法：按成绩排还是按身高排？(Key) ⚖️

普通的数字排序很简单。但是学生有名字、有分数、有学号，电脑怎么知道按哪个排呢？ 🤔

Python 的 sort 函数有一个超级强大的参数叫 key（钥匙）。你需要给它一把“尺子”，告诉它只量哪个属性。

🔑 Key 的工作原理：只看一点
# 规则1：按分数从高到低 (reverse=True)
# lambda s: s.score 意思是：对于每个学生 s，我只看 s.score
students.sort(key=lambda s: s.score, reverse=True)

# 规则2：按学号从小到大
students.sort(key=lambda s: s.sid)
💡 复合排序（进阶）：
如果想“分数高优先，分数一样看学号小优先”怎么办？
Python 可以返回一个元组：key=lambda s: (-s.score, s.sid)
(负号是为了让分数降序，因为默认是升序)

五、应用案例：学生成绩管理系统 🛠️

任务： 输入 n 个学生的信息（姓名、语文、数学），按总分从高到低排序输出。

class Student:
    def __init__(self, name, ch, math):
        self.name = name
        self.total = ch + math # 直接算出总分存起来

n = int(input())
stus = []

for _ in range(n):
    data = input().split()
    s = Student(data[0], int(data[1]), int(data[2]))
    stus.append(s)

# 排序：按 total 总分，倒序
stus.sort(key=lambda x: x.total, reverse=True)

print("--- 成绩单 ---")
for s in stus:
    print(s.name, s.total)
