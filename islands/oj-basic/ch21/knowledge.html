一、复习回顾：基本数据类型的局限 🤔

以前我们要存一个同学的信息，可能需要这样做，定义好几个独立的变量：

string name1 = "Xiaoming"; // 名字
int age1 = 10;             // 年龄
double score1 = 95.5;      // 分数

如果有 100 个同学，就需要 300 个变量！这就像把所有人的书、笔、作业本都混在地上，太乱了！🤯 我们希望能有一个“大箱子”，把一个人的所有信息整整齐齐地打包在一起。

二、结构体定义：定制专属的“档案袋” 📂

结构体 (Structure) 就是我们定制的这个“大箱子”。它允许我们创造一个新的数据类型，就像搭积木一样，把 string、int、double 组合在一起。

🏗️ 结构体：从图纸到实物
// 1. 定义图纸 (告诉电脑 Student 长什么样)
struct Student {
    string name;  // 姓名
    int id;       // 学号
    double score; // 分数
}; // ⚠️ 这里的封口分号千万别忘了！

int main() {
    // 2. 制造实物 (申请一个具体的档案袋)
    Student s1; 
    
    // 3. 存取数据 (用“点”号 . 这把钥匙打开档案袋)
    s1.name = "Xiaoming"; 
    s1.id = 1001;
    s1.score = 99.5;
    
    cout << s1.name << " 的分数是: " << s1.score << endl;
}

三、结构体数组：存储全班同学的信息 📚

既然 Student 已经是一个类型了（和 int 一样地位），那我们就可以创建 Student 类型的数组！

这就像是一本厚厚的“班级花名册”，每一页都是一个同学的完整档案。

Student class1[50]; // 这是一个能存50个同学档案的数组

// 访问第0个同学的名字和分数
class1[0].name = "Zhang San";
class1[0].score = 100;

// 用循环输入全班同学的信息
for(int i=0; i<50; i++) {
    cin >> class1[i].name >> class1[i].id >> class1[i].score;
}

四、结构体排序：按成绩排还是按身高排？(Sort cmp) ⚖️

整数排序很简单，谁大谁排后面。但是结构体有名字、有分数、有学号，电脑怎么知道按哪个排呢？ 🤔

这时候我们需要告诉 sort 函数一个规则，写一个 cmp (比较) 函数，就像请了一位**“排序裁判”**。

⚖️ 裁判员 cmp 的工作原理
🧠 记忆口诀 (cmp函数返回值)：
想让 a 排在 b 前面，就返回 true。
return a.score > b.score; ➡️ 分数高的在前（降序）
return a.id < b.id; ➡️ 学号小的在前（升序） 
// 规则：按分数从高到低排。如果分数一样，按学号从小到大排。
bool cmp(Student a, Student b) {
    if (a.score != b.score) {
        // 分数不同，分数高的(a)排在前面，返回 true
        return a.score > b.score; 
    }
    // 分数相同，学号小的(a)排在前面，返回 true
    return a.id < b.id; 
}

// 在主函数中使用 sort，带上裁判员 cmp
sort(class1, class1 + n, cmp);

五、应用案例：学生成绩管理系统 🛠️

任务： 输入 n 个学生的信息（姓名、语文成绩、数学成绩），计算每个人的总分，并按总分从高到低排序输出。

#include <iostream>
#include <string>
#include <algorithm> // sort 需要这个头文件
using namespace std;

// 1. 定义结构体
struct Stu {
    string name;
    int ch, math, total;
};

// 2. 定义排序规则(裁判)
bool cmp(Stu a, Stu b) {
    return a.total > b.total; // 总分高的排前面
}

int main() {
    int n;
    Stu a[105]; // 结构体数组
    cin >> n;
    // 3. 输入数据并计算总分
    for(int i=0; i<n; i++) {
        cin >> a[i].name >> a[i].ch >> a[i].math;
        a[i].total = a[i].ch + a[i].math; 
    }
    // 4. 排序
    sort(a, a+n, cmp);
    
    // 5. 输出结果
    cout << "--- 成绩单 (按总分排名) ---" << endl;
    for(int i=0; i<n; i++) {
        cout << "姓名: " << a[i].name << ", 总分: " << a[i].total << endl;
    }
    return 0;
}
