一、复习回顾：数组的定义与遍历

在使用数组解决复杂问题前，我们先确认一下基本功是否扎实。

int a[105], n; 
cin >> n;
// 1. 输入数据
for(int i=0; i<n; i++) {
    cin >> a[i];
}
// 2. 遍历输出
for(int i=0; i<n; i++) {
    cout << a[i] << " ";
}

二、逆序输出：倒着说话的艺术

有时候，我们需要把队伍倒过来，比如“回文数”或者“倒流回声”。

秘诀： 控制循环变量 i，让它从 n-1 开始，走到 0 结束，每次减 1。

// 假设 n=5，下标范围是 0~4
// 倒着数：4, 3, 2, 1, 0
for (int i = n - 1; i >= 0; i--) {
    cout << a[i] << " ";
}

三、查找元素：谁是我们要找的人？(线性查找)

任务： 在一堆数字中，找到是否存在数字 x，如果存在，输出它在第几个位置。

方法： 拿着 x 的照片，从第一个人开始比对，直到找到为止。这叫“线性查找”。

int x; 
cin >> x;
bool found = false; // 标记是否找到

for (int i = 0; i < n; i++) {
    if (a[i] == x) {
        cout << "Found at index: " << i;
        found = true;
        break; // 找到了就停止，不用再找了
    }
}
if (!found) cout << "Not Found";

四、最值位置：谁是班级第一名且坐在哪里

我们不仅要找最高分（Value），还要知道是哪位同学（Index）考的。

擂台法升级版： 同时记录 maxVal (最高分) 和 maxIdx (最高分下标)。

// 假设第一个人就是冠军
int maxVal = a[0]; 
int maxIdx = 0;

for (int i = 1; i < n; i++) {
    if (a[i] > maxVal) {
        maxVal = a[i]; // 更新最高分
        maxIdx = i;    // 更新位置
    }
}
cout << "Max: " << maxVal << " Index: " << maxIdx;

五、应用案例：评分系统 (去掉最高分与最低分)

在跳水或体操比赛中，为了公平，评委打分后通常会去掉一个最高分，去掉一个最低分，然后算平均分。

📊 算法公式

有效总分 = (所有分之和 - 最高分 - 最低分)

平均分 = 有效总分 / (人数 - 2)

int sum = 0, maxVal = 0, minVal = 100; // 假设分数0-100
for (int i = 0; i < n; i++) {
    cin >> a[i];
    sum += a[i];               // 累加所有分
    if (a[i] > maxVal) maxVal = a[i]; // 找最高
    if (a[i] < minVal) minVal = a[i]; // 找最低
}
double avg = (sum - maxVal - minVal) * 1.0 / (n - 2);
cout << avg;
