一、复习回顾：列表的定义与遍历

在使用列表解决复杂问题前，我们先确认一下基本功是否扎实。

# 1. 动态输入 n 个数存入列表
n = int(input())
a = []
for i in range(n):
    val = int(input())
    a.append(val)

# 2. 遍历输出
for x in a:
    print(x, end=" ")

二、逆序输出：倒着说话的艺术

有时候，我们需要把队伍倒过来，比如“回文数”或者“倒流回声”。

核心逻辑： 让下标 i 从 n-1 开始，走到 0 结束，每次减 1。

# 算法逻辑：下标法 (推荐掌握，锻炼思维)
n = len(a)
for i in range(n - 1, -1, -1):
    print(a[i], end=" ")
✨ 魔法小贴士：
Python 有个超简单的“魔法切片”：a[::-1] 可以直接生成一个新的倒序列表哦！

三、查找元素：谁是我们要找的人？(线性查找)

任务： 在一堆数字中，找到是否存在数字 x，如果存在，输出它在第几个位置。

方法： 拿着 x 的照片，从第一个人开始比对，直到找到为止。这叫“线性查找”。

x = int(input())
found = False # 标记是否找到

for i in range(len(a)):
    if a[i] == x:
        print("Found at index:", i)
        found = True
        break # 找到了就停止，不用再找了

if not found:
    print("Not Found")

四、最值位置：谁是第一且在哪？

我们不仅要找最高分（Value），还要知道是哪位同学（Index）考的。

擂台法升级版： 同时记录 max_val (最高分) 和 max_idx (最高分下标)。

# 假设第一个人就是冠军
max_val = a[0]
max_idx = 0

for i in range(1, len(a)):
    if a[i] > max_val:
        max_val = a[i] # 更新最高分
        max_idx = i    # 更新位置

print("Max:", max_val, "Index:", max_idx)

五、应用案例：评分系统 (去掉最高分与最低分)

在跳水或体操比赛中，为了公平，评委打分后通常会去掉一个最高分，去掉一个最低分，然后算平均分。

📊 算法公式

有效总分 = (所有分之和 - 最高分 - 最低分)

平均分 = 有效总分 / (人数 - 2)

# Python的魔法函数让这件事变得超简单！
total_sum = sum(a)
max_val = max(a)
min_val = min(a)

# 注意：要除以 (n-2)
avg = (total_sum - max_val - min_val) / (len(a) - 2)
print(avg)
