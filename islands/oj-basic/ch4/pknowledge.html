一、复习回顾：单双分支与布尔值

上节课我们学习了 bool 类型（只有真假）和 if-else 结构。就像生活中的开关和岔路口。

💡 知识点滴
  • 布尔值： True (真) 和 False (假)。(注意Python首字母大写)
  • 单分支 (if)： 条件成立就做，不成立就跳过。
  • 双分支 (if-else)： 非黑即白，条件成立做这件事，否则做那件事。

二、逻辑运算：与、或、非的逻辑迷宫 (and, or, not)

生活中的条件往往不止一个。比如：“我想出去玩，但是必须写完作业 并且 天气晴朗”。

这时候，我们需要请出三位“魔法门卫”来帮我们组合条件！

👮‍♂️
and (与)

绰号：严格的门卫大叔

“必须两个条件都满足（带工牌 且 穿制服），我才放行！”

True and True = ✅ True
👩‍🦱
or (或)

绰号：宽容的宿管阿姨

“只要满足一个条件（是本楼学生 或 有登记），就可以进！”

True or False = ✅ True
👻
not (非)

绰号：调皮的捣蛋鬼

“我就喜欢唱反调！你说东，我偏往西。”

not True = ❌ False
🧙‍♂️ 老师有话说：
  • and (与)： 两个条件都必须为真，结果才为真。
  • or (或)： 只要有一个条件为真，结果就为真。
  • not (非)： 颠倒黑白，真变假，假变真。
# 例子：判断是否可以玩游戏
homework_done = True   # 作业写完了
is_weekend = False     # 不是周末

# 规则：作业做完 并且 是周末，才能玩（严格门卫 and）
if homework_done and is_weekend:
    print("Go play!")
else:
    print("Study hard!") # 因为不是周末，所以输出这个

📜 魔法速查表 (真值表)

条件 A 条件 B A and B (严格门卫) A or B (宽容阿姨)
✅ True ✅ True ✅ True ✅ True
✅ True ❌ False ❌ False ✅ True
❌ False ✅ True ❌ False ✅ True
❌ False ❌ False ❌ False ❌ False

三、嵌套结构：俄罗斯套娃般的判断 (if套if)

有时候，通过了第一关，还有第二关。这就是嵌套 —— 一个 if 里面包裹着另一个 if。

想象一下，你要打开一个大宝箱，里面可能还有一个小宝箱！

只有层层闯关成功，才能到达核心！

if is_holiday:
    # 第一层：如果是假期，进入下一关
    if is_sunny:
        # 第二层：而且天气好，挑战成功！
        print("去公园！")
    else:
        # 第二层失败
        print("在家看书。")
else:
    # 第一层就失败了
    print("去上学。")

四、应用案例：闰年判断的逻辑

这是一个经典的逻辑组合题。怎么判断某一年是不是闰年呢？我们需要用到刚才学的“魔法门卫”！

📅 闰年规则（满足其一即可）：
  1. 能被 4 整除，并且( and ) 不能被 100 整除。（普通闰年，如 2024）
  2. 或者( or ) 能被 400 整除。（世纪闰年，如 2000）
year = int(input())

# 翻译规则为代码： (规则1) 或者 (规则2)
if (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0):
    print("是闰年")
else:
    print("是平年")
