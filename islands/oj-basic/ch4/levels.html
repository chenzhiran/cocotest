三个宝箱 (求最大值)

思路分析

我们要找出三个数 a, b, c 中谁最重。

想象成打擂台：

  • 如果 a 比 b 重，且 a 比 c 重，那 a 就是冠军。
  • 否则，如果 b 比 a 重，且 b 比 c 重，那 b 就是冠军。
  • 剩下情况 c 就是冠军。
a,b,c = map(int,input().split())
输入三个重量
int a,b,c; cin>>a>>b>>c;
if a>=b and a>=c:
a是最大的吗？
if(a>=b && a>=c){
  print(a)
输出 a
  cout<< a; }
elif b>=a and b>=c:
否则 b是最大的吗？
else if(b>=a && b>=c){
  print(b)
输出 b
  cout<< b; }
else:
  print(c)
否则 c最大
else { cout<< c; }

思路分析

擂台法是编程中求最值的经典算法。

步骤：

  1. 假设第一个瓶子(a)是最小的，让它站在擂台上。
  2. 让 b 上来挑战：如果 b 比擂主还小，b 成为新擂主。
  3. 让 c 上来挑战：如果 c 比擂主还小，c 成为新擂主。
  4. 最后留在擂台上的就是最小的。
a,b,c = map(int,input().split())
输入三个数
int a,b,c; cin>>a>>b>>c;
ans = a
假设a是最小
int ans = a;
if b < ans:
b更小吗？
if(b < ans){
  ans = b
更新最小值
  ans = b; }
if c < ans:
c更小吗？
if(c < ans){
  ans = c
更新最小值
  ans = c; }
print(ans)
输出结果

  3. 比较 b 和 c：如果 b > c，交换它们（保证 b 比 c 小）。
  4. 输出 a, b, c。

此时a最小
