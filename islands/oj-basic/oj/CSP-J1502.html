🌸 开启季节之门 (题目描述)

你来到了森林里的四季神殿。神殿有四扇大门，分别通往春、夏、秋、冬。 守门精灵说，只要你说出一个月份（1-12），对应的大门就会打开！

石碑上的规则：

  • 🌱 春： 3, 4, 5 月
  • ☀️ 夏： 6, 7, 8 月
  • 🍂 秋： 9, 10, 11 月
  • ❄️ 冬： 12, 1, 2 月 (跨年啦！)

🧠 魔法原理 (知识点讲解)

1. 区间判断的逻辑

判断一个数 m 是否在 3 到 5 之间，不能写成 3 <= m <= 5。
✅ 必须写成：m >= 3 && m <= 5 (使用逻辑与 &&)。

2. 冬天的特殊处理

冬天包含了 12月、1月和 2月，它们在数字上是不连续的。
✅ 方法一 (逻辑或)：if (m == 12 || m == 1 || m == 2)
✅ 方法二 (排除法)：如果不是春、夏、秋，那剩下的肯定就是冬！直接用 else。

3. Switch 的“穿透”技巧

switch 语句中，如果多个 case 的结果一样，可以省去中间的 break，让它们“掉”到一起执行。
例如：case 3: case 4: case 5: cout << "春"; break;

1. 简洁的区间判断

Python 非常聪明，支持数学写法！
✅ 你可以直接写：if 3 <= m <= 5:

2. 集合判断 (in 操作符)

判断一个数是否在几个特定的数里面，比如冬天 (12, 1, 2)。
✅ 可以写：if m in (12, 1, 2): 或者 if m in [12, 1, 2]:。这比写一堆 or 要优雅得多！

3. 排除法的智慧

既然题目保证输入是 1-12，如果前面判断了春(3-5)、夏(6-8)、秋(9-11)，那剩下的 else 一定是冬天，不需要再写条件了。

⚔️ C++ 的两种解法

方法一：if - else if (区间法)

使用逻辑运算符判断数字范围。

#include <iostream>
using namespace std;

int main() {
    int m;
    cin >> m; // 读入月份

    // 判断春天: 3, 4, 5
    if (m >= 3 && m <= 5) {
        cout << "春";
    }
    // 判断夏天: 6, 7, 8
    else if (m >= 6 && m <= 8) {
        cout << "夏";
    }
    // 判断秋天: 9, 10, 11
    else if (m >= 9 && m <= 11) {
        cout << "秋";
    }
    // 剩下的 (12, 1, 2) 肯定是冬天
    else {
        cout << "冬";
    }

    return 0;
}

方法二：switch - case (穿透法)

利用 switch 的特性，把相同季节的月份放在一起。

#include <iostream>
using namespace std;

int main() {
    int m;
    cin >> m;

    switch (m) {
        // 春天组
        case 3:
        case 4:
        case 5:
            cout << "春";
            break; // 遇到 break 才停止

        // 夏天组
        case 6:
        case 7:
        case 8:
            cout << "夏";
            break;

        // 秋天组
        case 9:
        case 10:
        case 11:
            cout << "秋";
            break;

        // 冬天组 (也可以用 default)
        case 12:
        case 1:
        case 2:
            cout << "冬";
            break;
    }

    return 0;
}

🐍 Python 的两种解法

方法一：if - elif (连写比较)

利用 Python 独特的连续比较语法。

# 1. 读入月份并转为整数
m = int(input())

# 2. 判断季节
if 3 <= m <= 5:     # 春天: 3, 4, 5
    print("春")
elif 6 <= m <= 8:   # 夏天: 6, 7, 8
    print("夏")
elif 9 <= m <= 11:  # 秋天: 9, 10, 11
    print("秋")
else:                # 剩下的 12, 1, 2 是冬天
    print("冬")

方法二：列表查找 (in 操作符)

这种方法非常直观，把可能的月份放进一个“篮子”(列表或元组)里。

# 1. 读入月份
m = int(input())

# 2. 使用 in 判断是否在集合中
if m in (3, 4, 5):
    print("春")
elif m in (6, 7, 8):
    print("夏")
elif m in (9, 10, 11):
    print("秋")
else:
    print("冬")

🌟 魔法笔记 (重点复习)

  • ❄️ 冬天的特殊性： 12月、1月、2月数字不连续，不能简单写成一个区间，用 else 处理最方便。
  • 🛠️ 工具选择：
    C++: 推荐 if-else (逻辑通用) 或 switch (结构清晰)。
    Python: 推荐 3 <= m <= 5 (简洁) 或 in (3,4,5) (直观)。
