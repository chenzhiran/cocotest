平安夜的雪花飘落，圣诞老人正在派发礼物。 今年的规则很特别：只有当英雄的门牌号里，刚好有 3 个数字 '3' 时，才能获得限定大礼包！🎁

👉 任务： 输入一个正整数 m（门牌号），判断其中是否恰好包含 3 个 '3'。
✅ YES： 比如 43833 (有三个3)。
❌ NO： 比如 3931 (只有两个)，或者 13333 (有四个，太多了！)。

🧠 魔法原理 (数值拆位 & Do-While)

我们不用字符串，而是用纯数学的方法，把数字一个个拆下来检查。

1. Do-While 循环

do { ... } while (条件);
它和 while 的区别是：先斩后奏。不管条件符不符合，至少执行一次。
这在处理数字拆位时特别好用，因为即使输入是 0，我们也能进去处理一次。

2. 数值拆位大法

  • 🔪 切个位： m % 10 (拿到最后一位数字)。
  • 🗑️ 丢个位： m /= 10 (把最后一位扔掉，准备切下一个)。

1. Python 模拟 Do-While

Python 没有原生的 do-while 语句。
我们通常使用标准的 while m > 0: 来实现同样的效果。
虽然字符串方法 count('3') 更快，但在 CSP 考试中，掌握数学拆位法（取余、整除）是必备的基本功哦！

2. 数值拆位大法

  • 🔪 切个位： m % 10 (拿到最后一位)。
  • 🗑️ 丢个位： m //= 10 (注意要用双斜杠进行整除！)。

⚔️ C++ 解题代码 (Do-While版)

使用 do-while 循环，一位一位地检查数字。

#include <iostream>
using namespace std;

int main() {
    long long m;   // 门牌号可能很大，用 long long 保险
    cin >> m;

    int cnt = 0;   // 计数器，记录 '3' 的个数

    // do-while 循环：至少执行一次
    do {
        int digit = m % 10;  // 1. 取出最后一位数字
        
        if (digit == 3) {    // 2. 如果是 3，计数器加 1
            cnt++;
        }
        
        m /= 10;             // 3. 删掉最后一位
        
    } while (m > 0);         // 4. 如果还有数字剩下，继续循环

    // 最后的审判：必须刚好等于 3
    if (cnt == 3) {
        cout << "YES" << endl;
    } else {
        cout << "NO" << endl;
    }

    return 0;
}

🐍 Python 解题代码 (数值法)

虽然 Python 可以直接用字符串 count，但为了练习循环逻辑，我们这里使用数学方法。

# 读入整数
m = int(input())

cnt = 0  # 计数器

# 使用 while 循环进行拆位
while m > 0:
    digit = m % 10  # 1. 取出最后一位
    
    if digit == 3:  # 2. 判断是否为 3
        cnt += 1
    
    m //= 10        # 3. 整除 10，去掉最后一位

# 判断结果
if cnt == 3:
    print("YES")
else:
    print("NO")

🌟 魔法笔记 (重点复习)

  • 🔪 拆位公式： 牢记 % 10 取个位，/ 10 删个位。这是处理数字题的万能钥匙。
  • 🔄 Do-While： 适合那种“无论如何都要先做一次检查”的场景。
  • 🎯 刚好等于： 题目说“恰好 3 个”，所以判断条件是 == 3，大于或小于都不行。
