📜 解开进制的谜题 (题目描述)

魔法塔的门上有一个数字 x，它是 n 进制的。
守护者要求你把这个数字转换成 m 进制，才能打开宝藏大门！

👉 任务：
输入：数字 x，原进制 n，目标进制 m。
输出：x 转换成 m 进制后的结果。

例如：把二进制的 101 (n=2) 转换成十进制 (m=10)。
结果是：5

n 进制
101 (2)
➡️
中转站
10
➡️
m 进制
5 (10)

🧠 魔法原理：程序是怎么悄悄变魔术的？

嗨，小朋友！你可能会觉得奇怪，为什么我们要找十进制当翻译官呢？
因为直接把 n 进制切到 m 进制太复杂啦，所以我们要建立一个两步走的智慧中转站！

第一步：像剥洋葱一样，把 n 进制变成十进制中转数

当你在电脑里输入数字 101 的时候，电脑会把它当成一百零一。但题目说它是二进制！
所以，程序需要使用特殊魔法，从右往左把每一位数字单独抠出来。
1. 拿掉末尾：程序用 x % 10 抓到最右边的那位数。
2. 缩减队伍：程序用 x / 10（Python 里是 x // 10）把最右边已经算过的数字切掉。
3. 权重魔法棒：最右边第一位的面值乘以 1，向左进一位就要乘以进制底数 n，再向左就要再乘以 n。程序里的 power（Python 里的 a）就是这根不断变强的魔法棒！

第二步：搭建除法小梯子，把十进制改造成目标 m 进制

现在，我们要把中转站里的十进制数 5，变成全新的 3 进制数。
程序里用的是聪明的短除法：让这个数不停地除以目标进制 m，每次切下它的余数。
1. 找余数：通过 ten % m 算出余数，放进新进制的小口袋里。
2. 变小：通过 ten / m（Python 里是 ten // m）让数字缩小，直到数字变成 0 为止。

第三步：为什么程序最后要倒序输出？

看上面的图解，我们在除法运算中，最先算出来的余数是 2，它是处于最右边的个位；而最后算出来的余数 1，才是前面的高位。
电脑顺着把它们装进口袋里会变成，这可就反了！
所以，不管是 C++ 里的反向 for 循环，还是 Python 里的 reversed 魔法，都是为了把小口袋底朝天翻过来，从后往前打印，拼出正确的密码 12！

⚔️ C++ 解题代码

标准的“先乘权求和，再取余逆序”模板。

#include <iostream>
using namespace std;

int main() {
    int x, n, m;
    cin >> x >> n >> m;

    // --- 第 1 步：n 进制 转 10 进制 ---
    int ten = 0;   // 存十进制结果
    int power = 1; // 权重，初始是 n^0 = 1

    while (x > 0) {
        int digit = x % 10; // 取出最后一位
        ten += digit * power; // 加上它的值
        x = x / 10;           // 删掉最后一位
        power = power * n;    // 权重翻倍 (变成 n^1, n^2...)
    }

    // 特殊情况：如果转换后是 0
    if (ten == 0) {
        cout << 0 << endl;
        return 0;
    }

    // --- 第 2 步：10 进制 转 m 进制 ---
    int ans[100]; // 存余数
    int len = 0;  // 记录有多少位

    while (ten > 0) {
        ans[len++] = ten % m; // 取余数，存入数组
        ten = ten / m;        // 变成原来的 1/m
    }

    // --- 第 3 步：倒序输出 ---
    for (int i = len - 1; i >= 0; i--) {
        cout << ans[i];
    }
    cout << endl;

    return 0;
}

🐍 Python 解题代码

使用手动循环计算，逻辑清晰。

# 输入数据
x, n, m = map(int, input().split())

# 处理数据：n 进制 -> 10 进制
num = 0
a = 1 # 权重
while x != 0:
    num += x % 10 * a  # 取出末位数字 * 当前权重
    x //= 10           # 去掉末位
    a *= n             # 权重变大 (n的倍数)

# 处理数据：10 进制 -> m 进制
ans = []
while num != 0:
    ans.append(str(num % m)) # 取余数
    num //= m                # 整除

# 输出数据
# 如果 ans 是空的(原数字是0)，通常题目会要求输出0
# 但根据题目样例逻辑，这里按常规翻转输出
print(''.join(reversed(ans)))

🌟 魔法笔记 (重点复习)

  • 🌉 中转站： 任何进制转换，先转成 10 进制最安全。
  • ➗ 短除法： 10 进制转其他进制，就是不停地 % m 和 / m。
  • 🔙 倒序： 取余数算出来的是“个位、十位...”，输出时要倒过来变成“...十位、个位”。
