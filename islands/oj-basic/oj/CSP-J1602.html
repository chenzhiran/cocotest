📢 奇数号角密码 (题目描述)

在平原森林村，农民们正在玩“奇数相加游戏”。 为了让游戏开始，必须先破解一串 “号角密码”。
这串密码非常简单：它由 1 到 100 之间所有的 奇数 组成！

👉 任务： 请你把 1~100 的所有奇数全部列出来，用空格隔开，输出在 同一行。
(例如：1 3 5 7 ... 99)

1
+2 ➔
2
3
+2 ➔
4
5
...
99

🧠 魔法原理 (While 循环)

要输出所有的奇数，我们只需要从 1 开始，每次跳过偶数（加 2）。这次我们用 While 循环 来实现。

While 循环三要素

  1. 🏁 初始化： int i = 1; (从第一个奇数开始)
  2. 🚦 条件判断： while (i <= 100) (只要还在范围内就继续)
  3. 👣 手动步长更新： i += 2; 关键！ (每次加2，跳过偶数)

While 循环三要素

  1. 🏁 初始化： i = 1 (从第一个奇数开始)
  2. 🚦 条件判断： while i <= 100: (只要还在范围内就继续)
  3. 👣 手动步长更新： i += 2 关键！ (每次加2，跳过偶数)

不换行输出： 使用 print(i, end=" ")，让数字排排坐。

⚔️ C++ 解题代码 (While版)

使用 while 循环，手动控制变量 i 每次增加 2。

#include <iostream>
using namespace std;

int main() {
    // 1. 初始化：从 1 开始
    int i = 1;
    
    // 2. 条件：只要 i 小于等于 100
    while (i <= 100) {
        cout << i << " "; // 输出数字和空格
        
        // 3. 更新：每次加 2 (找到下一个奇数)
        i += 2;
    }
    
    return 0;
}

🐍 Python 解题代码 (While版)

使用 while 循环，配合 end=" " 实现同行输出。

# 1. 初始化：从 1 开始
i = 1

# 2. 条件：只要 i 小于等于 100
while i <= 100:
    # end=" " 表示打印完不换行，而是加一个空格
    print(i, end=" ")
    
    # 3. 更新：每次加 2 (找到下一个奇数)
    i += 2

🌟 魔法笔记 (重点复习)

  • 👣 步长 (Step)： 找奇数不需要逐个判断，直接从 1 开始每次 i += 2 是最快的。
  • ☠️ 死循环预警： 在使用 while 循环时，千万不要忘记写 i += 2，否则 i 永远是 1，程序会一直停不下来！
  • 📏 同行输出：
    C++ 使用 cout << i << " "。
    Python 使用 end=" "。
