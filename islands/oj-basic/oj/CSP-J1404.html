森林边的小村庄热闹非凡，农民们又要坐大巴去森林啦！🌲 这次，村里的调度员制定了一个更严格的规则：

1️⃣ 大车规则： 只有 大于 10 人 才会派车。
2️⃣ 座位规则： 所有车都是 偶数 座位。

👉 任务： 如果人数 同时满足 “大于 10” 并且 “是偶数”，输出 Yes (出发)，否则输出 No (等待)。

❌ 结果
No
👥 人数 a

大于10? ✅
偶数? ✅
➡️
✅ 结果
Yes

🧠 魔法原理 (解题思路)

这道题需要通过 两道门禁 才能出发。

我们使用 逻辑与 (AND) 魔法，它可以把两个条件连在一起。只有两个条件 都为真，整体才算通过。

🔒 双重门禁

( a > 10 ) AND ( a % 2 == 0 )

必须 同时满足，缺一不可！

📘 C++ 的逻辑与：&&

  • 🔐 符号： 使用 && 表示“并且”。
  • 📝 写法： if (a > 10 && a % 2 == 0)。
  • ⚠️ 注意： 千万不要漏写成一个 & 哦！

📒 Python 的逻辑与：and

  • 🔐 单词： 直接使用英文单词 and 表示“并且”。
  • 📝 写法： if a > 10 and a % 2 == 0:。
  • ✨ 易读性： Python 的代码读起来就像英语句子一样自然。

💻 咒语吟唱 (代码详解)

调度员正在等待程序运行，快写代码帮大家出发吧！

#include <iostream>
using namespace std;

int main() {
    // 1️⃣ 声明变量 a (人数)
    int a;
    
    // 2️⃣ 输入人数
    cin >> a;
    
    // 3️⃣ 双重判断
    // 条件1：a > 10 (人数够多)
    // 条件2：a % 2 == 0 (是偶数)
    // 使用 && 连接，表示“同时满足”
    if (a > 10 && a % 2 == 0) {
        cout << "Yes"; // 可以出发
    } 
    else {
        cout << "No";  // 不满足条件
    }
    
    return 0;
}

# 1️⃣ 输入：读取人数，存入变量 a
a = int(input())

# 2️⃣ 双重判断
# 条件1：a > 10
# 条件2：a % 2 == 0
# 使用 and 连接
if a > 10 and a % 2 == 0:
    print("Yes")  # 可以出发
else:
    print("No")   # 还要等待

⭐ 举例： 如果来了 8 个人，虽然是偶数，但没超过 10，所以不能走。如果来了 11 个人，虽然超过 10，但不是偶数，也不能走！

🌟 魔法笔记 (重点复习)

  • 🔐 逻辑与： C++ 用 && Python 用 and 表示两个条件必须同时成立。
  • ➗ 偶数判断： x % 2 == 0。
  • 🔢 大于判断： x > 10。
