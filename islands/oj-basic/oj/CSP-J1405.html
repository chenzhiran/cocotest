🧠 魔法原理 (解题思路)

我们要像侦探一样，同时检查两个条件是否都成立。

这就需要用到 逻辑与 (AND) 魔法，把两个条件锁在一起。

🔍 双重线索

( c < 100 ) AND ( c % 2 != 0 )

小于 100 并且 是奇数

📘 C++ 的逻辑魔法

  • 🔐 并且： 使用符号 &&。
  • 🌀 奇数： 判断奇数可以用 c % 2 == 1 或者 c % 2 != 0。
  • 📝 组合： if (c < 100 && c % 2 == 1)。

📒 Python 的逻辑魔法

  • 🔐 并且： 使用单词 and。
  • 🌀 奇数： 判断奇数通常用 c % 2 != 0 (不等于 0)。
  • 📝 组合： if c < 100 and c % 2 != 0:。

💻 咒语吟唱 (代码详解)

大家都在等你的推理结果，快写代码找出真正的魔法分数！

#include <iostream>
using namespace std;

int main() {
    // 1️⃣ 声明变量 c (猜测的分数)
    int c;
    
    // 2️⃣ 输入分数
    cin >> c;
    
    // 3️⃣ 双重判断
    // 条件1：c < 100
    // 条件2：c % 2 == 1 (是奇数)
    if (c < 100 && c % 2 == 1) {
        cout << "Yes"; // 符合条件
    } 
    else {
        cout << "No";  // 不符合
    }
    
    return 0;
}

# 1️⃣ 输入：读取分数，存入变量 c
c = int(input())

# 2️⃣ 双重判断
# 条件1：c < 100
# 条件2：c % 2 != 0 (是奇数)
if c < 100 and c % 2 != 0:
    print("Yes")  # 符合条件
else:
    print("No")   # 不符合

⭐ 小技巧： 奇数除以 2 的余数一定是 1 (对于正整数)。所以 % 2 == 1 和 % 2 != 0 都可以用来判断奇数。

🌟 魔法笔记 (重点复习)

  • 🔐 逻辑与： C++ 用 && Python 用 and 表示“同时满足”。
  • 🌀 奇数判断： x % 2 != 0 或 x % 2 == 1。
  • 📉 范围判断： x < 100。
