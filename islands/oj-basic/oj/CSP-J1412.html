🧠 魔法原理 (擂台法)

当有多个数字需要比较时，“擂台法” 是最经典的算法！

🥊 打擂台步骤

  1. 设擂主： 先假设第一个人 a 是最小的，让他站上擂台 (变量 ans)。
  2. 挑战 1： 让 b 上台和擂主比。如果 b 更小，b 成为新擂主。
  3. 挑战 2： 让 c 上台和擂主比。如果 c 更小，c 成为新擂主。
  4. 胜利： 最后站在台上的，就是最小的！

📘 C++ 的擂台代码

  • 🏁 初始化： int ans = a;
  • 🥊 PK： 使用 if (b < ans) ans = b; 来更新擂主。

📒 Python 的擂台代码

  • 🏁 初始化： ans = a
  • 🥊 PK： 使用 if b < ans: ans = b 来更新擂主。

💻 咒语吟唱 (代码详解)

宠物已经准备好打擂台了，快帮它找出救命药水！

#include <iostream>
#include <algorithm> // 引入 min 函数需要的头文件
using namespace std;

int main() {
    // 1️⃣ 声明变量
    int a, b, c;
    cin >> a >> b >> c;

    // 2️⃣ 擂台法：寻找最小值
    int ans = a;      // 先假设 a 是最小的
    if (b < ans) ans = b; // 如果 b 更小，b 成为擂主
    if (c < ans) ans = c; // 如果 c 更小，c 成为擂主

    // 3️⃣ 输出最终的擂主
    cout << ans;
    
    return 0;
}

# 1️⃣ 输入：读取三个整数
a, b, c = map(int, input().split())

# 2️⃣ 擂台法：寻找最小值
ans = a          # 先假设 a 是最小的
if b < ans:      # 如果 b 更小
    ans = b      # b 成为擂主
if c < ans:      # 如果 c 更小
    ans = c      # c 成为擂主

# 3️⃣ 输出结果
print(ans)

✨ 高级魔法 (内置函数)

其实，编程语言里藏着一个直接找最小值的超级魔法 min()！

cout << min(a, min(b, c));
(C++一次只能比两个，所以要套娃)
print(min(a, b, c))
(Python 可以一次比很多个！)

🌟 魔法笔记 (重点复习)

  • 🥊 擂台思想： 设置一个变量 ans，不断地和后面的数 PK，保留更符合条件的那一个。
  • 📉 更新变量： if (新来的 < ans) ans = 新来的;
  • 🛠️ 通用性： 这个方法非常厉害，哪怕有 100 个、1000 个数，也可以用这个方法找出最小的！
