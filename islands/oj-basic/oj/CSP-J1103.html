📜 勇者的任务 (题目描述)

前方出现了光芒！是一群可爱的神秘精灵 ✨。 想要和它们成为朋友，必须学会"听"和"说"。

精灵会先告诉你它的名字（比如"陈之然"），你需要礼貌地回复：
"Hello 陈之然！"

👉 规则： 无论精灵叫什么，你都要先说 Hello，加上空格，加上它的名字，最后加上一个感叹号！

🧠 魔法原理 (解题思路)

我们需要一个能装文字的"神奇盒子"（变量），把精灵的名字装进去，然后再把"Hello"、名字和"！"拼起来！

📘 C++ 三步走：

1. 准备盒子 (string)：
我们需要一个专门装文字的盒子，这种盒子的类型叫 string。我们给盒子起个名字叫 a。
2. 放入名字 (cin)：
用 cin >> a 把精灵说的名字抓进盒子 a 里。
3. 拼接说话 (cout)：
用 cout 像排队一样输出：先是 "Hello "，再是盒子里的名字，最后是 "！"。

📒 Python 三步走：

1. 接收名字 (input)：
用 input() 听精灵说话，并把它放进一个叫 a 的变量盒子里。
2. 魔法胶水 (+)：
用加号 + 把 "Hello "、名字 a 和 "！" 粘在一起，变成一句话。
3. 打印出来 (print)：
最后用 print() 把这句粘好的话展示出来。

💻 咒语吟唱 (代码详解)

快来补全英雄的魔法书，第 8、9、10 行是空的哦！

#include <iostream>
using namespace std;

int main() {
    // 1️⃣ 声明变量：做一个叫 'a' 的字符串盒子
    string a; 

    // 2️⃣ 输入：把听到的名字放进 'a' 里面
    cin >> a; 

    // 3️⃣ 输出：Hello + 名字 + 感叹号
    // 注意 Hello 后面有个空格，最后是中文叹号！
    cout << "Hello " << a << "！";
    
    return 0;
}

# 1️⃣ 输入：听到名字，存到变量 a 里
a = input()

# 2️⃣ 拼接：把 Hello、名字、叹号 粘起来，存到 b
// ⚠️ 别忘了 Hello 后面有个小空格哦！
b = "Hello " + a + "！"

# 3️⃣ 输出：大声说出 b 里的内容
print(b)

🚨 易错点提醒：
1. 空格： "Hello" 和名字之间有一个空格，要写在引号里 "Hello "。
2. 符号： 最后的感叹号是中文全角的 ！，不要写成英文的 ! 哦。

🌟 魔法笔记 (重点复习)

  • 📦 变量： 就像一个小口袋，用来暂时装精灵的名字。
  • 📥 cin >>： 箭头指向右边，表示把外面的数据"流"进变量里。
  • 📤 cout <<： 箭头指向左边，表示把变量里的东西"流"向屏幕。
  • 🔗 字符串拼接： 在 Python 里，两个字符串可以用 + 号连成一个长字符串。
  • 🎯 格式： 计算机最看重格式，空格和标点符号一个都不能错！
