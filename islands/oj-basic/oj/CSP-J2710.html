冰雪国王建了一座神奇的迷宫，需要你按照“神秘规则”把宝石摆放整齐！

📜 任务清单 (题目描述)

国王给了你一个数字 n。你要建造一个比 n 更大的迷宫，大小是 (2n-1) × (2n-1)！

你需要在里面填上数字（宝石），从 1 开始，每次填入一个。宝石摆放的规则如下：

  1. 🌟 起点： 第 1 个宝石放在第一行的正中间。
  2. ↗️ 飞行： 下一个宝石总是想飞向“右上方”（往上一行，往右一列）。
  3. 🌀 传送门：
    • 如果飞出了“天花板”（上面没格子了），就会传送到最底下一行的对应列。
    • 如果飞出了“右墙壁”（右边没格子了），就会传送到最左边一列的对应行。
  4. 💥 撞车啦： 如果飞向右上角时，或者右上方的格子已经有宝石了，那就只能乖乖落在当前宝石的正下方（往下一行，列不变）。

💡 魔法口诀：
先算“右上”在哪儿 ➡️ 检查有没有“出界” ➡️ 检查目标位置是不是被“占用了”！

📖 C++ 魔法书 (知识点)

  • 🧊 二维数组： int a 就像一栋有行有列的魔法公寓大楼。第一个方括号是“楼层(行)”，第二个是“房间号(列)”。
  • 📏 对齐魔法： 题目要求每个数字占 3 个位置。我们需要用到 #include <iomanip> 工具箱里的 setw(3)。它能让数字自动站成整齐的队列哦！

📖 Python 魔法书 (知识点)

  • 🧊 二维列表： [ * L for _ in range(L)]，这句魔法咒语能瞬间造出一个 L×L 的全 0 迷宫！
  • 📏 对齐魔法： 题目要求每个数字宽度为 3。我们可以用 f-string 的炫酷格式化：f"{val:3}"，它会在数字前面自动加空格补齐 3 个宽度，超级方便！
  • ⚠️ 不换行打印： print(..., end='') 告诉电脑，印完这个数字先别换行，接着印下一个！

💻 C++ 魔法咒语 (代码详解)

#include <bits/stdc++.h> // 万能魔法口袋
using namespace std;

int a; // 🧊 准备一个超级大的迷宫，里面全是0

int main() {
    int n;
    cin >> n; // 📥 问国王n是多少
    
    int L = 2 * n - 1; // 🏰 算出真正的迷宫大小 L
    int num = 1;       // 💎 准备放第1个宝石
    
    // 起点位置：r代表行(row)，c代表列(col)
    int r = 1, c = (L + 1) / 2; 
    a[r][c] = num; // 🌟 把1放在第一行正中间
    
    // 开始不断的放宝石，直到填满 L * L 个
    while (num < L * L) {
        // 步骤1：计算想去的“右上方”目标 (next_r, next_c)
        int next_r = r - 1; 
        int next_c = c + 1;
        
        // 步骤2：检查是不是飞出天花板了
        if (r == 1) { 
            next_r = L; 
            next_c = c + 1; 
        }
        // 步骤3：检查是不是飞出右墙壁了
        if (c == L) { 
            next_r = r - 1; 
            next_c = 1; 
        }
        
        // 步骤4：右上角死角 或者是 目标位置已经被别的宝石占了（不为0）
        if ((r == 1 && c == L) || (a[next_r][next_c] != 0)) {
            next_r = r + 1; // 💥 只能往下落一格
            next_c = c;     // 列不变
        }
        
        // 放入新宝石！
        num++; 
        a[next_r][next_c] = num; 
        
        // 移动到新位置，准备下一次跳跃
        r = next_r; 
        c = next_c; 
    }
    
    // 步骤5：公布迷宫地图！
    for (int i = 1; i <= L; i++) {
        for (int j = 1; j <= L; j++) {
            // 📏 setw(3) 保证每个宝石占3个位置对齐
            cout << setw(3) << a[i][j]; 
        }
        cout << endl; // 换行！
    }
    return 0;
}

💻 Python 魔法咒语 (代码详解)

n = int(input()) # 📥 问国王n是多少

L = 2 * n - 1 # 🏰 算出真正的迷宫大小 L

# 🧊 造一个L*L的二维迷宫，全填满0
a = [ * L for _ in range(L)] 

num = 1 # 💎 准备放第1个宝石

# 起点位置：r代表行，c代表列。(注意Python从0开始数)
r = 0 
c = L // 2 
a[r][c] = num # 🌟 把1放在第一行正中间

# 开始不断的放宝石，直到填满 L * L 个
while num < L * L:
    # 步骤1：计算想去的“右上方”目标 (next_r, next_c)
    next_r = r - 1
    next_c = c + 1
    
    # 步骤2：检查是不是飞出天花板了
    if r == 0: 
        next_r = L - 1
        next_c = c + 1
        
    # 步骤3：检查是不是飞出右墙壁了
    if c == L - 1: 
        next_r = r - 1
        next_c = 0
        
    # 步骤4：右上角死角 或者是 目标位置已经被别的宝石占了（不为0）
    if (r == 0 and c == L - 1) or a[next_r][next_c] != 0:
        next_r = r + 1 # 💥 只能往下落一格
        next_c = c     # 列不变
        
    # 放入新宝石！
    num += 1
    a[next_r][next_c] = num
    
    # 移动到新位置，准备下一次跳跃
    r, c = next_r, next_c 

# 步骤5：公布迷宫地图！
for row in a:
    for val in row:
        # 📏 {val:3} 保证每个宝石占3个位置对齐
        print(f"{val:3}", end='') 
    print() # 印完一行要换行！

🌟 重点复习小贴士

  • 🎯 拆解任务： 遇到复杂的移动规则不要怕！用 if 一步一步判断：是不是出上界？是不是出右界？是不是撞车了？
  • 🔄 更新坐标： 每次放完宝石，一定要记得把现在的坐标变成新坐标 r = next_r; c = next_c;，不然宝石就在原地踏步啦！
  • 🛠️ C++ 小技巧： 在排版输出迷宫时，setw(3) 是你的好帮手，不要忘记 #include <iomanip> 哦！
  • 🛠️ Python 小技巧： 在排版输出迷宫时，用 f-string 的 f"{val:3}" 和 end='' 配合，就能印出漂亮的方阵！
