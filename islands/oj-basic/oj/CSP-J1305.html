✨ 寻找发光的秘密 (题目描述)

幽暗的地牢里，一群萤火虫在飞舞 🐝。 英雄发现了一个奇妙的规律：只有 特殊的萤火虫 才会发光！

🔍 规律： 萤火虫身上的数字，如果 末尾是 0，它就会发光。

👉 任务： 编写程序，读入一个数字，判断它的最后一位是不是 0。 如果是，输出 Yes；否则输出 No。

🔢 数字 a
120
取末位 ➡️
尾数
0
= 0 ➡️
💡 结果
Yes
🔢 数字 a
121
取末位 ➡️
尾数
1
≠ 0 ➡️
🌑 结果
No

🧠 魔法原理 (解题思路)

如何让计算机“看到”数字的最后一位呢？
我们有一个专门切尾巴的魔法：对 10 取余 (% 10)。

✂️ 切尾巴魔法

a % 10

任何数除以 10，
余数就是它的个位数！

123 % 10 = 3
50 % 10 = 0

📘 C++ 的侦探魔法

  • ✂️ 获取末位： 使用 a % 10 就可以算出个位数。
  • ⚖️ 判断是否为0： 使用 == 0 来判断尾巴是不是 0。
  • 🚦 完整咒语： if (a % 10 == 0)

📒 Python 的侦探魔法

  • ✂️ 获取末位： 同样使用 a % 10。
  • ⚖️ 判断： 加上双等号 == 0。
  • 🚦 完整咒语： if a % 10 == 0: (别忘了冒号哦！)

💻 咒语吟唱 (代码详解)

点亮萤火虫，照亮地牢的路吧！

#include <iostream>
using namespace std;

int main() {
    // 1️⃣ 声明变量 a
    int a;
    
    // 2️⃣ 输入数字
    cin >> a;
    
    // 3️⃣ 判断末位是否为 0
    // a % 10 可以得到最后一位数
    if (a % 10 == 0) {
        cout << "Yes"; // 末尾是0，发光！
    } 
    else {
        cout << "No";  // 末尾不是0，不亮
    }
    
    return 0;
}

# 1️⃣ 输入：读取数字，存入变量 a
a = int(input())

# 2️⃣ 判断末位
# a % 10 算出个位数，看它是不是 0
if a % 10 == 0:
    print("Yes")  # 发光
else:
    print("No")   # 不发光

⭐ 秘诀： 凡是和“个位数”、“最后一位”有关的题目，第一反应就是用 % 10！

🌟 魔法笔记 (重点复习)

  • ✂️ 取个位： number % 10 是获取整数最后一位的神器。
  • ⚖️ 判断相等： 记得用两个等号 ==。
  • 🚦 逻辑： 末尾是 0 Yes，否则 No。
