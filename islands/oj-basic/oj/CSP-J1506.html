🗺️ 地图鉴定官 (题目描述)

英雄准备在三角花园布置防御，但他手里的地图有真有假。 每张地图上标着三条边长 a, b, c。 你需要编写程序来鉴定地图：

  • ❌ 假地图： 如果三条边围不成三角形，输出 地图是假的。
  • ✅ 真地图： 如果能围成三角形，利用 海伦公式 计算花园面积（保留5位小数）。
输入: 3 4 5
➡️
判定: 3+4>5 ✅
➡️
计算: 面积=6
输入: 1 1 3
➡️
判定: 1+1>3 ❌
➡️
地图是假的。

🧠 魔法原理 (知识点讲解)

1. 三角形判定法则

任意两边之和必须 大于 第三边。只要有一个不满足，就不是三角形。
逻辑公式：(a+b>c) && (a+c>b) && (b+c>a)

2. 海伦公式 (已知三边求面积)

这是一个古老的数学魔法：
1️⃣ 先算半周长 s = (a + b + c) / 2
2️⃣ 再算面积 area = √(s * (s-a) * (s-b) * (s-c)) [Image of Heron's formula triangle]

📘 C++ 的数学工具箱

  • 🧮 开根号： 使用 sqrt() 函数，需要引入头文件 #include <cmath>。
  • 🎯 保留小数： 使用 fixed << setprecision(5)，需要引入头文件 #include <iomanip>。
  • 🔢 数据类型： 边长和面积可能是小数，必须使用 double 类型。

📒 Python 的数学工具箱

  • 🧮 开根号： 使用 math.sqrt() 函数，需要先写 import math。
  • 🎯 保留小数： 使用 f-string 格式化，例如 f"{area:.5f}" 表示保留 5 位小数。
  • 🔢 数据类型： 输入时使用 map(float, ...) 将字符串转为浮点数。

⚔️ C++ 解题代码

别忘了引入 <cmath> 和 <iomanip> 哦！

#include <iostream>
#include <cmath>    // 包含 sqrt 函数
#include <iomanip>  // 包含 setprecision 函数
using namespace std;

int main() {
    // 1. 定义三个变量存储边长，使用 double 以防有小数
    double a, b, c;
    cin >> a >> b >> c;

    // 2. 判断能否构成三角形 (任意两边之和大于第三边)
    if (a + b > c && a + c > b && b + c > a) {
        
        // 3. 如果是真地图，计算半周长 s
        double s = (a + b + c) / 2;
        
        // 4. 使用海伦公式计算面积
        double area = sqrt(s * (s - a) * (s - b) * (s - c));
        
        // 5. 输出面积，保留5位小数
        cout << fixed << setprecision(5) << area;
        
    } else {
        // 6. 否则是假地图
        cout << "地图是假的。";
    }

    return 0;
}

🐍 Python 解题代码

利用 math 库和简洁的逻辑判断。

import math  # 导入数学库，为了使用 sqrt (开根号)

# 1. 读入三个浮点数
a, b, c = map(float, input().split())

# 2. 判断能否构成三角形
if a + b > c and a + c > b and b + c > a:
    
    # 3. 计算半周长 s
    s = (a + b + c) / 2
    
    # 4. 使用海伦公式计算面积
    area = math.sqrt(s * (s - a) * (s - b) * (s - c))
    
    # 5. 输出面积，保留5位小数
    print(f"{area}:.5f")
    
else:
    # 6. 否则是假地图
    print("地图是假的。")

🌟 魔法笔记 (重点复习)

  • 📐 判定逻辑： 必须 同时满足 三个条件 (&& 或 and)，缺一不可。
  • 🔢 精度控制： 题目要求保留 5 位小数，不要直接输出，一定要格式化。
  • 🧮 公式应用： 海伦公式是已知三边求面积的神器，记得先算 s！
