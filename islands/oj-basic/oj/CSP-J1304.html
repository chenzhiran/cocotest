💎 英雄的抉择 (题目描述)

在幽暗的地牢深处，英雄发现了两个神秘的宝箱：宝箱 a 和 宝箱 b。 可是，英雄的体力只能带走其中一个。

经过观察，英雄发现一个秘密：
👉 越重的宝箱，里面的宝物越珍贵！

为了获得更好的宝物，英雄决定带走 重量更大 的那个。 请你编写程序，读入两个宝箱的重量，告诉英雄该拿哪一个！

📦 宝箱 a
10
🆚
📦 宝箱 b
20
➡️ 选大的
🏆 结果
20

🧠 魔法原理 (解题思路)

这道题的任务是“找最大值”。我们需要用到 比较运算符。

PK 规则

如果 a > b

👉 a 赢了，输出 a

否则 (else)

👉 b 赢了，输出 b

📘 C++ 的比较魔法

  • ⚖️ 比较： 使用 if (a > b) 来判断 a 是否比 b 重。
  • 📢 输出： 如果条件成立，cout << a;；否则（else），cout << b;。

📒 Python 的比较魔法

  • ⚖️ 比较： 使用 if a > b: 来进行判断。
  • 📢 输出： 记得缩进！如果 a 大，print(a)；否则 print(b)。

💻 咒语吟唱 (代码详解)

帮英雄做出最明智的选择吧！

#include <iostream>
using namespace std;

int main() {
    // 1️⃣ 声明变量：a 和 b 表示两个宝箱的重量
    int a, b;
    
    // 2️⃣ 输入：读入两个重量
    cin >> a >> b;
    
    // 3️⃣ 比较大小
    if (a > b) {
        // 如果 a 更重，就拿走 a
        cout << a;
    } 
    else {
        // 否则（b 更重或者一样重），拿走 b
        cout << b;
    }
    
    return 0;
}

# 1️⃣ 输入：读取两个重量，分别存入 a 和 b
a, b = map(int, input().split())

# 2️⃣ 比较大小
if a > b:
    # 如果 a 更重，打印 a
    print(a)
else:
    # 否则，打印 b
    print(b)

⭐ 思考： 如果有两个宝箱一样重（比如都是 20），代码会输出什么呢？
答案是：走 else 分支，输出 b（也就是 20）。结果也是对的哦！

🌟 魔法笔记 (重点复习)

  • ⚖️ 大于号： > 用来判断左边是否比右边大。
  • 🛤️ 分支逻辑： 想要在两个东西里选一个，就用 if...else...。
  • 🏆 目标： 我们的目的是输出 数值较大 的那一个。
