🍿 食人魔的观影秩序 (题目描述)

食人魔三兄弟去看电影 🎬。电影院的座位很窄，每排坐 3 个人。 为了不挡住后面人的视线，他们必须遵守一个规则：

“矮的坐前面，高的坐后面！”

👉 任务： 读入三个食人魔的身高，帮他们把顺序排好，从小到大 输出。
(例如：输入 9 6 10，输出 6 9 10)

📥 原顺序
9   6   10
🔄 交换
📤 排好序
6   9   10

🧠 魔法原理 (解题思路)

只有三个数，我们不需要复杂的排序算法。只要通过 “两两比较，交换位置” 就能解决！

我们的目标是：让 a 变成最小的，c 变成最大的。

第1步： 比较 a 和 b。如果 a 比 b 大，就交换。
(确保 a 小于 b)
第2步： 比较 a 和 c。如果 a 比 c 大，就交换。
(现在 a 肯定是三个数里最小的了！)
第3步： 比较 b 和 c。如果 b 比 c 大，就交换。
(搞定！顺序变成 a < b < c)

📘 C++ 的交换魔法

  • 🔄 swap(x, y)： 这是一个标准库函数，可以直接交换两个变量的值。
  • 📝 逻辑： 三次 if 判断，配合 swap。

📒 Python 的交换魔法

  • 🔄 平行赋值： a, b = b, a。这句话的意思是：把 b 的值给 a，把 a 的值给 b，同时完成，不需要临时变量！
  • 📝 逻辑： 同样是三次 if 判断。

💻 咒语吟唱 (代码详解)

食人魔们等着入场呢，快写代码帮他们排队！

#include <iostream>
#include <algorithm> // swap 函数的头文件
using namespace std;

int main() {
    // 1️⃣ 声明变量：a, b, c
    int a, b, c;
    cin >> a >> b >> c;

    // 2️⃣ 排序逻辑：把最小的换到最前面，最大的换到最后面
    
    // 比较 a 和 b，确保 a 比 b 小
    if (a > b) {
        swap(a, b);
    }
    
    // 比较 a 和 c，确保 a 比 c 小
    // 这一步做完，a 就是三个数里最小的了
    if (a > c) {
        swap(a, c);
    }
    
    // 比较 b 和 c，确保 b 比 c 小
    if (b > c) {
        swap(b, c);
    }

    // 3️⃣ 输出：此时顺序一定是 a < b < c
    cout << a << " " << b << " " << c << endl;
    
    return 0;
}

# 1️⃣ 输入：读取三个身高
a, b, c = map(int, input().split())

# 2️⃣ 排序逻辑：利用 Python 的多重赋值进行交换

# 比较 a 和 b，如果 a 大，就交换
if a > b:
    a, b = b, a

# 比较 a 和 c，如果 a 大，就交换
# 做完这步，a 就是最小的
if a > c:
    a, c = c, a

# 比较 b 和 c，如果 b 大，就交换
if b > c:
    b, c = c, b

# 3️⃣ 输出结果
print(a, b, c)

⭐ 思考： 这种“两两比较交换”的方法，其实就是著名的 冒泡排序 的微缩版哦！

🌟 魔法笔记 (重点复习)

  • 🔄 交换： C++ 用 swap(a, b)。 Python 用 a, b = b, a。
  • 📊 排序思想： 确定一个位置（比如让 a 最小），然后再处理剩下的位置。
  • 🎬 结果： 无论输入顺序如何，经过这三步，数字都会乖乖排好队。
