🏔️ 翻越山顶的奇迹 (题目描述)

英雄来到一座高山前。在山南，他看到三面旗帜写着数字，例如 3 5 8。
当他翻过山顶来到山北，回头一看，旗帜的顺序竟然完全颠倒了！变成了 8 5 3。

👉 任务： 输入一个三位整数（如 358），输出翻转后的结果（如 853）。
⚠️ 特别注意： 如果输入 100，翻转后要输出 001（前面的 0 不能丢，因为旗帜还在那里！）。

🧠 魔法原理 (知识点讲解)

1. 数字拆位法 (数学解法)

我们可以通过数学运算把三位数拆开：
🔹 个位： n % 10 (如 358%10=8)
🔹 十位： (n / 10) % 10 (如 358/10=35, 35%10=5)
🔹 百位： n / 100 (如 358/100=3)
然后按 个位、十位、百位 的顺序直接打印，这样就算百位是 0，也能打印出来！

2. 循环取余法 (通用解法)

利用 while 循环，每次把数字的最后一位“切”下来打印。
1️⃣ 打印 n % 10 (最后一位)
2️⃣ 砍掉最后一位 n /= 10
3️⃣ 重复，直到 n 变成 0。

1. 字符串切片 (String Slicing)

Python 处理翻转最快的方法是把它当成字符串！
咒语：s[::-1]
意思是：从头到尾，步长为 -1 (倒着走)。它会原样保留所有字符，包括 0。

2. 数学拆位法

也可以像 C++ 一样用数学方法拆分：
🔹 个位： n % 10
🔹 十位： (n // 10) % 10 (注意用整除 //)
🔹 百位： n // 100
然后利用 f-string 格式化输出：print(f"{c}{b}{a}")

⚔️ C++ 解题代码

方法一：数学拆位法 (推荐)

显式地拆出每一位，直接拼接输出，完美解决 0 的问题。

#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;

    int a = n / 100;        // 百位
    int b = (n / 10) % 10;  // 十位
    int c = n % 10;         // 个位

    // 倒序输出：先输出个位，再十位，最后百位
    // 这样输入100时，c=0, b=0, a=1，输出就是 001
    cout << c << b << a << endl;

    return 0;
}

方法二：While 循环法 (通用)

不断取出最后一位并打印，直到数字变为 0。

#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;

    // 只要 n 还不为 0，就继续拆
    while (n != 0) {
        cout << n % 10; // 打印最后一位
        n /= 10;        // 砍掉最后一位
    }
    // 注意：如果输入是0，这个循环不会执行。
    // 但题目说输入是三位整数，所以不用担心0的问题。
    
    return 0;
}

🐍 Python 解题代码

方法一：字符串切片 (最快)

直接把它当成字符串翻转，既简单又不会丢 0。

# input() 读进来默认就是字符串
a = input()

# [::-1] 表示从头到尾，步长为-1(倒着走)
print(a[::-1])

方法二：数学拆位法

利用数学运算拆分，再拼接。

n = int(input())

a = n // 100          # 百位
b = (n // 10) % 10     # 十位
c = n % 10            # 个位

# 使用 f-string 直接拼接，保证顺序
print(f"{c}{b}{a}")

🌟 魔法笔记 (重点复习)

  • 🚩 前导零陷阱： 题目如果问“翻转后的数值”，100变成1；如果问“翻转后的样子/字符串”，100变成001。本题是后者。
  • 🔪 拆位公式：
    个位：% 10
    砍掉最后一位：/ 10 (C++) 或 // 10 (Python)
