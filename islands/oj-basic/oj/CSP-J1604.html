🧠 魔法原理 (While + If)

我们要从 1 走到 n，把路上的偶数捡起来放到篮子里。

1. 累加器 (Accumulator)

我们需要一个变量来存总和，通常命名为 result 或 sum。
⚠️ 注意： 一定要初始化为 0！
int result = 0;

2. 偶数判断

怎么知道一个数 i 是不是偶数？看它除以 2 也就是余数是不是 0。
✅ 写法：if (i % 2 == 0)

3. 组合技：While 循环遍历

让 a 从 1 变到 n。每一步都检查是不是偶数，是就加到 result 里。

⚔️ C++ 解题代码

使用 while 循环遍历所有数字，用 if 筛选偶数。

#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;  // 输入范围 n

    int i = 1;        // 循环变量，从 1 开始
    int result = 0;   // 累加器，初始化为 0

    // 只要 i 不超过 n，就继续找
    while (i <= n) {
        // 判断：如果是偶数
        if (i % 2 == 0) {
            result += i;  // 把偶数加到结果里
        }
        i++;  // 继续看下一个数
    }

    cout << result;
    return 0;
}

🐍 Python 解题代码

标准的 while + if 组合写法。

n = int(input())  # 输入范围 n

a = 1            # 循环变量，从 1 开始
result = 0       # 累加器

# 只要 a 不超过 n
while a <= n:
    # 判断：如果是偶数
    if a % 2 == 0:
        result += a  # 加到结果里
    
    a += 1  # 别忘了让数字变大，否则死循环！

print(result)

🌟 魔法笔记 (重点复习)

  • ➕ 累加思想： 就像存钱罐，每次遇到符合条件的数（偶数），就扔进去 (+=)。
  • ➗ 取模运算： % 2 == 0 是判断偶数的标准咒语。
  • 🚧 循环变量： i++ (C++) 或 a += 1 (Python) 必须放在 if 外面，确保不管是不是偶数，都要走向下一个数字。
