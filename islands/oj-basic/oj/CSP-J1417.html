🏪 英雄的采购难题 (题目描述)

英雄要带领小勇士们参加 NOIP 试炼，需要准备至少 n 支铅笔作为礼物。 商店里有 3 种 不同的包装，每种包装的数量和价格都不一样。

⚠️ 规则：
1. 只能选择 其中一种 包装购买。
2. 包装 不能拆开 (意味着你买的数量可能比 n 多一点点)。

👉 任务： 计算出买够 n 支铅笔，最少 需要花多少金币？

🎯 目标：57 支
方案 1
每包2支 / 2元
需 29 包
💰 58元
方案 2
每包50支 / 30元
需 2 包
💰 60元
方案 3
每包30支 / 27元
需 2 包
💰 54元 (最少)

🧠 魔法原理 (解题思路)

这道题有两个难点：

  1. 怎么算买几包？ 不能只买半包，如果算出来是 2.1 包，你也得买 3 包。这叫做 “向上取整” (Ceiling)。
  2. 怎么找最便宜？ 算出三种方案的总价，然后像“打擂台”一样找出最小值。

⬆️ 向上取整魔法

假设你需要 n 支，每包有 cnt 支。

C++ 公式： (n + cnt - 1) / cnt

原理：如果是整除，结果不变；如果有余数，分子加上 (cnt-1) 后会自动进位！

Python 函数： math.ceil(n / cnt)

需要先 import math。它会自动帮你把小数向上进位成整数。

📘 C++ 的策略

  • 🏁 初始值： 把 ans 设为一个很大的数（比如 1亿），这样任何正常的方案都会比它小，从而更新它。
  • 🔄 循环： 循环 3 次，每次读入一种包装，算出总价，用 min() 函数更新答案。

📒 Python 的策略

  • 🏁 初始值： 可以设为无穷大 float('inf')。
  • 🔄 循环： 使用 for i in range(3): 读取三次数据，计算并比较。

💻 咒语吟唱 (代码详解)

快帮英雄计算最划算的方案吧！

#include <iostream>
#include <algorithm> // 包含 min 函数
using namespace std;

int main() {
    int n;          // 需要的铅笔总数
    cin >> n;
    
    int ans = 1e9;  // 初始化答案为一个很大的数(10的9次方)
    int cnt, price; // 每包数量，单价
    
    // 循环 3 次，处理 3 种包装
    for(int i=0; i<3; i++) {
        cin >> cnt >> price;
        
        // 核心魔法：向上取整计算包数
        int packs = (n + cnt - 1) / cnt;
        
        // 计算这种包装的总花费
        int total = packs * price;
        
        // 更新最小花费 (打擂台)
        if (total < ans) {
            ans = total;
        }
    }
    
    cout << ans << endl;
    return 0;
}

import math  # 导入数学库，为了使用 ceil (向上取整)

n = int(input())      # 需要的铅笔总数
ans = float('inf')    # 初始化答案为无穷大

# 循环 3 次，处理 3 种包装
for i in range(3):
    # 读入每包数量 cnt 和 单价 price
    cnt, price = map(int, input().split())
    
    # 核心魔法：向上取整计算包数
    # math.ceil(57 / 30) 会得到 2
    packs = math.ceil(n / cnt)
    
    // 计算总花费
    total = packs * price
    
    # 更新最小花费
    if total < ans:
        ans = total

print(ans)

⭐ C++ 小技巧： 公式 (n + a - 1) / a 是 C++ 程序员的必备常识，它不用调用函数就能实现向上取整，速度很快哦！

🌟 魔法笔记 (重点复习)

  • ⬆️ 向上取整： 只要有余数就进位，不能四舍五入。
  • 🥊 最小值： 初始化为极大值，然后不断比较更新。
  • 🏪 审题： 题目说“只能选一种包装”，所以我们是比较 3 个结果，而不是混合购买。
