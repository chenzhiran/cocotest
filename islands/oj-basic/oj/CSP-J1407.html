🧠 魔法原理 (解题思路)

以前我们只有两个选择（是/否），现在有 三个选择。这时候就需要用到 多重分支 魔法！

< 60 分
不满足，往下走
< 100 分
还是不满足
满分！
(剩下的情况)

📘 C++ 的三岔路口

  • 🚦 if (a < 60)：第一关，检查是否不及格。
  • 🚧 else if (a < 100)：如果第一关没进（说明至少60分了），再看是不是没到100。
    （💡这里不需要写 a >= 60 && a < 100，因为能走到这里肯定 >= 60！）
  • 🏁 else：前两关都没进，那肯定是满分（>=100）了！

📒 Python 的三岔路口

  • 🚦 if a < 60:：第一道门。
  • 🚧 elif a < 100:：elif 是 else if 的缩写。如果第一道门没进，再检查这道门。
  • 🏁 else:：剩下的所有情况（也就是满分）。

💻 咒语吟唱 (代码详解)

大屏幕准备好了，快写代码来显示结果吧！

#include <iostream>
using namespace std;

int main() {
    // 1️⃣ 声明变量 a (分数)
    int a;
    
    // 2️⃣ 输入分数
    cin >> a;
    
    // 3️⃣ 多重判断 (三岔路)
    if (a < 60) {
        // 情况1：不及格
        cout << "很遗憾，您没有通关。";
    } 
    else if (a < 100) {
        // 情况2：及格但没满分 (隐含了 a >= 60)
        cout << "恭喜通关！";
    } 
    else {
        // 情况3：剩下的情况 (满分)
        cout << "恭喜满分通关！";
    }
    
    return 0;
}

# 1️⃣ 输入：读取分数，存入变量 a
a = int(input())

# 2️⃣ 多重判断
if a < 60:
    # 情况1：不及格
    print("很遗憾，您没有通关。")
elif a < 100:
    # 情况2：及格但没满分
    print("恭喜通关！")
else:
    # 情况3：满分
    print("恭喜满分通关！")

⭐ 逻辑小贴士： 判断顺序很重要！如果把 a < 100 写在最前面，那么 50 分也会被判断成“恭喜通关”，这就错啦！要从小到大（或从特殊到一般）排队判断。

🌟 魔法笔记 (重点复习)

  • 🛤️ else if / elif： 用来处理“如果不是第一种情况，那是不是第二种...”的逻辑。
  • 🧹 排除法： 程序是按顺序执行的，走到 else if 时，意味着前面的 if 肯定是假的。
  • 🏁 else： 处理所有剩下的情况，不需要写条件。
