🐂 双十一大促 · 阶梯折扣

🎉 牦牛集市的大促销 (题目描述)

在一望无际的金色沙漠里 🏜️，小英雄准备买一些牦牛 🐂 当伙伴。 正好赶上双十一，老板给出了超值的 阶梯折扣：

  • 🟡 5头以内： 不打折 (原价)
  • 🟢 5 ~ 10头： 打九折 (x 0.9)
  • 🔵 11 ~ 20头： 打八折 (x 0.8)
  • 🟣 21 ~ 30头： 打六折 (x 0.6)
  • 🔴 30头以上： 打四折 (x 0.4)

👉 任务： 读入 单价 (p) 和 数量 (n)，帮小英雄算算一共要付多少钱！

💰 单价 p
10.0
🐂 数量 n
2
< 5 (不打折)
➡️
🪙 总价
20.0

🧠 魔法原理 (解题思路)

这是一个典型的 多重分支 (if - else if - else) 问题。想象你在爬楼梯，每上一层，规则就变一次。

❓ n < 5 ? (是 -> 原价)
❓ n ≤ 10 ? (是 -> 0.9折)
❓ n ≤ 20 ? (是 -> 0.8折)
❓ n ≤ 30 ? (是 -> 0.6折)
🏁 剩下所有 ( -> 0.4折)

💡 小技巧： 当我们走到第二层 else if (n <= 10) 时，其实已经隐含了 n >= 5 (因为如果小于5，第一层就进去了)，所以不需要重复写 n >= 5 && n <= 10。

📘 C++ 的小数计算

  • 🔢 变量类型： 价格可能是小数，所以 p 和 total 要用 double。数量 n 是整数，用 int。
  • 🛤️ 分支结构： 使用 if ... else if ... else 逐层判断。

📒 Python 的小数计算

  • 🔢 变量类型： 输入时记得转换，p 转 float，n 转 int。
  • 🛤️ 分支结构： 使用 if ... elif ... else 逐层判断。

💻 咒语吟唱 (代码详解)

沙漠的风在呼啸，快写下代码完成交易吧！

#include <iostream>
using namespace std;

int main() {
    // 1️⃣ 声明变量
    double p, total; // p是单价，total是总价，可能有小数
    int n;           // n是数量，一定是整数
    
    // 2️⃣ 输入
    cin >> p >> n;
    
    // 3️⃣ 阶梯折扣计算
    if (n < 5) {
        total = p * n;        // 不打折
    } 
    else if (n <= 10) {
        total = p * n * 0.9;  // 5~10头，9折
    } 
    else if (n <= 20) {
        total = p * n * 0.8;  // 11~20头，8折
    } 
    else if (n <= 30) {
        total = p * n * 0.6;  // 21~30头，6折
    } 
    else {
        total = p * n * 0.4;  // 超过30头，4折
    }
    
    // 4️⃣ 输出总价
    cout << total;
    
    return 0;
}

# 1️⃣ 输入：一行读入两个数，切分
line = input().split()
p = float(line[0])  # 单价转小数
n = int(line[1])    # 数量转整数

# 2️⃣ 阶梯折扣计算
if n < 5:
    total = p * n         # 不打折
elif n <= 10:
    total = p * n * 0.9   # 9折
elif n <= 20:
    total = p * n * 0.8   # 8折
elif n <= 30:
    total = p * n * 0.6   # 6折
else:
    total = p * n * 0.4   # 4折

# 3️⃣ 输出总价
print(total)

⭐ 公式记忆： 总价 = 单价 × 数量 × 折扣率。我们通过 if 判断来改变公式里的“折扣率”。

🌟 魔法笔记 (重点复习)

  • 🛤️ 多重分支： 只要前面的条件满足了，后面的 else if 就不会再执行，这就是“截断效应”。
  • 🔢 数据类型： 涉及到钱和折扣，一定要用浮点数 (double / float)。
  • 🐫 顺序： 判断区间时，按照从小到大（或从大到小）的顺序写，逻辑最清晰。
