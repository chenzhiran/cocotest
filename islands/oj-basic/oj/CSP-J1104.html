春风花草香。

🧠 魔法原理 (解题思路)

这道题就像是我们在作业本上抄写课文，最重要的就是：按顺序 和 换行。

第一句
迟日...
⬇️
第二句
春风...
⬇️
第三句
泥融...
⬇️
第四句
沙暖...

📘 C++ 的换行魔法：endl

  • 🗣️ cout： 依然是我们熟悉的大喇叭。
  • ↩️ endl： 这是 "End Line" 的缩写。每次说完一句诗，都要加一个 << endl，就像你在写作业时按了一下回车键，光标就跑到下一行去了。
  • 🧱 积木： 我们需要堆4块积木（写4行代码），每一块负责输出一句诗。

📒 Python 的自动换行

  • 🖨️ print()： Python 的打印机非常聪明。
  • ✨ 自动回车： 每次你用一次 print()，它打印完内容后，会自动帮你按一下回车键换到下一行。
  • 📜 四次打印： 所以，你只需要写 4 次 print，分别把四句诗放进去就好啦！

💻 咒语吟唱 (代码详解)

快把这首优美的诗变成代码吧！

#include <iostream>
using namespace std;

int main() {
    // 第一行：说完后加上 endl 换行
    cout << "迟日江山丽，" << endl;
    
    // 第二行
    cout << "春风花草香。" << endl;
    
    // 第三行
    cout << "泥融飞燕子，" << endl;
    
    // 第四行
    cout << "沙暖睡鸳鸯。" << endl;

    return 0;
}

# Python 会自动换行，所以我们只需要排队打印四次

print('迟日江山丽，')
print('春风花草香。')
print('泥融飞燕子，')
print('沙暖睡鸳鸯。')

# 每一句都写在引号里，别写错字哦！

💡 小贴士： 如果不想自己打字，可以把题目里的诗句复制粘贴到代码里，这样就不会有错别字啦！

🌟 魔法笔记 (重点复习)

  • ⤵️ 顺序执行： 计算机是从上往下读代码的，所以你的 cout 或 print 顺序不能乱。
  • ↩️ 换行： C++ 必须用 endl 才能换行。 Python 的 print 自带换行功能。
  • 📝 符号准确： 诗句里的标点是中文的（， 和 。），代码里的符号（如分号 ; 引号 ""）是英文的。
