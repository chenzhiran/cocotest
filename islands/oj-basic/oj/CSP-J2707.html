📜 修复古老的数字石门 (题目描述)

英雄来到神庙，面前是一扇巨大的石门。
石门上需要刻满数字，规则如下：

1. 顺序填充： 从 1 开始，一直刻到 n * n。
2. 排列方式： 按行排列，刻满一行 n 个数，再换下一行。
3. 严苛排版：
    ⚠️ 每个数字占 5 个字符宽度。
    ➡️ 数字要 右对齐。

当 n=3 时的刻印顺序
1
2
3
4
5
6
7
8
9

🧠 魔法原理 (Formatting)

这一关的难点不在于生成数字，而在于 怎么输出得整整齐齐。

📐 格式化魔法：
题目要求：“占 5 个宽度，右对齐”。
这意味着，如果数字是 1，它前面要补 4 个空格：" 1"。
如果数字是 100，前面补 2 个空格：" 100"。

➕ 额外空格：
题目还说“相邻数字间再加一个空格”。所以最终输出应该是：
宽度5的数字 + 空格

C++ 工具：iomanip

需要引入头文件 #include <iomanip>。
使用 setw(5) 来设定下一个输出项的宽度。
默认就是右对齐，非常方便！

Python 工具：格式化字符串

可以使用 "%5d" % num 或者 f-string f"{num:5}"。
5d 的意思就是：宽度为 5 的整数 (decimal)。

⚔️ C++ 解题代码

利用 setw 控制宽度，双重循环填充。

#include <iostream>
#include <iomanip> // 必须包含，为了使用 setw
using namespace std;
const int MAXN = 105;
int a[MAXN][MAXN]; // 定义二维数组
int n;
int main() {
    cin >> n;
    int tot = 0; // 计数器，记录当前要刻的数字
    // 外层循环：行
    for(int i = 0; i < n; i++) {
        // 内层循环：列
        for(int j = 0; j < n; j++) {
            // 1. 填入数字
            a[i][j] = ++tot;

            // 2. 格式化输出
            // setw(5) 保证数字占5格
            // 后面再加 " " 是题目要求的额外间隔
            cout << setw(5) << a[i][j];
        }
        // 每一行刻完要换行
        cout << endl;
    }
    return 0;
}

🐍 Python 解题代码

无函数脚本，使用 %5d 占位符。

# 1. 读入 n
n = int(input())
# 计数器，从 0 开始，后面先 +1 再用
cnt = 0
# 2. 双重循环打印
for i in range(n):
    for j in range(n):
        # 数字递增
        cnt += 1
        # 格式化输出
        # %5d 表示占 5 位宽度的整数
        # end=" " 表示数字后面加个空格，且不换行
        print("%5d" % cnt, end="")
    # 每一行结束，打印一个换行符
    print()

🌟 魔法笔记 (重点复习)

  • 🔢 计数器： 在双重循环外面定义 tot = 0，然后在循环内部每次 ++tot，就能产生连续的数字。
  • 📏 setw(n)： 这是 C++ 控制输出宽度的神器。注意它只对 紧跟在后面 的那一个输出有效。
  • 🚧 空格陷阱： 题目说“相邻数字间有一个空格”，如果只用 setw(5)，当数字很大（如10000）时，它们会连在一起，所以必须额外输出一个空格。
