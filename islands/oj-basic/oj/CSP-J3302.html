⭐ 重新排列荣誉榜 (题目描述)

期末考试结束后，英雄被请到校园排行榜大厅。
大厅里记录着 N 位学生的成绩信息，每位学生都有：姓名、语文成绩、数学成绩。

👉 任务：
请按照总分（语文 + 数学）从高到低重新排列名单。
注意：如果两位学生的总分一样，必须保持他们原来的输入顺序！

📥 输入格式

第 1 行：一个整数 N (1 <= N <= 100)
接下来 N 行：每行包含 姓名（不超过 10 个小写字母）、语文成绩（0~100 的整数）、数学成绩（0~100 的整数），中间用空格分隔。

📤 输出格式

输出排好序后的名单，共 N 行。每行格式为：
姓名 语文 数学 总分

输入样例 1
4
gaoxiang 78 96
wangxin 70 99
liujia 90 87
zhangjin 78 91
输出样例 1
liujia 90 87 177
gaoxiang 78 96 174
wangxin 70 99 169
zhangjin 78 91 169
💡 什么是“保持原来的输入顺序”？（以总分169分的两位同学为例）
排序前 (输入顺序)
1. wangxin 169分
2. zhangjin 169分
➔
排序后 (依然保持)
wangxin 169分
zhangjin 169分

🧠 魔法原理 (Stable Sort Logic)

这一关的核心是 多维度打包存储 以及 稳定排序 (Stable Sort) 的处理。

📊 算好总分再干活：
读入每一位同学的语文和数学成绩后，先在程序里把它们加起来得到 score (总分)，作为之后排序的唯一依据。

⚖️ 稳定排序的防线：
当分数值相同的时候，谁先来谁就排在前面。
  • Python： 默认的内置排序算法（Timsort）天生就是稳定的，所以直接针对总分做降序即可！
  • C++： 常用的 std::sort 算法是不稳定的。为了不打破先来后到的规则，我们需要在结构体中多加一个变量 id 来记录它在输入时的序号。排序时，如果总分相同，就对比序号，序号小的（即先输入的）排在前面。

C++ 实现细节

在定义结构体 struct student 时，新增一个成员 int id;。
编写比较函数 cmp 时，使用条件分支判断：
如果 stu1.score != stu2.score，返回分大者；否则，返回 stu1.id < stu2.id。

Python 实现细节

我们可以利用二维列表存储每一个学生。比如：[姓名, 语文, 数学, 总分]。
使用列表的 sort() 方法，配合 key=lambda x: x 提取总分，并设置 reverse=True 来达到降序排列。

⚔️ C++ 解题代码

通过在结构体中记录输入序号，在自定义比较函数中完美守护稳定排序！

#include <iostream>
#include <algorithm> // 包含 sort 排序函数的头文件
using namespace std;

// 定义学生档案卡
struct student {
    char name[30];   // 姓名 
    int chinese;     // 语文分数 
    int math;        // 数学分数 
    int score;       // 总分 
    int id;          // 输入序号（用来保证稳定排序）
} stu[101];

// 自定义排序规则函数
bool cmp(student stu1, student stu2) {
    // 如果总分不相等，按总分从高到低排序
    if (stu1.score != stu2.score) {
        return stu1.score > stu2.score; 
    }
    // 如果总分相同，输入序号小的（先输入的）排在前面
    return stu1.id < stu2.id; 
}

int main() {
    int n;
    cin >> n;
    
    for (int i = 0; i < n; i++) {
        cin >> stu[i].name >> stu[i].chinese >> stu[i].math; // 输入信息 
        stu[i].score = stu[i].chinese + stu[i].math;         // 自动算好总分 
        stu[i].id = i;                                       // 记住当前的输入序号
    }
    
    // 魔法重排：利用写好的规则进行稳定排序
    sort(stu, stu + n, cmp); 
    
    // 输出排好序后的成绩单
    for (int i = 0; i < n; i++) {
        cout << stu[i].name << " " << stu[i].chinese << " " 
             << stu[i].math << " " << stu[i].score << endl;
    }
    
    return 0;
}

🐍 Python 解题代码

利用 Python 原生稳定的内置排序，简洁高效地解决战斗。

# 1. 读入学生总人数
n = int(input())

students = []

# 2. 循环读入并解析每一行的数据
for i in range(n):
    data = input().split()
    name = data[0]
    chinese = int(data[1])
    math = int(data[2])
    
    # 计算总分
    total_score = chinese + math
    
    # 将该同学的四大完整信息打包成一个列表，存入大名单
    students.append([name, chinese, math, total_score])

# 3. 按照总分进行降序排序
# key=lambda x: x 代表用第四项（即总分）作为排序依据
# reverse=True 代表从大到小降序排列
students.sort(key=lambda x: x[3], reverse=True)

# 4. 循环输出排序好的新榜单
for s in students:
    print(f"{s} {s} {s} {s}")

🌟 魔法笔记 (重点复习)

  • 🧮 衍生数据先计算： 在输入完毕的第一时间就把总分算好并和数据绑定，在排序时就能直接拿来比对，效率最高。
  • 🛡️ 稳定排序（Stable Sort）： 排序时如果遇到两值相等，不改变它们原来的相对顺序。Python 自带的排序天生稳定，而 C++ 中需要我们人为通过记录序号 id 的方式或使用 stable_sort 来满足要求。
  • 📝 输出格式校对： 本题输出时除了原本读入的三个信息外，还需要额外在尾部打印出刚刚算好的总分，千万不要漏掉哦！
