🧠 魔法原理 (While 循环)

我们需要遍历 1 到 100 的每一个数字，计算它的平方。使用 While 循环 是个好主意！

1. 平方计算

在编程中，求平方最简单的方法就是“自己乘自己”。
✅ 写法：i * i

2. While 循环结构

  • 🏁 初始化： int i = 1; (从1开始)
  • 🚦 条件： while (i <= 100) (直到100结束)
  • 👣 更新： i++; (每次加1，不要忘了！)

1. 平方计算

计算平方可以使用乘法符号。
✅ 写法：a * a (或者 a ** 2)

2. 不换行输出

默认的 print() 会换行。为了让所有数字排成一行，我们需要修改结尾符。
✅ 写法：print(计算结果, end=" ")
这表示打印完数字后，不换行，而是打印一个空格。

⚔️ C++ 解题代码 (While版)

注意乘法运算和循环变量的更新。

#include <iostream>
using namespace std;

int main() {
    // 1. 初始化变量，从速度 1 开始
    int i = 1;
    
    // 2. 只要速度小于等于 100，就继续计算
    while(i <= 100) {
        // 3. 输出 i 的平方 (i * i) 和一个空格
        cout << i * i << ' ';
        
        // 4. 速度加 1，准备计算下一个
        // ⚠️ 千万别忘了这行，否则会死循环！
        i++; 
    }
   
    return 0;
}

🐍 Python 解题代码 (While版)

使用 end=" " 来实现单行输出。

# 1. 初始化变量，从速度 1 开始
a = 1

# 2. 只要速度小于等于 100，就继续计算
while a <= 100:
    # 3. 计算平方并输出，end=" " 表示不换行，用空格隔开
    print(a * a, end=" ")
    
    # 4. 速度加 1
    # ⚠️ 必须手动更新，否则程序停不下来
    a += 1

🌟 魔法笔记 (重点复习)

  • ⚡ 动能公式： 题目说“动能是速度的平方”，代码里就是 i * i。
  • 🔄 While 陷阱： 再次提醒，while 循环里必须有让变量变化的代码 (i++ 或 a+=1)，否则你会看到满屏幕的 "1 1 1 1..." 停不下来！
  • 📏 空格分隔： 题目要求“用空格分开”，所以每次输出数字后都要紧跟一个空格。
