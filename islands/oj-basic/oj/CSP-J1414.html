⚙️ 商人的批发规则 (题目描述)

年轻的英雄正在为士兵们集结装备。 神秘的装备商人给出了一个诱人的“批发价”规则：

  • 💎 普通价： 如果买的不多（不超过 100 套），每套 80 宝石。
  • 💎 批发价： 如果买很多（超过 100 套），每套只要 50 宝石！

👉 任务： 读入订购的套数，计算英雄一共需要支付多少宝石。

🧠 魔法原理 (解题思路)

这是一道典型的 分段计算 题目。价格取决于数量。

⚖️ 价格天平

条件： x > 100

  • ✅ 成立 (True)： 单价 = 50
  • ❌ 不成立 (Else)： 单价 = 80

最终计算： 总价 = 单价 * x

📘 C++ 的分段计算

  • ⚖️ 判断： 使用 if (x > 100)。
  • 💰 计算： 可以直接在 cout 中计算并输出，例如 cout << x * 50;。

📒 Python 的分段计算

  • ⚖️ 判断： 使用 if x > 100:。
  • 💰 计算： 可以算出一个总价变量 total，最后再打印。

💻 咒语吟唱 (代码详解)

士兵们正在等待装备，快写代码帮英雄付款吧！

#include <iostream>
using namespace std;

int main() {
    // 1️⃣ 声明变量 x (套数)
    int x;
    
    // 2️⃣ 输入套数
    cin >> x;
    
    // 3️⃣ 判断数量并计算
    if (x > 100) {
        // 超过100套，享受批发价 50
        cout << 50 * x;
    } 
    else {
        // 没超过100套，原价 80
        cout << 80 * x;
    }
    
    return 0;
}

# 1️⃣ 输入：读取套数，存入变量 x
x = int(input())

# 2️⃣ 判断数量并计算
if x > 100:
    # 超过100套，单价 50
    total = 50 * x
else:
    # 没超过，单价 80
    total = 80 * x

# 3️⃣ 输出总价
print(total)

⭐ 数学小提醒： 买 101 套比买 90 套还要便宜哦！(101*50=5050, 90*80=7200)。这就是批发的魔力！

🌟 魔法笔记 (重点复习)

  • ⚖️ 条件分支： 根据不同的情况（数量多 vs 数量少），执行不同的计算公式。
  • 🔢 临界点： 题目说“超过 100”，所以用 > 100。如果是 100 套整，还是按 80 算哦！
