📜 遗迹使命 (题目描述)

在沙漠遗迹里，英雄发现了一本“星辰密码书”📘。
书里说：有些数字星星总是成对出现，它们 相差 2，而且两颗星都必须是 素数（只能被 1 和自己整除，且大于 1）。
这种神秘搭档叫做 素数对✨，比如 (5,7)、(17,19)。

现在，遗迹之门给了你一个数字 n (1 ≤ n ≤ 100)：

如果一个都找不到，请输出：empty。

输入样例 1： 10
输出样例 1：
3 5
5 7

输入样例 2： 3
输出样例 2：
empty

🧠 魔法原理 (Algorithm Logic)

寻找“双星素盟”需要分为两步来施展魔法：

🔍 第一步：真·素数星检测 (判断素数)
我们需要写一个判断素数的小函数 isPrime(m)。
为了让魔法生效更快，我们不需要一直除到 m-1，只需要循环试除到 √m 即可（在代码中可以写成 i * i <= m）。

🔗 第二步：配对与标记 (寻找相差为2的搭档)
我们让变量 i 从 2 一直遍历到 n-2。
如果 i 是素数，并且 i+2 也是素数，那么它们就是一对！
同时，我们需要一个 标记变量 (Flag)。如果找到了一对，就把标记设为 1。遍历结束后，如果标记还是 0，说明一对都没找到，输出 empty。

⚔️ C++ 挑战：补全遗迹法阵

这是一个填空题！请看原题中缺失了【①】到【⑤】的魔法铭文，下面是完整的解答：

  • 【①】填 m % i == 0 ：如果能被整除，说明不是素数。
  • 【②】填 0 ：初始化标记变量 bj 为 0（代表还没找到）。
  • 【③】填 isPrime(i) ：检查第一个数是否为素数。
  • 【④】填 isPrime(i+2) ：检查第二个数是否为素数。
  • 【⑤】填 bj == 0 ：如果循环结束标记还是 0，说明一个都没找到。
#include <iostream>
using namespace std;

int a = 3, b = 5; // 遗迹中预留的变量，本题逻辑中并未使用

// 魔法函数：判断是否为素数
bool isPrime(int m){
    for(int i = 2; i * i <= m; i++)
        if(m % i == 0)  // 【①】如果能被整除，就不是素数
            return false;
    return true;
}

int main(){
    int n, bj = 0; // 【②】bj是标记(标记是否找到了素数对)
    cin >> n;
    
    // 遍历寻找素数对，保证 i+2 不超过 n
    for(int i = 2; i <= n - 2; i++){
        if(isPrime(i) && isPrime(i+2)){ // 【③】和【④】两个都是素数
            cout << i << ' ' << i + 2 << endl;
            bj = 1; // 找到了，修改标记
        }
    }
    
    if(bj == 0) // 【⑤】如果一整圈找下来都没有
        cout << "empty";
        
    return 0;   
}

🐍 Python 解题代码

Python 试炼需要我们自己写出完整版代码。逻辑与 C++ 完全一致！

注意：Python 中的 range(a, b) 是不包含 b 的。为了让 i 能够取到 n-2，我们的右边界要写成 n-1 哦！

def is_prime(m):
    # 判断素数：循环到根号m的整数部分
    for i in range(2, int(m**0.5) + 1):
        if m % i == 0:
            return False
    return True

n = int(input())
bj = 0 # 标记变量

# 遍历范围：从 2 到 n-2 (包含n-2，所以range填n-1)
for i in range(2, n - 1):
    if is_prime(i) and is_prime(i + 2):
        print(i, i + 2)
        bj = 1 # 找到了，修改标记

# 如果标记还是 0，说明没找到
if bj == 0:
    print('empty')

🌟 魔法笔记 (重点复习)

  • ⚡ 素数判断优化： 不需要从 2 循环到 m-1。如果一个数 m 有因数，它肯定有一个因数是 ≤ √m 的！所以循环条件写 i * i <= m 大大加快了运行速度！
  • 🚩 Flag 标记法： 编程中经常遇到“如果找到了就输出，如果一个都没找到就输出无”的问题。设立一个初始为 0 的 bj 变量是最经典的套路。
  • 🛑 边界条件： 要求两颗星都不大于 n。既然第一颗星是 i，第二颗星是 i+2，我们只要保证 i+2 ≤ n 即可。这就推导出循环的终点应该是 i ≤ n-2。
