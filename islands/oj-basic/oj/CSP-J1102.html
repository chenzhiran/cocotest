📘 C++ 的魔法道具：cout

  • 🎒 字符串 (String)： 用双引号 " " 包裹起来的一串文字。里面可以放英文、数字，也可以放中文！
  • 📢 cout << ...：继续使用我们的大喇叭。只要把这段中英文混合的文字放在 << 后面，大喇叭就能把它喊出来。
  • 🔚 endl：别忘了养成好习惯，说完话按一下"回车键"换行哦。

📒 Python 的魔法道具：print

  • 🖨️ print(...)：Python 的打印机非常强大，它不挑食，英文中文都能打印。
  • 📜 引号： 只要用引号 ' ' 或 " " 把它们包起来，Python 就会知道这是一段完整的文字（字符串）。
  • ✨ 混合双打： "Hello" 是英文，"扣哒世界" 是中文，放在一起完全没问题！

🌟 魔法笔记 (重点复习)

  • 🧩 中英混合： 字符串里可以同时包含英文单词和中文字符，电脑都能听懂。
  • 🎯 精准匹配： 输出的内容必须和题目要求的一模一样，多一个空格、错一个符号都打不开门。
  • 🧱 结构完整： C++ 程序必须有头文件、main 函数和花括号 {}。
  • ⚡ 简洁有力： Python 程序通常很短，但每一个缩进和符号都很重要。
