🧠 魔法原理 (Logic)

想要消灭那些讨厌的“复读机”重复数据，我们有两个非常强大的编程武器可以挑选：

🧙‍♂️ 武器一：投石入桶法 (Bucket Technique)
由于题目透露魔力值都在 100 以内，我们可以准备 100 个小桶。
捡到一个数字 x，就在第 x 号桶里插一面红色旗子（标志值变为 1）。
不管同一个数字捡到多少次，对应的那个小桶也永远只有一面小旗子。最后数一数有几面小旗子即可！
✨ 武器二：自动净化集合法 (Set Structure)
如果你嫌准备桶太麻烦，还可以直接召唤高级数据结构：集合 (Set)。
集合就像是一个具有“自动蒸发重复项”功能的神秘收纳盒！
你只管把所有捡到的魔法石扔进去，只要集合发现里面已经有相同的魔力值了，就会自动把重复的魔法石融化蒸发，永远只留下一块。最后数数收纳盒里剩下几块石头就可以啦！

C++ 两种实现细节

解法一 (桶标记)： 定义一个数组 int book = {0};，读入 x 时让 book[x] = 1;，最后循环统计 1 的个数。
解法二 (Set集合)： 引入 #include <set>，创建一个集合 set<int> s;，利用 s.insert(x); 塞入元素。最后直接通过 s.size() 拿到种类数！

Python 两种实现细节

解法一 (桶标记)： 创建一个初始全为 0 的列表：book = * 105，每次标记 book[x] = 1，最后数一数有多少个 1。
解法二 (Set集合)： 读取整行魔力值存成普通的列表后，直接使用内置魔法命令 set(nums)，一行代码瞬间完成去重。最后使用 len() 测量大小即可！

⚔️ C++ 解题代码

解法一：经典桶标记法

使用 book 数组（桶）来记录数字是否出现过。

#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;

    // 1. 准备 105 个桶，全部初始化为 0
    // 题目说数字不超过 100，所以 105 足够了
    int book[105] = {0};

    // 2. 读入数字并标记
    for (int i = 0; i < n; i++) {
        int x;
        cin >> x;
        
        // 在第 x 号桶做记号，表示这个数字出现过
        book[x] = 1;
    }

    int cnt = 0; // 计数器

    // 3. 数一数有多少个桶被标记了
    // 只需要遍历可能的数字范围 1 到 100
    for (int i = 1; i <= 100; i++) {
        if (book[i] == 1) {
            cnt++;
        }
    }

    cout << cnt << endl;

    return 0;
}

解法二：神奇集合(std::set)自动去重

利用 C++ STL 的 set 容器，丢进去的重复数字会自动合二为一，非常省心！

#include <iostream>
#include <set> // 引入集合头文件
using namespace std;

int main() {
    int n;
    cin >> n;

    // 1. 创建一个叫 s 的魔力整数集合
    set<int> s;

    // 2. 循环读入数据
    for (int i = 0; i < n; i++) {
        int x;
        cin >> x;
        
        // 3. 施加丢入魔法：重复数据放入时会被自动略过
        s.insert(x);
    }

    // 4. 集合的 size() 就是留在里面的不重复石头的种类数
    cout << s.size() << endl;

    return 0;
}

🐍 Python 解题代码

解法一：经典桶标记列表

用列表当桶，简单高效。

# 1. 读入 n
n = int(input())

# 2. 读入所有数字
nums = list(map(int, input().split()))

# 3. 准备桶：创建一个全为 0 的列表，长度 105
book = [0] * 105

# 4. 标记
for x in nums:
    # 把第 x 号桶标记为 1
    book[x] = 1

# 5. 统计
ans = 0
# 遍历所有可能的桶 (0到100)
for i in range(101):
    if book[i] == 1:
        ans += 1

print(ans)

借助 Python 原生的 set 数据类型，一句话让重复元素全部“蒸发”。

unique_stones = set(nums)

🌟 魔法笔记 (重点复习)

  • 🗑️ 桶思想： 当数字范围比较小（比如 100 以内）时，用数组下标直接代表数字，是去重最快的方法。
  • 🚩 标记法： book[x] = 1 表示 x 出现过。无论出现多少次，我们只关心它是否“存在”。
  • 📦 集合魔法（Set）： 集合最擅长的事情就是“自动去重”。如果你不想手动开辟大数组标记，可以用集合来帮你把重复的数据自动消灭掉。
