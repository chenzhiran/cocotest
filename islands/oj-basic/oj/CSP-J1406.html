炎炎夏日，安雅和安小风买了一个大西瓜 🍉，重量是 n 斤。 他们想把西瓜切成两半，但有一个奇怪的要求：

👉 两个人分到的重量，都必须是偶数斤！
（比如：2斤和4斤是OK的，但1斤和5斤就不行）

👉 任务： 读入西瓜的总重量，判断能不能切成两个偶数。 如果可以，输出 Yes；否则输出 No。

🧠 魔法原理 (解题思路)

我们来分析一下数学规律：

  • 偶数 + 偶数 = 偶数
  • 奇数 + 奇数 = 偶数
  • 奇数 + 偶数 = 奇数

所以，西瓜总重量 必须是偶数，才能分成两份一样的性质（要么两份都偶，要么两份都奇）。 但我们要的是“两份偶数”，所以总重量一定要是偶数。

⚠️ 特殊陷阱：数字 2

如果西瓜只有 2斤 呢？

2 = 1 + 1 (两个奇数 ❌)

2 = 0 + 2 (0不是正数，不能算分到了西瓜 ❌)

所以，2 是不行的！ 必须是 大于 2 的偶数 (也就是 >= 4)。

🗝️ 通关密码

( n > 2 ) AND ( n % 2 == 0 )

📘 C++ 的判断魔法

  • 🔐 双重条件： 使用 && 连接。
  • 🔢 代码： if (a > 2 && a % 2 == 0) 或者 if (a >= 4 && a % 2 == 0)。

📒 Python 的判断魔法

  • 🔐 双重条件： 使用 and 连接。
  • 🔢 代码： if n > 2 and n % 2 == 0:。

💻 咒语吟唱 (代码详解)

快帮安雅判断西瓜能不能切吧！

#include <iostream>
using namespace std;

int main() {
    // 1️⃣ 声明变量 a (西瓜重量)
    int a;
    
    // 2️⃣ 输入重量
    cin >> a;
    
    // 3️⃣ 双重判断
    // 条件1：必须至少4斤 (a >= 4)，也就是大于2
    // 条件2：必须是偶数 (a % 2 == 0)
    if (a >= 4 && a % 2 == 0) {
        cout << "Yes"; // 可以切分
    } 
    else {
        cout << "No";  // 不行
    }
    
    return 0;
}

# 1️⃣ 输入：读取西瓜重量，存入变量 n
n = int(input())

# 2️⃣ 双重判断
# 条件1：必须是偶数
# 条件2：必须大于 2 (因为2只能分成1+1)
if n % 2 == 0 and n > 2:
    print("Yes")  # 可以切分
else:
    print("No")   # 不行

⭐ 思考： 为什么 10 可以？因为 10 = 2 + 8 (偶+偶)，或者 4 + 6 (偶+偶)。只要总数是大于2的偶数，就一定能切出 2 斤和其他偶数斤！

🌟 魔法笔记 (重点复习)

  • 🍉 特例： 数字 2 虽然是偶数，但不能分成两个正偶数，这是一个大陷阱！
  • 🔐 逻辑与： 必须同时满足“偶数”和“大于2”两个条件。
  • ➗ 偶数判断： 永远记得 % 2 == 0。
