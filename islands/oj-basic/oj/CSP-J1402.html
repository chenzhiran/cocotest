🍿 影院的规矩 (题目描述)

在冰冷的雪山影院 🏔️，一群可爱的食人魔想看电影。 为了让后排的观众不被挡住，影院有一个严格的规定：

每列的 3 个食人魔，必须按身高从低到高排列！
也就是：(最矮) < (中等) < (最高)

👉 任务： 你是检票员，读入 3 个身高，判断它们是不是已经排好了顺序。 如果是，输出 YES；否则输出 NO。

🧠 魔法原理 (解题思路)

这道题其实是检查：a 是否小于 b，并且 b 是否小于 c。

🔗 逻辑链条

( a < b ) AND ( b < c )

必须 同时 满足两个条件，才算排好序。

📘 C++ 的逻辑陷阱

  • ⚠️ 警告： 在 C++ 里，a < b < c 是错误的写法！计算机读不懂连续的比较。
  • 🔗 正确写法： 必须把它们拆开，用 逻辑与 (&&) 连接起来。
    写法：if (a < b && b < c)
  • 💡 含义： "a 小于 b" 并且 "b 小于 c"。

📒 Python 的逻辑魔法

  • ✨ 连续比较： Python 很聪明，它允许直接写 a < b < c。这在数学上叫不等式链。
  • 🔗 逻辑： 它自动帮你检查了是否每一个数都比前一个大。

💻 咒语吟唱 (代码详解)

检票员，请按下你的判断按钮吧！

#include <iostream>
using namespace std;

int main() {
    // 1️⃣ 声明变量：a, b, c 表示三个身高
    int a, b, c;
    
    // 2️⃣ 输入：读入三个整数
    cin >> a >> b >> c;
    
    // 3️⃣ 判断顺序 (从小到大)
    // ⚠️ 必须用 && 连接两个条件
    if (a < b && b < c) {
        cout << "YES"; // 符合规定
    } 
    else {
        cout << "NO";  // 不符合规定
    }
    
    return 0;
}

# 1️⃣ 输入：读取三个身高
a, b, c = map(int, input().split())

# 2️⃣ 判断顺序
# Python 可以直接连着写比较
if a < b < c:
    print("YES")  # 符合规定
else:
    print("NO")   # 不符合

⭐ C++ 小贴士： 千万别写成 if (a < b < c)！这在 C++ 里是语法错误或逻辑错误。一定要用 && 分开写！

🌟 魔法笔记 (重点复习)

  • 🔗 逻辑与 (&&)： 只有当两边的条件 都为真 时，结果才为真。
  • 🚫 C++ 禁忌： 禁止连续比较 (如 3 < x < 5)，必须拆分为 x > 3 && x < 5。
  • 🍿 排序： 从小到大意味着前一个数必须严格小于后一个数。
