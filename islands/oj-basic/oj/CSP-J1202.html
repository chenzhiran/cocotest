📘 C++ 的小数运算：double

  • 📦 变量： 因为面积和半径通常都带有小数点，所以我们要使用 double 类型来声明变量 r (半径) 和 s (面积)。
  • 🔢 乘法： r * r 表示半径的平方，千万别写成 r * 2 哦！

📒 Python 的小数运算：float

  • 📦 变量： r 用来存半径，s 用来存计算出来的面积。
  • 🔢 公式： 把 r * r * 3.14 的结果算出来，交给 print 打印。

💻 咒语吟唱 (代码详解)

英雄拿起了魔法计算机，快帮他写出代码吧！

#include <iostream>
using namespace std;

int main()
{
    // 1️⃣ 声明变量：r是半径，s是面积
    // 使用 double 来存放带小数的数字
    double r, s;

    // 2️⃣ 输入：读取花坛的半径
    cin >> r;

    // 3️⃣ 计算：面积 = 半径 * 半径 * 3.14
    // ⚠️ 注意：是 r * r，不是 r * 2
    s = r * r * 3.14;

    // 4️⃣ 输出：打印出面积
    cout << s;
    
    return 0;
}

# 1️⃣ 输入：读取半径，并转为小数 (float)，存入变量 r
r = float(input())

# 2️⃣ 计算：面积 = 半径 * 半径 * 3.14
# ⚠️ 记得用乘号 * 把它们连起来，存入变量 s
s = r * r * 3.14

# 3️⃣ 输出：打印变量 s (面积)
print(s)

⭐ 易错点提醒： 一定要区分 半径的平方 ($r \times r$) 和 半径的两倍 ($r \times 2$)。算面积用的是平方哦！

🌟 魔法笔记 (重点复习)

  • ⭕ 圆面积公式： $S = \pi \times r \times r$。
  • 🔢 变量名： 我们统一用 r 代表半径，s 代表面积，这样代码更清晰。
