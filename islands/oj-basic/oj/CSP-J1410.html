🎉 广场上的变换队形 (题目描述)

丰收的季节，农民们在广场上开晚会 🌾。 最受欢迎的节目是集体舞蹈，这个舞蹈非常神奇，可以变换三种队形：

  • 3️⃣ 人一组
  • 5️⃣ 人一组
  • 7️⃣ 人一组

只有当人数 同时能被 3、5、7 整除 时，舞蹈才能顺利变换，每个人都有位置！

👉 任务： 读入人数，判断它是否同时是 3、5、7 的倍数。 如果是，输出 Yes；否则输出 No。

🕺 人数 n
15
%3==0 ✅
%5==0 ✅
%7==0 ❌
➡️
❌ 结果
No
🕺 人数 n
105
%3==0 ✅
%5==0 ✅
%7==0 ✅
➡️
✅ 结果
Yes

🧠 魔法原理 (解题思路)

我们要通过 三道门禁，必须全部通过才能出发。

只要使用 逻辑与 (AND) 魔法，把三个条件串成一串糖葫芦！

🔒 三重锁定

(n % 3 == 0) AND (n % 5 == 0) AND (n % 7 == 0)

三个条件缺一不可！

📘 C++ 的逻辑链

  • 🔐 连接符： 使用 && 连接所有条件。
  • 📝 写法： if (n%3==0 && n%5==0 && n%7==0)。

📒 Python 的逻辑链

  • 🔐 连接词： 使用 and 连接所有条件。
  • 📝 写法： if n%3==0 and n%5==0 and n%7==0:。

💻 咒语吟唱 (代码详解)

晚会即将开始，快写代码指挥大家排队吧！

#include <iostream>
using namespace std;

int main() {
    // 1️⃣ 声明变量 n (人数)
    int n;
    
    // 2️⃣ 输入人数
    cin >> n;
    
    // 3️⃣ 三重判断
    // 必须同时被3、5、7整除
    // 使用 && 把三个条件连起来
    if (n % 3 == 0 && n % 5 == 0 && n % 7 == 0) {
        cout << "Yes"; // 可以灵活变形
    } 
    else {
        cout << "No";  // 人数不合适
    }
    
    return 0;
}

# 1️⃣ 输入：读取人数，存入变量 n
n = int(input())

# 2️⃣ 三重判断
# 必须同时被3、5、7整除
# 使用 and 把三个条件连起来
if n % 3 == 0 and n % 5 == 0 and n % 7 == 0:
    print("Yes")  # 可以灵活变形
else:
    print("No")   # 人数不合适

⭐ 数学小奥秘： 如果一个数同时是 3、5、7 的倍数，那它一定是 3 * 5 * 7 = 105 的倍数哦！

🌟 魔法笔记 (重点复习)

  • 🔐 多重与运算： 可以无限连接 条件1 && 条件2 && 条件3 ...。
  • ➗ 整除判断： x % n == 0。
  • 💃 逻辑清晰： 所有条件都满足，才能说 Yes。
