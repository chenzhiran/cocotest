💎 英雄的最终抉择 (题目描述)

在地牢的最深处 🏰，英雄发现了并排摆放的 三个宝箱！ 它们分别刻着字母 a、b 和 c。

英雄的体力只能带走一个。为了获得最大的价值，他决定：
👉 带走最重（数值最大）的那一个！

👉 任务： 读入三个整数，找出它们中的 最大值 并输出。

📦 a
3
📦 b
4
📦 c
5
⬇️ 寻找最大值 ⬇️
🏆 结果
5

🧠 魔法原理 (解题思路)

这次有三个对手，怎么比呢？我们可以采用 “擂台赛” 的方法！

🥊 第一轮：a 和 b PK

如果 a > b，那 a 赢了，b 淘汰。
接下来 a 和 c 比。

如果 a 不大于 b，那 b 赢了，a 淘汰。
接下来 b 和 c 比。

这在编程里叫做 “嵌套分支”，就是一个 if 里面还包着另一个 if。

📘 C++ 的嵌套魔法

  • 🏗️ 结构： 
    if (a > b) {
    if (a > c) ...
} else {
    if (b > c) ...
}
  • 逻辑就像剥洋葱，一层一层地比较，最后留下的就是最大的。

📒 Python 的嵌套魔法

  • 🏗️ 结构： 
    if a > b:
    if a > c:
        ...
else:
    if b > c:
        ...
  • 注意 缩进！里面的 if 要比外面的 if 多缩进一格，表示它是在上一层条件成立时才执行的。

💻 咒语吟唱 (代码详解)

英雄正在等待最终的判断，快写代码帮他带走最大的宝箱！

#include <iostream>
using namespace std;

int main() {
    // 1️⃣ 声明变量：a, b, c 表示三个宝箱
    int a, b, c;
    
    // 2️⃣ 输入：一次读入三个重量
    cin >> a >> b >> c;
    
    // 3️⃣ 第一轮 PK：a 和 b 比
    if (a > b) {
        // 🏆 a 赢了，现在是 a 和 c 争冠军
        if (a > c) {
            cout << a << endl; // a 最大
        } 
        else {
            cout << c << endl; // c 比 a 还大，c 最大
        }
    } 
    else {
        // 🏆 b 赢了 (或者 a=b)，现在是 b 和 c 争冠军
        if (b > c) {
            cout << b << endl; // b 最大
        } 
        else {
            cout << c << endl; // c 比 b 还大，c 最大
        }
    }
    
    return 0;
}

# 1️⃣ 输入：读取三个整数
a, b, c = map(int, input().split())

# 2️⃣ 第一轮 PK：a 和 b 比
if a > b:
    # 🏆 a 赢了，现在 a 和 c 比
    if a > c:
        print(a)  # a 最大
    else:
        print(c)  # c 最大
else:
    # 🏆 b 赢了，现在 b 和 c 比
    if b > c:
        print(b)  # b 最大
    else:
        print(c)  # c 最大

⭐ 小思考： 如果 a, b, c 都是 5，程序会怎么跑？
它会走 else 分支，最后输出 5。结果也是正确的！

🌟 魔法笔记 (重点复习)

  • 🥊 擂台法： 多个数比较时，先让两个比，赢的那个再去和剩下的比。
  • 📦 嵌套分支： if 里面套 if，用来处理更复杂的逻辑关系。
  • 👀 注意细节： 每一层花括号（C++）或缩进（Python）都要对齐，不然容易迷路哦！
