🤕 英雄的急救包 (题目描述)

在激烈的战斗中 ⚔️，英雄受伤了，身体非常虚弱。 忠诚的宠物 🐾 跑去药房找药水。柜子上有三个药水瓶，大小各不相同。

宠物知道一个秘密：
👉 药效最好的药水，装在最小的瓶子里！

👉 任务： 读入三个整数（药水瓶大小），找出其中的 最小值，帮助英雄尽快恢复！

🧪 a
5
🧪 b
6
🧪 c
7
⬇️ 寻找最小值 ⬇️
🌟 结果
5

🧠 魔法原理 (解题思路)

这道题和“找最大值”非常像，只不过这次我们要找的是 最小 的。

我们依然使用 “擂台赛” 的方法，只不过这次比的是“谁更小”。

📉 第一轮：a 和 b 比小

如果 a < b，那 a 比较小，b 淘汰。
接下来 a 和 c 比。

如果 a 不小于 b，那 b 比较小，a 淘汰。
接下来 b 和 c 比。

📘 C++ 的嵌套魔法

  • ⚖️ 符号： 找最大用 >，找最小用 <。
  • 🏗️ 结构： 
    if (a < b) {
    if (a < c) ...
} else {
    if (b < c) ...
}

📒 Python 的嵌套魔法

  • ⚖️ 符号： 同样使用小于号 <。
  • 🏗️ 结构： 
    if a < b:
    if a < c:
        ...
else:
    if b < c:
        ...

💻 咒语吟唱 (代码详解)

宠物正焦急地等待，快用代码找出救命药水！

#include <iostream>
using namespace std;

int main() {
    // 1️⃣ 声明变量：a, b, c 表示三个药水瓶
    int a, b, c;
    
    // 2️⃣ 输入：一次读入三个大小
    cin >> a >> b >> c;
    
    // 3️⃣ 第一轮 PK：看 a 和 b 谁小
    if (a < b) {
        // 📉 a 更小，现在看 a 和 c 谁小
        if (a < c) {
            cout << a; // a 最小
        } 
        else {
            cout << c; // c 比 a 还小，c 最小
        }
    } 
    else {
        // 📉 b 更小 (或相等)，现在看 b 和 c 谁小
        if (b < c) {
            cout << b; // b 最小
        } 
        else {
            cout << c; // c 比 b 还小，c 最小
        }
    }
    
    return 0;
}

# 1️⃣ 输入：读取三个整数
a, b, c = map(int, input().split())

# 2️⃣ 第一轮 PK：a 和 b 比小
if a < b:
    # 📉 a 比较小，现在 a 和 c 比
    if a < c:
        print(a)  # a 最小
    else:
        print(c)  # c 最小
else:
    # 📉 b 比较小，现在 b 和 c 比
    if b < c:
        print(b)  # b 最小
    else:
        print(c)  # c 最小

⭐ 对比学习： 这道题的代码结构和上一题（找最大值）一模一样，唯一的区别就是把 > 换成了 <！

🌟 魔法笔记 (重点复习)

  • 📉 找最小值： 使用 < (小于号) 进行比较。
  • 📦 嵌套逻辑： 先比较两个，胜者（较小者）再和第三个比较。
  • 🐾 细心： 不要把 < 写反成 > 哦！
