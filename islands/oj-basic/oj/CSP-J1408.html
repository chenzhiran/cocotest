💐 浪漫森林篇 · 闰年判断

💍 特别的纪念日 (题目描述)

在幽深的森林里 🌲，生活着一对幸福的食人魔夫妇。 他们的结婚纪念日非常特别，是 2月29日！

这意味着，他们只能在 闰年 才能过纪念日，收到鲜花和礼物 💐。 为了不再错过每一个纪念日，他们决定用编程来算出哪一年是闰年。

👉 任务： 读入一个年份，判断它是不是闰年。 如果是，输出 是闰年；否则输出 不是闰年。

📅 年份 y
2016
❓ 判断
💖 结果
是闰年
📅 年份 y
2100
❓ 判断
🌑 结果
不是闰年

🧠 魔法原理 (解题思路)

判断闰年的规则有点复杂，需要用到 组合逻辑。规则有两个，满足其中 任意一个 就是闰年：

📜 规则一：普通闰年

能被 4 整除 并且 不能被 100 整除。

(y % 4 == 0) AND (y % 100 != 0)

—— 或者 (OR) ——

📜 规则二：世纪闰年

能被 400 整除。

(y % 400 == 0)

📘 C++ 的逻辑积木

  • 🧩 并且 (AND)： 符号是 &&。
  • 🧩 或者 (OR)： 符号是 || (按住Shift键+回车键上方那个键)。
  • 🧩 不等于： 符号是 !=。
  • 🏗️ 组合： (规则1) || (规则2)。

📒 Python 的逻辑积木

  • 🧩 并且 (AND)： 单词 and。
  • 🧩 或者 (OR)： 单词 or。
  • 🧩 不等于： 符号 !=。
  • 🏗️ 组合： (规则1) or (规则2)。

💻 咒语吟唱 (代码详解)

食人魔夫妇正在等待你的计算，快帮他们找出纪念日吧！

#include <iostream>
using namespace std;

int main() {
    // 1️⃣ 声明变量 y (年份)
    int y;
    
    // 2️⃣ 输入年份
    cin >> y;
    
    // 3️⃣ 组合判断
    // 规则1：能被4整除 且 不能被100整除
    // 规则2：能被400整除
    // 两个规则满足一个即可，中间用 || (或者) 连接
    if ((y % 4 == 0 && y % 100 != 0) || (y % 400 == 0)) {
        cout << "是闰年" << endl;
    } 
    else {
        cout << "不是闰年" << endl;
    }
    
    return 0;
}

# 1️⃣ 输入：读取年份，存入变量 y
y = int(input())

# 2️⃣ 组合判断
# (能被4整除 and 不能被100整除) or (能被400整除)
if (y % 4 == 0 and y % 100 != 0) or (y % 400 == 0):
    print("是闰年")
else:
    print("不是闰年")

⭐ 易错点： 2100年虽然能被4整除，但它能被100整除且不能被400整除，所以它是平年！这就是为什么要有 % 100 != 0 这个条件。

🌟 魔法笔记 (重点复习)

  • 🧩 逻辑与： && / and (两边都要对)。
  • 🧩 逻辑或： || / or (只要有一边对就行)。
  • 📅 口诀： 四年一闰，百年不闰，四百年再闰。
