🧠 魔法原理 (解题思路)

如何告诉计算机什么是“偶数”呢？我们要用到一个超级好用的运算符：取余 (%)。

偶数 (Even)

能被 2 整除

a % 2 == 0

除以2，余数是0

奇数 (Odd)

不能被 2 整除

a % 2 != 0

除以2，余数是1

📘 C++ 的奇偶魔法

  • ➗ 取余号 %： 这个符号不是百分号哦！它是用来求余数的。比如 5 % 2 结果是 1。
  • ⚖️ 等于号 ==： 判断两个东西是否相等，要用两个等号！if (a % 2 == 0) 就是在问：“a除以2的余数是0吗？”

📒 Python 的奇偶魔法

  • ➗ 取余号 %： 也是用这个符号！a % 2 会算出 a 除以 2 剩下的数。
  • ⚖️ 等于号 ==： 记得用双等号 == 来做比较。if a % 2 == 0: 意思是“如果a是偶数”。

💻 咒语吟唱 (代码详解)

帮农民伯伯算算能不能出发吧！

#include <iostream>
using namespace std;

int main() {
    // 1️⃣ 声明变量 a (人数)
    int a;
    
    // 2️⃣ 输入人数
    cin >> a;
    
    // 3️⃣ 判断奇偶
    // a % 2 == 0 意思是：a 除以 2 余数为 0
    if (a % 2 == 0) {
        cout << "Yes"; // 偶数，可以出发
    } 
    else {
        cout << "No";  // 奇数，不能出发
    }
    
    return 0;
}

# 1️⃣ 输入：读取人数，转为整数存入 a
a = int(input())

# 2️⃣ 判断奇偶
# a % 2 == 0 意思是：a 除以 2 余数为 0
if a % 2 == 0:
    print("Yes")  # 偶数
else:
    print("No")   # 奇数

⭐ 秘诀： 凡是判断奇数偶数，永远记得找 % 2 帮忙！

🌟 魔法笔记 (重点复习)

  • ➗ 取余运算： % 用来计算除法剩下的数。
  • 🚌 偶数判断： x % 2 == 0。
  • 🚫 奇数判断： x % 2 == 1 (或者 != 0)。
