🧠 魔法原理 (解题思路)

想要模仿别人说话，总共分三步：1. 听到耳朵里 -> 2. 记在脑子里 -> 3. 用嘴巴说出来。

📘 C++ 的"听"与"说"

1. 准备记忆空间 (string)：
我们需要一个能装文字的变量，类型是 string。
2. 听进去 (cin)：
用 cin >> a 把英雄说的话读进来，存到变量 a 里。
3. 说出来 (cout)：
用 cout << a 把变量 a 里的内容原封不动地送回屏幕。

📒 Python 的"听"与"说"

1. 听并记住 (input)：
input() 负责听英雄说话，然后用等号 = 把它存进变量 a。
2. 说出来 (print)：
print(a) 负责把 a 里面记住的话，大声地打印在屏幕上。

💻 咒语吟唱 (代码详解)

这段代码非常短，但是非常神奇哦！

#include <iostream>  // 🎒 引入工具箱
using namespace std; // 🏷️ 使用标准名字

int main() {
    // 1️⃣ 准备一个字符串变量，用来装话
    string a; 

    // 2️⃣ 听：从键盘读入英雄说的话
    cin >> a; 

    // 3️⃣ 说：把刚才听到的话输出来
    cout << a;
    
    return 0;
}

# 1️⃣ 听：接收输入，并存到变量 a 里
a = input()

# 2️⃣ 说：打印出变量 a 的内容
print(a)

# 🦜 是不是很简单？鹦鹉的任务完成啦！

⭐ 秘诀： 这里的关键是"原封不动"。不需要加 Hello，也不需要加感叹号，变量里有什么，就输出什么！

🌟 魔法笔记 (重点复习)

  • 📥 输入 (Input)： C++ 用 cin >> Python 用 input()，这是程序在"听"。
  • 📤 输出 (Output)： C++ 用 cout << Python 用 print()，这是程序在"说"。
  • 📦 变量： 就像鹦鹉的脑子，必须先记住（存入变量），才能复述（输出变量）。
