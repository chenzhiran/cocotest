一、复习回顾：两种While循环的异同

在解锁新技能之前，我们先来回顾一下之前的两位老朋友。它们都是用来处理“重复”任务的，但性格不同。

While 循环
👮‍♂️

绰号：严格的门卫

👉 先检查条件，再放行。
👉 如果一开始条件就不满足，一次都不执行。

Do-While 循环
🏃

绰号：热情的推销员

👉 先试用（执行）一次，再决定要不要继续。
👉 无论如何，至少执行一次。

二、For循环：最完美的循环结构

有没有觉得写 while 的时候，把初始值、条件、步长写在三个不同的地方，很容易漏掉？
For 循环把这“循环三要素”打包在了一起，像一个高度集成的自动化机器！

🪄 For 循环语法解析
🧙‍♂️ 老师口诀：
For 循环，三兄弟，住在一个括号里。
老大定起点，老二看终点，老三迈步子。
顺序是：1 → 2 → 3 → 4 → 2 → 3 → 4 ...
注意：老大只工作一次哦！ 
// 打印 1 到 5
for (int i = 1; i <= 5; i++) {
    cout << i << " ";
}
// 输出：1 2 3 4 5

三、流程控制：Break与Continue的交通指挥

循环有时候像一辆停不下来的车，我们需要“交通指挥”来控制它。这两个命令在 while 和 for 里通用！

🛑

Break (终止)

含义： 彻底结束循环，跳出跑道，不玩了！

生活场景： 在操场跑步，突然下大雨了，立刻停止跑步，回教室。

⏭️

Continue (继续)

含义： 结束本次循环，直接跳到步骤4（更新步长），开始下一圈。

生活场景： 老师点名，喊到请假的同学，跳过他，继续喊下一个。

// 游戏规则：打印 1~10
// 遇到 4 的时候跳过 (Continue)
// 遇到 8 的时候游戏结束 (Break)

for (int i = 1; i <= 10; i++) {
    if (i == 4) {
        continue; // 跳过 4，直接去 i++
    }
    if (i == 8) {
        break;    // 遇到 8，彻底停止循环
    }
    cout << i << " ";
}
// 猜猜输出什么？ 1 2 3 5 6 7
