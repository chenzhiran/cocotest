一、复习回顾：While循环的回声

在学习新魔法之前，我们先回顾一下 while 循环。它就像一个“看门人”，每次都要检查条件，只有条件为真，才让你进门跑一圈。

👮‍♂️
While 循环

“只要条件成立，就一直做”

适合：不知道要跑多少圈的时候

但是，如果我们明确知道“我要跑 10 圈”，用 while 就要写初始化、条件、步长三行代码，很容易漏掉！这时候，for 循环就登场了。

二、For循环：最完美的循环结构

Python 的 for 循环和 C++ 不太一样，它更像是一个“遍历者”。它通常配合一个神奇的魔法函数 range() 使用。

✨ 核心语法：
for 变量 in range(起点, 终点, 步长):
🧙‍♂️ 魔法口诀：
range(start, stop)
包头不包尾！
就像买票看电影，你能坐 start 号座位，但 stop 号是墙壁，不能坐！ 
# 打印 1 到 5 (记得终点要写 6)
for i in range(1, 6):
    print(i, " ", end="")
# 输出：1 2 3 4 5

Advanced：倒着数数 (步长为负)

如果要从 10 数到 1 怎么办？把步长设为 -1 就可以啦！

# 发射倒计时：10, 9, ... 1
# 终点写 0，因为不包含 0，所以会停在 1
for i in range(10, 0, -1):
    print(i)

三、流程控制：Break与Continue的交通指挥

循环有时候像一辆停不下来的车，我们需要“交通指挥”来控制它。这两个命令在 while 和 for 里通用！

🛑 Break (终止)

意思： 彻底结束循环，跳出跑道，不玩了！

场景： 找东西，找到了就不用再找了。

⏭️ Continue (跳过)

意思： 结束本次循环，直接进入下一轮。

场景： 逢 7 跳过，或者老师点名跳过请假的同学。

# 游戏规则：打印 1~10
# 遇到 4 的时候跳过 (Continue)
# 遇到 8 的时候游戏结束 (Break)

for i in range(1, 11):
    if i == 4:
        continue # 跳过 4，去下一个数字
    if i == 8:
        break    # 遇到 8，彻底停止循环
    print(i, end=" ")

# 猜猜输出什么？ 1 2 3 5 6 7
