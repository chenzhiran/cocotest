一、复习回顾：顺序与选择结构总结

在学习强大的“循环魔法”之前，我们先看看已经掌握了哪些控制程序的法宝。

二、while循环：只要满足条件就一直做

生活中有时候我们需要重复做某件事，比如：“只要肚子饿（条件），就一直吃包子（动作）”。

在 Python 中，我们用 while 来实现这个功能。它的意思是“当……的时候”。

while 肚子饿:
    # 只要肚子饿为真，就一直执行这里
    吃包子()

三、循环三要素：起、止、变

要让循环正常工作，不变成永远停不下来的“疯子”，我们需要给它安装三个关键零件。

⚙️ 循环三要素
🏁
1. 初始值 (起点)
从哪里开始？
🛑
2. 循环条件 (终点)
什么时候继续？
👣
3. 变量更新 (步长)
每一步怎么变？

例子： 也就是围着操场跑 5 圈。

i = 1             # 1. 初始值：我刚跑了第1圈

while i <= 5:     # 2. 条件：只要跑的圈数 <= 5，就继续跑
    print("我跑了第", i, "圈")
    i += 1        # 3. 更新：i 增加 1 (注意：Python没有i++)

四、死循环：停不下来的噩梦与强制退出

如果你忘记写“变量更新”（比如忘记写 i += 1），或者条件永远是真的（比如 1 > 0），循环就会变成死循环！

😵‍💫

电脑：你不喊停，我就一直跑！累死我了！

🚨 紧急救援：

如果程序运行后一直没反应，或者疯狂输出，说明你可能写了死循环！

请立即点击控制台的 停止按钮 (Stop) 或者按 Ctrl + C 强行终止程序。

五、统计思想：计数器与累加器的使用

循环最常用的两个功能：数数（一共多少个？）和 求和（一共多少钱？）。

🔢 计数器 (Counter)

作用： 统计满足条件的个数。

口诀： cnt = 0 (清零)，cnt += 1 (加一)。

cnt = 0 # 1. 空本子
i = 1
while i <= 5:
    cnt += 1 # 2. 画一笔
    i += 1
print(cnt) # 5

💰 累加器 (Accumulator)

作用： 计算总和。

口诀： total = 0 (空罐子)，total += x (扔进去)。

total = 0 # 1. 空罐子
i = 1
while i <= 5:
    total += i # 2. 扔进去
    i += 1
print(total) # 1+2+3+4+5=15
