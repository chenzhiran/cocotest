一、复习回顾：一维数组与双重循环

在进入二维世界之前，我们先回忆两个关键工具。

工具1：一维数组
[ 10, 20, 30, 40 ]

像一条线，只有一个下标 a[i]。

工具2：双重循环
for (i) {
  for (j) { ... }
}

像时钟，分针走一圈，时针走一格。

二、二维数组：从一排箱子到一面柜子

试想一下，如果我们要存全班同学的座位表（6排8列），用一维数组 a[48] 虽然也能存，但想知道“第3排第5个是谁”就很麻烦。

这时候，我们需要二维数组！

// 定义一个 3 行 4 列 的二维数组
int map[3][4]; 

// 就像是一个 3层楼，每层有 4个房间 的公寓

三、行与列：确定你的坐标 (Row & Col)

要在这个大柜子里找到指定的东西，我们需要两个坐标：行号 (Row) 和 列号 (Column)。

  • 行 (Row)：横着的，决定在第几层。
  • 列 (Column)：竖着的，决定在第几个。
🧙‍♂️ 记忆口诀：
先说楼层（行），再说房间号（列）。
a[1][2] 意思是：第 1 行，第 2 列的那个格子。
注意： 既然是 C++，行和列的编号都是从 0 开始的！

四、矩阵遍历：双重循环的威力

如果我们要把这个二维数组填满，或者把里面的东西都打印出来，需要怎么做？

我们需要两个 for 循环嵌套：外层循环控制行，内层循环控制列。

// 输入一个 3行 4列 的矩阵
for (int i = 0; i < 3; i++) {      // 外层：遍历每一行
    for (int j = 0; j < 4; j++) {  // 内层：遍历该行的每一列
        cin >> a[i][j];
    }
}

五、应用案例：打印一个数字方阵

我们来实战一下：读入行数 n 和列数 m，然后读入矩阵，最后把它们整齐地打印出来。

💡 打印技巧

1. 每打印完一个数字，输出一个空格。

2. 每打印完一行（内层循环结束），输出一个换行符 endl。

int n, m;
int a[105][105]; // 数组通常开大一点

cin >> n >> m;

// 1. 读入
for (int i = 0; i < n; i++) {
    for (int j = 0; j < m; j++) {
        cin >> a[i][j];
    }
}

// 2. 打印
for (int i = 0; i < n; i++) {
    for (int j = 0; j < m; j++) {
        cout << a[i][j] << " ";
    }
    cout << endl; // 关键：换行！
}
