一、复习回顾：列表与双重循环

在进入二维世界之前，我们先回忆两个关键工具。

工具1：普通列表
[ 10, 20, 30, 40 ]

像一条线，只有一个下标 a[i]。

工具2：双重循环
for i in range(3):
  for j in range(4):

像时钟，分针走一圈，时针走一格。

二、嵌套列表：从一排箱子到一面柜子

试想一下，如果我们要存全班同学的座位表（6排8列），用一个长列表存起来虽然可行，但很难知道“第3排第5个是谁”。

在 Python 中，我们可以让列表里装的元素，还是列表！这就是二维列表（Nested List）。

# 定义一个 3 行 4 列 的二维列表
matrix = [
    [1, 2, 3, 4],
    [5, 6, 7, 8],
    [9, 0, 1, 2]
]

三、行与列：确定你的坐标 (Row & Col)

要在这个大柜子里找到指定的东西，我们需要两个坐标：行号 (Row) 和 列号 (Column)。

  • 行 (Row)：matrix[0], matrix[1]... 每一个都是一个小列表。
  • 列 (Column)：小列表里的第几个元素。
🧙‍♂️ 记忆口诀：
a[1][2] 意思是：先找第 1 个列表（行），再找里面的第 2 个元素（列）。
注意： Python 的下标也是从 0 开始的！

四、矩阵遍历：双重循环的威力

如果我们要把这个二维列表填满，或者把里面的东西都打印出来，需要怎么做？

我们需要两个 for 循环嵌套：外层循环控制行，内层循环控制列。

# 读取一个 n 行 m 列的矩阵
n, m = map(int, input().split())
matrix = []

for i in range(n):
    # 读入一行，变成一个列表
    row = list(map(int, input().split()))
    matrix.append(row) # 把小列表装进大列表

五、应用案例：打印一个数字方阵

我们来实战一下：读入行数 n 和列数 m，然后读入矩阵，最后把它们整齐地打印出来。

💡 打印技巧

1. print(x, end=" ")：打印数字和空格，不换行。

2. 内层循环结束后，单独写一个 print()：负责换行。

n, m = map(int, input().split())
# 1. 读入矩阵
matrix = []
for i in range(n):
    matrix.append(list(map(int, input().split())))

# 2. 打印矩阵
for i in range(n):
    for j in range(m):
        print(matrix[i][j], end=" ")
    print() # 关键：一行打完要换行！
