CSP-J1801

和与积 (累加累乘)

思路分析

我们需要两个“篮子”：

  • 求和篮子 (sum)
    初始必须是 0。
    就像捡苹果，空篮子开始捡。
  • 求积篮子 (fac)
    初始必须是 1。
    因为任何数乘0都变0了，所以要从1开始乘。

循环 1 到 n，每次把当前的数加到sum里，乘到fac里。

n = int(input())
s = 0; f = 1
1. 初始化
积从1开始
long long n; cin>>n;
long long s=0, f=1;
for i in range(1, n+1):
2. 循环 1~n
Python范围左闭右开
for(int i=1; i<=n; i++) {
  s += i
  f *= i
3. 累加与累乘
  s += i;
  f *= i;
}
print(s)
print(f)
4. 输出结果
cout << s << endl;
cout << f << endl;
C++ 提示：乘积可能很大，记得用 long long 哦！
CSP-J1802

整除筛选 (筛选法)

思路分析

我们要从 1 到 100 挑出符合条件的数字。

筛选条件：

1. 能被3整除：i % 3 == 0
2. 不能被5整除：i % 5 != 0

把这两个条件用 且 (and/&&) 连接起来，满足了就打印出来。

for i in range(1, 101):
1. 遍历 1~100
for(int i=1; i<=100; i++) {
  if i % 3 == 0 and i % 5 != 0:
2. 判断条件
  if(i%3==0 && i%5!=0)
    print(i, end=" ")
3. 输出
空格隔开
    cout << i << " ";
}
CSP-J1803

多重筛选 (逻辑与)

思路分析

这道题比上一题多了一个条件：必须是 偶数。

三个关卡：

  1. 被3整除 (i%3==0)
  2. 不被5整除 (i%5!=0)
  3. 是偶数 (i%2==0)

虽然条件多了，但逻辑是一样的，用 && 把它们串起来！

for i in range(1, 101):
1. 遍历 1~100
for(int i=1; i<=100; i++) {
  if i%3==0 and i%5!=0 and i%2==0:
2. 三重条件
  if(i%3==0 && i%5!=0 && i%2==0)
    print(i, end=" ")
3. 输出
    cout << i << " ";
}
CSP-J1804

水仙花数 (位值分离)

思路分析

水仙花数是一个3位数 (100-999)。

规则：

153 = 1³ + 5³ + 3³
153 = 1 + 125 + 27

关键在于把 个、十、百 位拆出来。

  • 个位 (g): n % 10
  • 十位 (s): (n // 10) % 10
  • 百位 (b): n // 100
for n in range(100, 1000):
1. 遍历三位数
for(int n=100; n<=999; n++) {
  b = n // 100
  s = (n // 10) % 10
  g = n % 10
2. 拆分各位
  int b = n/100;
  int s = n/10%10;
  int g = n%10;
  if b**3 + s**3 + g**3 == n:
3. 立方和判断
  if(b*b*b + s*s*s + g*g*g == n)
    print(n, end=" ")
4. 输出
    cout << n << " ";
}
CSP-J1805

特殊的四位数 (循环输入)

思路分析

我们要处理 n 个四位数。

对每一个数，都要拆分它的千、百、十、个位。

判断条件：

个位 - 千位 - 百位 - 十位 > 0

需要一个计数器 cnt，每发现一个符合条件的数，就 +1。

n = int(input())
arr = list(map(int, input().split()))
cnt = 0
1. 输入与初始化
C++在循环中读入
int n; cin >> n;
int x, cnt = 0;
for x in arr:
2. 遍历每个数
for(int i=0; i<n; i++) {
  cin >> x;
  q = x // 1000
  b = (x // 100) % 10
  s = (x // 10) % 10
  g = x % 10
3. 拆解四位数
  int q = x/1000;
  int b = x/100%10;
  int s = x/10%10;
  int g = x%10;
  if g - q - b - s > 0:
    cnt += 1
4. 条件判断计数
  if(g-q-b-s > 0) cnt++;
}
print(cnt)
5. 输出总数
cout << cnt;
CSP-J1806

奇数和 (区间求和)

思路分析

计算 m 到 n 之间所有 奇数 的和。

步骤：

  1. 准备一个空篮子 sum = 0。
  2. 从 m 走到 n (包括 n)。
  3. 如果是奇数 (i % 2 != 0)，就扔进篮子里。

比如 3 到 7：
3是奇数(加)，4不是，5是(加)，6不是，7是(加)。
Sum = 3 + 5 + 7 = 15。

m, n = map(int, input().split())
s = 0
1. 输入与初始化
int m, n; cin >> m >> n;
int s = 0;
for i in range(m, n+1):
2. 遍历 m~n
for(int i=m; i<=n; i++) {
  if i % 2 != 0:
    s += i
3. 如果是奇数则累加
  if(i % 2 != 0)
    s += i;
}
print(s)
4. 输出
cout << s;
CSP-J1807

弹跳气球 (物理模拟)

思路分析

这是一个物理模拟过程。

过程拆解 (以 h=2 为例)：

  • 第1次落地：经过 2米。反弹 1米。
  • 第2次落地：经过 2 + 1(上) + 1(下) = 4米。反弹 0.5米。
  • ...

规律：
每次落地，先加上落下的距离。
然后高度减半。
下次循环时，落下的距离其实就是“反弹高度”的2倍(一上一下)，但这里题目问的是第10次落地时。
我们分开算：初始高度h算一次落下。之后9次反弹，每次经过 2 * 反弹高度。

h = int(input())
total = h
cur_h = h / 2
1. 第一次落地
total=初始高度
反弹高度减半
double h; cin >> h;
double total = h;
double cur_h = h / 2.0;
for i in range(2, 11):
2. 第2到10次落地
for(int i=2; i<=10; i++) {
  total += cur_h * 2
  cur_h /= 2
3. 累加路程并减半
往返路程 = 高度*2
  total += cur_h * 2;
  cur_h /= 2;
}
print(f"{total:.4f}")
print(f"{cur_h:.4f}")
4. 输出保留4位小数
printf("%.4f\n", total);
printf("%.4f", cur_h);
