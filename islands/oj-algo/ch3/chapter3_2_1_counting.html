2.1 计数排序 (Counting Sort)

👋 小朋友们好！ 想象一下，你手里有一把乱七八糟的扑克牌，或者一堆不同颜色的乐高积木。如果让你把它们从小到大排好，你会怎么做？

如果你不想比较“这张牌大”还是“那张牌大”，你可以直接准备几个带有编号的盒子，把每张牌直接扔进对应的盒子里！这，就是计数排序的魔法！✨

🎨 详细图解步骤

假设我们要排序这组数字：2, 5, 3, 0, 2, 3, 0, 3

  • 第一步：找老大 (Find Max)
    先看看这堆数字里最大的是几？如果是 5，我们就需要准备 0, 1, 2, 3, 4, 5 号，一共 6 个盒子。
  • 第二步：投票进桶 (Counting)
    拿起一个数字，它是几，就往几号盒子里投一票（或者放一颗糖）。
    关键点：数字本身 = 盒子的编号 (下标)。
  • 第三步：依次点名 (Output)
    从 0 号盒子开始检查。0 号盒子里有 2 票，就写两个 0；1 号盒子是空的，跳过；2 号盒子里有 2 票，就写两个 2……一直写到最后。

💻 编程时间！

看看在电脑里怎么实现这个过程吧。注意看注释哦！

#include <iostream> #include <vector> #include <algorithm> // max函数在这里 using namespace std; int main() { // 1. 准备一堆乱序的数字 int a[] = {2, 5, 3, 0, 2, 3, 0, 3}; int n = 8; // 一共有8个数字 // 2. 找出最大的数字 (为了知道要准备多少个盒子) int max_val = 0; for(int i = 0; i < n; i++) { if(a[i] > max_val) max_val = a[i]; } // 3. 准备盒子 (计数数组) // 只要最大值是5，我们就需要到下标5，所以大小要是 max_val + 1 // vector自动把所有盒子初始化为0 (都是空的) vector<int> count(max_val + 1, 0); // 4. 开始计数 (核心步骤！) for(int i = 0; i < n; i++) { int num = a[i]; // 拿到这个数字 count[num]++; // 对应编号的盒子数量 +1 } // 5. 倒出来 (输出) cout << "排序结果: "; for(int i = 0; i <= max_val; i++) { // 只要盒子不为空，就一直倒出来 while(count[i] > 0) { cout << i << " "; count[i]--; // 倒出来一个，少一个 } } return 0; }
def counting_sort(a): # 1. 找出最大值 if not a: return [] max_val = max(a) # 2. 准备盒子 (列表)，长度要是 最大值+1 # [0] * 6 就会生成 [0, 0, 0, 0, 0, 0] count = [0] * (max_val + 1) # 3. 统计每个数字出现的次数 for num in a: count[num] += 1 # 4. 把它们按顺序倒回到原数组 (或者生成新数组) output = [] for i in range(max_val + 1): # 如果 i 号盒子里有 count[i] 个球 # 我们就追加 count[i] 个 i while count[i] > 0: output.append(i) count[i] -= 1 return output # 测试一下 a = [2, 5, 3, 0, 2, 3, 0, 3] result = counting_sort(a) print("排序结果:", result)

🚀 只有聪明人才知道的小秘密

🌟 优点：超级快！

它不需要像冒泡排序那样两个两个比来比去。只要扫一遍数据，放进桶里，再倒出来就好了。时间复杂度是 O(n+k)，比快速排序还要快！

⚠️ 缺点：太费地儿！

如果我们要排的数字里有一个是 1亿，那我们就得准备 1亿个盒子！这太浪费空间了。而且，它只能排整数，不能排小数哦。
