4.3 子集枚举

👋 魔法师语录：

给定一个包含 n 个元素的集合，枚举其所有子集（包括空集和全集），共计 2n 个。
核心思想： 每个元素在子集中只有两种状态：选 或 不选。我们就像在做一系列的“单选题”，从而组合出所有的可能性。

4.3.1 二进制枚举（状态压缩）

🖼️ 图解：二进制与子集的完美映射

知识点讲解

  • 核心技巧： 使用一个整数 mask 表示当前子集的选择情况。mask 的二进制表示中，第 i 位（最低位为第 0 位）为 1 表示选中第 i 个元素，为 0 表示不选。
  • 枚举范围： mask 从 0 到 (1 << n) - 1。例如 n = 3 时，范围是 000 到 111（二进制）。
  • 常用位运算：
    • 判断第 i 位是否为 1：if (mask & (1 << i))
    • 获取子集大小：__builtin_popcount(mask)（C++） 或 bin(mask).count('1')（Python）
  • 复杂度： O(2n × n)，适用于 n ≤ 20 的情况。

例题 3：输出集合的所有子集

给定集合 nums = {1, 2, 3}，输出其所有子集。

#include <iostream> #include <vector> using namespace std; int main() { vector<int> nums = {1, 2, 3}; int n = nums.size(); // 枚举所有 mask：0 ~ 2^n - 1 for (int mask = 0; mask < (1 << n); ++mask) { cout << "{ "; for (int i = 0; i < n; ++i) { if (mask & (1 << i)) { // 如果第 i 位为 1 cout << nums[i] << " "; } } cout << "}" << endl; } return 0; }
nums = [1, 2, 3] n = len(nums) for mask in range(1 << n): # 0 到 2^n - 1 subset = [] for i in range(n): if mask & (1 << i): # 如果第 i 位为 1 subset.append(nums[i]) print("{", *subset, "}")

运行结果：{ }, { 1 }, { 2 }, { 1 2 }, { 3 }, { 1 3 }, { 2 3 }, { 1 2 3 }

例题 4：砝码称重

给定 n 个砝码，每个可以使用一次或不使用。求能称出的不同重量的个数（不计 0 重量）。
思路：枚举所有使用方案，用 set（集合）自动去重。

#include <iostream> #include <set> using namespace std; int w[25]; int main() { int n; cin >> n; for (int i = 0; i < n; ++i) cin >> w[i]; set<int> weights; for (int mask = 0; mask < (1 << n); ++mask) { int sum = 0; for (int i = 0; i < n; ++i) { if (mask & (1 << i)) { sum += w[i]; } } weights.insert(sum); } // 输出非零重量的个数（空集重量为 0，需减去 1） cout << weights.size() - 1 << endl; return 0; }
n = int(input()) w = list(map(int, input().split())) weights = set() for mask in range(1 << n): total = 0 for i in range(n): if mask & (1 << i): total += w[i] weights.add(total) # 去除 0 重量的情况 weights.discard(0) print(len(weights))

4.3.2 递归枚举子集

🌳 图解：递归搜索树 (DFS Tree)

知识点讲解

  • 适用场景： 当枚举过程中需要携带额外信息、进行复杂剪枝，或题目要求按特定顺序输出子集时。
  • 递归思想： 依次考虑每个元素“选”或“不选”。每层递归处理一个元素，递归深度为 n。
  • 回溯： 选择某个元素后，在递归返回时需要撤销选择 (pop)，以便尝试不选的另一条分支。
  • 复杂度： 同样为 O(2n)，但常数因子略大。优势在于极其灵活！

递归模板：输出集合的所有子集

#include <iostream> #include <vector> using namespace std; vector<int> subset; int n = 3; int a[] = {1, 2, 3}; void dfs(int k) { if (k == n) { // 抵达叶子节点，输出当前子集 cout << "{ "; for (int x : subset) cout << x << " "; cout << "}" << endl; return; } // 分支 1：不选第 k 个元素 dfs(k + 1); // 分支 2：选第 k 个元素 subset.push_back(a[k]); dfs(k + 1); subset.pop_back(); // ⚠️ 回溯：恢复原状 } int main() { dfs(0); return 0; }
subset = [] a = [1, 2, 3] n = len(a) def dfs(k): if k == n: print("{", *subset, "}") return # 分支 1：不选第 k 个元素 dfs(k + 1) # 分支 2：选第 k 个元素 subset.append(a[k]) dfs(k + 1) subset.pop() # ⚠️ 回溯：恢复原状 dfs(0)

4.3.3 二进制枚举 vs 递归枚举 对比

方法 优点 缺点 适用数据规模
二进制枚举 代码极简，位运算常数小，执行速度快。 不易添加复杂的剪枝逻辑，不方便记录路径状态。 n ≤ 20
递归枚举 易于扩展，极其方便添加剪枝和传递额外参数。 代码稍长，递归调用有额外的函数开销。 n ≤ 25
(配合强力剪枝)

✅ 本节要点：每个元素选或不选，构成 2n 种可能。二进制巧用 & 运算，递归妙在 回溯与剪枝。
