4.1 枚举法的基本思想

👋 魔法师语录：

枚举法（Enumeration），俗称“暴力破解”。它的精髓不在于技巧，而在于“不放过任何一个嫌疑人”！只要你能把所有答案列出来并一个一个检查，就没有解决不了的难题。

一、定义与核心公式

1. 什么是枚举？

从可能解的集合中一一列举各个元素，用题目给定的条件判断哪些是有效的。能够通过考验的，就是我们要找的真理！

2. 核心公式

枚举算法 = 确定解空间 + 逐一列举 + 逐一检验

  • 解空间： 所有可能答案的“大池子”。（比如1~100）
  • 逐一列举： 用循环或递归，一个不漏地把池子里的东西捞上来。
  • 逐一检验： 用 if 看看捞上来的东西对不对。

二、竞赛定位：得分的关键

级别 作用
CSP-J (入门) 前2题的常客，确保“基础分”到手的生存技能。
CSP-S (提高) 骗分（暴力分）神器，也是学习高级算法的敲门砖。

💡 教练提醒： 写一个暴力枚举来“对拍”（验证更高级算法的结果）是顶尖选手的必修课。

三、复杂度估算：生存避坑指南

在动手之前，必须学会“掐指一算”，否则 TLE (超时) 会找上门！

⏱️ 1秒钟内能跑多少次？

  • C++： 约 108 次简单运算。
  • Python： 约 107 次循环（效率较低）。

⚖️ 判断法则：

  • ≤ 106：绝对安全，随便写！
  • 107 ~ 108：卡线边缘，小心优化。
  • ≥ 109：必然超时，必须换思路！

四、代码模板：水仙花数

题目：找出所有三位数，满足：各位数字立方和 = 该数本身。 (例：153 = 1³ + 5³ + 3³)

#include <iostream> using namespace std; int main() { // 1. 确定解空间：100 到 999 for (int num = 100; num <= 999; ++num) { // 2. 逐一列举：提取各位数字 int a = num / 100; // 百位 int b = (num / 10) % 10; // 十位 int c = num % 10; // 个位 // 3. 逐一检验 if (a*a*a + b*b*b + c*c*c == num) { cout << num << endl; } } return 0; }
# 方法一：经典数学拆分 for num in range(100, 1000): a = num // 100 # 百位 b = (num // 10) % 10 # 十位 c = num % 10 # 个位 if a**3 + b**3 + c**3 == num: print(num) # 方法二：Python 魔法（字符串法） for num in range(100, 1000): a, b, c = map(int, str(num)) if a**3 + b**3 + c**3 == num: print(num)

五、本节小结

✅ 核心思想： 列出所有可能 → 逐一检查。

✅ 生存意识： 算好复杂度，别让电脑“累死”。

✅ 适用场景： 数据规模小、逻辑直接的问题。

✏️ 课后小练习：尝试修改代码，找出所有的“四位玫瑰花数”吧！
(提示：玫瑰花数是指各位数字的四次方之和等于本身)
