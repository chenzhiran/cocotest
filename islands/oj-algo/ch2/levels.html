CSP-J1205

负重前行 (交换变量)

思路分析

安雅和安小风要交换行李。手里都拿着东西是没法交换的，必须借助一张“石桌”。

安雅(A) 石桌(Temp) 1. A先把东西放桌上
小风(B) 安雅(A) 2. B把东西给A
石桌(Temp) 小风(B) 3. B拿走桌上的东西
Python
交换动作
C++
a, b = map(int, input().split())
输入行李重量
cin >> a >> b;
# Python魔法交换
a, b = b, a

交换过程
// 借助石桌 temp
int temp = a;
a = b;
b = temp;
print(a, b)
输出新顺序
cout << a << " " << b;
Python太聪明了，可以直接 a, b = b, a。但在C++里，我们要理解“临时变量(temp)”这个经典的编程思想，它就是故事里的那张“石桌”！
CSP-J1206

方圆之争 (综合计算)

思路分析

这道题是前面知识的大集合！我们需要分别计算正方形和圆形。

正方形 (边长 a)

面积 $S1 = a \times a$

周长 $C1 = a \times 4$

圆形 (半径 r)

面积 $S2 = r \times r \times 3.14$

周长 $C2 = 2 \times r \times 3.14$

Python
逻辑步骤
C++
# 自动处理
变量准备
边长和半径
double a, r;
a, r = map(int, input().split())
输入
两个参数
cin >> a >> r;
print(a*a, a*4)

正方形计算
cout << a*a << " " << a*4 << endl;
print(r*r*3.14, 2*3.14*r)

圆形计算
cout << r*r*3.14 << " " << 2*3.14*r;
提示：C++中 endl 用于换行，第一行输出正方形数据后必须换行。
