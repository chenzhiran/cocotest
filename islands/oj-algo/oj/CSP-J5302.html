📜 任务清单 (题目描述)

沙漠里有一条由 n 块“跳石”铺成的道路。勇者每次要从一块石头跳到下一块。

沙灵的规则： 我们把相邻两块石头上数字的差（大数减小数）叫做“跳跃距离”。如果这些跳跃距离刚好包含了从 1 到 n-1 的所有数字，不重复也不漏掉，这就是一次 “欢乐的跳” (Jolly)！否则就是 “不够欢乐” (Not jolly)。

📥 魔法输入阵：
(第1个数字是n，表示有4块石头，后面是石头的数字)
4 1 4 2 3
📤 奇迹输出阵：
(刚好跳出了距离 3, 2, 1)
Jolly

🧠 魔法道具箱 (核心思路：打卡签到法)

要想知道我们有没有收集齐从 1 到 n-1 的所有跳跃距离，最好的方法就是准备一张“任务打卡表”！📋

✨ 施法步骤：

  1. 准备打卡表： 如果有 4 块石头，我们需要收集的距离就是 1、2、3。准备一张表格，初始状态都是“未收集”（False）。
  2. 测量距离： 勇者每跳一步，就算一下距离（用魔法函数 `abs()` 取绝对值，保证距离是正数）。
  3. 盖章打卡： 如果跳出的距离正好在 1 到 n-1 之间，就在打卡表上对应的数字上画个“✅”。
  4. 检查通关： 最后看一眼打卡表，是不是 1 到 n-1 全都打满了“✅”？全满就是 Jolly，缺一个就是 Not jolly！
勇者在石头上跳跃：
1
➡️
4
➡️
2
➡️
3
距离：|1-4|=3
距离：|4-2|=2
距离：|2-3|=1
📋 打卡表情况：
[距离 1]: ✅    [距离 2]: ✅    [距离 3]: ✅
🎉 全收集！输出 Jolly！

📘 C++ 魔法秘籍 (知识点)

  • 📏 魔法尺子 abs()： 电脑可不知道距离永远是正数！如果用小石头减大石头，会出现负数。用 abs() 把数字包裹起来，负数也会乖乖变成正数啦！（需要 #include <cmath> 工具箱哦）。
  • 🛡️ 超级变身术 static_cast<long long>： 这道题的数字可能超级大或者超级小。为了防止相减的时候数字大到“爆炸”（超出 int 的范围），我们临时把它变身成超级无敌大的 long long 类型再减，就绝对安全了！
  • 📋 布尔打卡本 vector<bool> seen(n, false)： 创建一个专门记录真假的本子，一开始全都是 false (没打卡)。

💻 C++ 终极法术 (代码详解)

#include <iostream>
#include <vector>
#include <cmath>  // 📏 魔法尺子 abs() 在这里面！
using namespace std;

int main() {
    // ⚡ 念动加速咒语，读入速度飞起来！
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n; // 🖥️ 电脑问：有几块石头呀？
    vector<int> a(n);
    for (int i = 0; i < n; ++i) {
        cin >> a[i]; // 依次数出每一块石头上的数字
    }

    // 🌟 特殊情况：如果只有1块石头，不用跳，直接算成功！
    if (n == 1) {
        cout << "Jolly\n";
        return 0;
    }

    // 📋 准备一个长度为 n 的打卡本，一开始全是 false (没打卡)
    vector<bool> seen(n, false); 
    bool jolly = true; // 先假设我们能通关！(立个Flag)

    // 🏃‍♂️ 勇者开始跳跃啦！（从第2块石头开始算跟前一块的距离）
    for (int i = 1; i < n; ++i) {
        // 📏 算距离：为了防止数字爆炸，先变成 long long 再相减，然后用 abs 取绝对值
        long long diff = abs(static_cast<long long>(a[i]) - a[i-1]);
        
        // ⚠️ 警报：如果跳的距离太小（不到1）或者太大（>=n），直接失败！
        if (diff < 1 || diff >= n) {
            jolly = false; // Flag 倒了...
            break;         // 不用往下看了，直接退出！
        }
        seen[diff] = true; // ✅ 距离合法，在打卡本上盖章！
    }

    // 🧐 最后检查环节：如果还没失败，就检查打卡本有没有漏掉的
    if (jolly) {
        for (int d = 1; d < n; ++d) { // 检查 1 到 n-1 每页纸
            if (!seen[d]) { // 如果发现有没打卡的 (!seen)
                jolly = false;
                break;
            }
        }
    }

    // 📢 公布结果：如果 jolly 是真的，输出 Jolly，否则 Not jolly
    cout << (jolly ? "Jolly" : "Not jolly") << '\n';
    return 0; 
}

🌟 C++ 重点复习小贴士

  • 💡 为什么要写 break？ 当勇者跳跃时，一旦发现有一个距离不对（比如距离等于 0 或者跑到 n 外面去了），这条路就已经“不够欢乐”了，直接用 break 跑路，能让程序跑得更快！
  • 💡 边界小测试： 如果输入 1 5 (也就是 n=1，只有一块石头 5)，程序会不会崩溃？我们在代码前面单独加了 if (n == 1) 的防线，把它保护得很好！

🐍 Python 魔法秘籍 (知识点)

  • 📏 魔法尺子 abs()： 在 Python 里，这是随身携带的法宝，不用召唤工具箱就能用。不管里面是正数还是负数，abs(-5) 都会变成 5！
  • 📋 瞬间造表 [False] * n： 想要一本有 n 页，每一页都写着“没打卡”(False) 的小册子吗？只要用乘法咒语，一秒钟就能造出来！
  • 🌪️ 全能吸尘器 sys.stdin.read().strip().split()： 这道题的数字可能长得很奇怪，可能好多行。这个超级吸尘器能把所有文字吸过来，然后切成一个个干净的单词！

💻 Python 终极法术 (代码详解)

import sys # 召唤高级工具箱！

def main():
    # 🌪️ 用超级吸尘器吸入所有文字，去掉多余空白，并切成小块！
    data = sys.stdin.read().strip().split()
    if not data:
        return # 如果啥也没输入，就悄悄溜走

    # 📦 第1个包裹是 n(石头的数量)
    n = int(data[0])
    
    # 🪨 剩下的全是石头上的数字啦，全部变身成整数装进列表里
    a = list(map(int, data[1 : 1+n]))

    # 🌟 特殊情况：如果只有1块石头，直接宣布胜利！
    if n == 1:
        print("Jolly")
        return

    # 📋 准备打卡本：n页纸，全写着 False (表示距离还没收集到)
    seen = [False] * n  
    jolly = True # 先假设我们能通关！(立个Flag)

    # 🏃‍♂️ 勇者开始跳跃！（从下标 1 走到 n-1，每次和前一块石头 a[i-1] 算距离）
    for i in range(1, n):
        # 📏 算距离，用 abs() 把负数变正数！
        diff = abs(a[i] - a[i-1])
        
        # ⚠️ 警报：距离是不是太小了（不到1）或者太大了（>=n）？
        if diff < 1 or diff >= n:
            jolly = False # 糟糕，任务失败...
            break         # 直接停下来，不跳了！
            
        seen[diff] = True # ✅ 这个距离是合法的，在打卡本上盖章！

    # 🧐 结算时刻：如果勇者没摔倒，检查打卡本有没有漏页的
    if jolly:
        for d in range(1, n): # 检查 1 到 n-1 的打卡情况
            if not seen[d]:   # 发现有没打卡的！
                jolly = False
                break

    # 📢 公布成绩单：根据 jolly 的真假大声喊出来
    print("Jolly" if jolly else "Not jolly")

if __name__ == "__main__":
    main() # 🚀 启动魔法！

🌟 Python 重点复习小贴士

  • 💡 `range(1, n)` 的秘密： 在 Python 中，这个小魔法只会生成从 1 开始到 **n-1** 的数字，刚好跟我们要检查的“跳跃距离”是一模一样的！
  • 💡 超级简写的魔法： print("Jolly" if jolly else "Not jolly") 这是一句非常地道的 Python 魔法咒语（三元运算符），一行代码就把 if...else 给写完了，是不是很酷？
