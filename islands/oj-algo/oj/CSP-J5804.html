📜 任务清单 (题目描述)

平面直角坐标系中给定了 n 个巨人的坐标以及 n 个鬼的坐标。

你的任务是将每一个巨人与唯一的一只鬼进行一对一连接，使得所有的连线（线段）在平面上互不相交。

题目保证满足条件的无交叉连接方案总是存在。

星域输入与输出法则：

  • 输入包含多组独立的数据集，直到文件末尾（EOF）结束。
  • 每组数据的第一行是一个正整数 n，表示巨人的数量（同时也是鬼的数量）。
  • 接下来的 n 行，每行包含两个整数 x, y，代表第 i 个巨人的平面坐标。
  • 再接下来的 n 行，每行包含两个整数 x, y，代表第 i 个鬼的平面坐标。
  • 绝对安全保证： 平面上所有的巨人和鬼的位置是唯一的，并且没有任何三点共线。
  • 对于每组数据集，输出 n 行。第 i 行的数字表示第 i 个巨人所连接的鬼的编号（编号从 1 到 n）。
  • 相邻的两个测试数据集输出之间，用一个空行隔开。

🧠 魔法道具箱 (核心图解：基准点锚定与极角扫瞄分水岭)

如果在平面上任意乱连，纠缠的线段会让我们寸步难行。但如果我们借助分治递归（Divide and Conquer）的思想，就可以把庞大的点集切开：

🌟 极角切分法术原理模拟 🌟
(空心圆 = 巨人，实心圆 = 鬼)
第一步（找基准）： 找出当前区域里 Y 坐标最小（最靠下方）的点，将其作为锚定基准点 Pivot。
第二步（极角排队）： 让剩下的所有点，相对于这个基准点计算极角，并按照极角从小到大（或大到小）排序。此时，所有点就像一把扇子一样围绕着基准点整齐排开。
第三步（颜色平衡切分）： 从极角最小的射线开始往另一侧扫描，用计数器统计碰到的巨人和鬼的数量（遇到同色加1，异色减1）。
第四步（一分为二）： 一旦在中途某个点，计数器刚好归零，且这个点和基准点的颜色截然不同！
这就意味着：基准点与当前点连成的一条直线，完美地把剩余的所有点划分成了左、右两个独立的区域，并且每个区域内巨人的数量依然刚好等于鬼的数量！

因为左边区域和右边区域被基准点与当前点的连线彻底隔开，我们直接把左边和右边分别丢进递归函数里去继续切分，两边的连线绝对不可能跨过防线发生交叉！这就是几何分治的极致魅力。

📘 C++ 魔法秘籍 (知识点)

  • 多属性结构体 pnt： 分别存储坐标 x, y，用 col 标记颜色阵营（0代表巨人，1代表鬼），用 idx 牢记原本的 1-Based 初始编号，防止排序后迷失身份。
  • 坐标相对平移： 找到基准点后，通过重载减号运算符 cor[i] = cor[i] - cor[l] 将其转换为以基准点为原点的相对坐标，方便极角运算。
  • 数学库极角排序 atan2： 利用内置数学工具 atan2(x, y) 计算出高精度的斜率极角值，配合标准快排对子区间 sort(cor + l + 1, cor + r, cmp) 实施扇形排队。

💻 C++ 终极法术 (极角扫描分治版)

#include <iostream>
#include <vector>
#include <cmath>
#include <algorithm>
#include <cstring>
#include <cstdio>

#define rep(i,a,b) for(int i=a;i<=b;++i)
using namespace std;

const int maxn = 114;
int n;

// 🔮 点结构体：整合空间物理坐标与属性
struct pnt {
    int x, y, col, idx; // col==0: 巨人; col==1: 鬼; idx表示原始下标
    bool operator <(const pnt &a)const {
        return y < a.y; // 用于快速筛选最下方的基准锚点
    }
};

// 📐 重载减号：实现坐标系的平移转化
pnt operator -(const pnt &a, const pnt &b) {
    return pnt{a.x - b.x, a.y - b.y, a.col, a.idx};
}

// 📥 读取函数：注入阵营颜色与天然索引
void rd(pnt &a, int col, int idx) {
    scanf("%d%d", &a.x, &a.y);
    a.col = col, a.idx = idx;
}

// ⚖️ 极角比较函数：按 atan2 计算的极角从大到小降序排列
bool cmp(const pnt &a, const pnt &b) {
    return atan2(a.x, a.y) >= atan2(b.x, b.y);
}

pnt cor[maxn << 1]; // 存储大阵中所有的巨人和鬼（2 * n 个点）
int ans[maxn];      // 🏆 存储最终每个巨人匹配到的鬼的编号

// 🧙‍♂️ 核心几何分治破阵函数（左闭右开区间）
void solve(int l, int r) {
    if (l >= r) return;
    
    // ⏱️ 步骤一：通过 Y 轴排序，让当前区间的首元素 cor[l] 变成最下方的基准锚点
    sort(cor + l, cor + r);
    
    // ⏱️ 步骤二：坐标平移，将剩余点全部转换为相对于基准点 cor[l] 的相对坐标
    rep(i, l + 1, r - 1) {
        cor[i] = cor[i] - cor[l];
    }
    
    // ⏱️ 步骤三：除了基准点之外，其余点按照极角大小降序排序成扇形
    sort(cor + l + 1, cor + r, cmp);
    
    int cnt = 0; // 颜色平衡平衡计数器
    
    // 🔍 步骤四：线性扇形扫描，寻找完美的不交叉分水岭连线
    rep(i, l + 1, r - 1) {
        // 如果当前前缀区域内颜色正好保持平衡，且当前点与基准点异色（可以连线）
        if (cnt == 0 && (cor[l].col ^ cor[i].col)) {
            // 建立主从契约：将鬼的编号（idx）赋给对应巨人的输出位置
            cor[l].col ? ans[cor[i].idx] = cor[l].idx : ans[cor[l].idx] = cor[i].idx;
            
            // 🚀 顺着这条连线将大阵一分为二，左右两个子防线各自展开独立递归
            solve(l + 1, i);
            solve(i + 1, r);
            return; // 本层使命完成，安全撤退
        }
        // 累加当前的阵营平衡度（巨人让计数加1，鬼让计数减1）
        cnt += (cor[i].col ? -1 : 1);
    }
}

int main() {
    // 🔄 循环吞入多组测试数据，直到文件末尾结束
    while (scanf("%d", &n) == 1) {
        // 顺序读取：前 n 个是巨人(col=0)，后 n 个是鬼(col=1)
        rep(i, 1, (n << 1)) {
            rd(cor[i], ((i - 1) / n) ? 1 : 0, (i - 1) % n + 1);
        }
        
        // 开启几何大分裂，1到2n+1进行闭开区间递归
        solve(1, (n << 1) | 1);
        
        // 📤 施放结果，整齐输出每个巨人的护卫鬼编号
        rep(i, 1, n) {
            printf("%d\n", ans[i]);
        }
        puts(""); // 每个数据集之间打印空行隔开
    }
    return 0;
}
