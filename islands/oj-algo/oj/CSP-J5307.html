📜 任务清单 (题目描述)

在营地里，有 n 位同学。我们要根据他们的出生年月日给他们排队。

排队规则：

  1. 越早出生（年龄越大）越靠前！ 先比年份，再比月份，最后比日子。
  2. 如果两个人同年同月同日生，怎么办呢？那就让后排队（输入比较晚）的同学先站出来！
📥 魔法输入阵：
(4位同学：名字 年 月 日)
4
Alice 2005 1 2 (第1个来)
Bob 2005 1 2 (第2个来)
Carol 2004 12 31 (第3个来)
Dave 2006 1 1 (第4个来)
📤 奇迹输出阵：
(2004年的Carol最大。Alice和Bob都是2005-1-2，但Bob后输入，Bob优先！Dave最小)
Carol
Bob
Alice
Dave

🧠 魔法道具箱 (核心思路：多级比较闯关)

比较两个人谁该排前面，就像是过五关斩六将！如果第一关分出胜负，就不用比后面的了；如果打平，就进入下一关！

🛡️ Alice (先来) VS Bob (后来) 的对决 🛡️
第一关：比年份 (Year)
Alice 是 2005，Bob 是 2005。打平！继续！
第二关：比月份 (Month)
Alice 是 1月，Bob 也是 1月。打平！继续！
第三关：比日子 (Day)
Alice 是 2日，Bob 也是 2日。又打平！继续！
终极关：比入营顺序 (Index)
规则是：同一天出生的，后来的人排前面！
Alice是第1个来的 (编号大不占优)，Bob是第2个来的 (编号大优先)！
👦 Alice
编号: 1
Bob 胜出！
👦 Bob
编号: 2

📘 C++ 魔法秘籍 (知识点)

  • 🎒 超级大背包 struct Student： 这次每个同学有 5 个属性！名字、年、月、日，还有他们是第几个来的 (idx)，全都打包存起来！
  • 🧑‍⚖️ 四重裁判 cmp： 在写 cmp 裁判函数时，我们可以用 != 来判断。如果年份不一样，直接返回年份小的排前面！如果年份一样，再往下比。
  • ⚠️ 逆向思维 return a.idx > b.idx;： 到了最后一关，因为我们想让后来的排前面，而后来的 idx 更大，所以我们让 a.idx > b.idx 返回真！

💻 C++ 终极法术 (代码详解)

#include <iostream>
#include <vector>
#include <algorithm>
#include <string>
using namespace std;

// 🎒 制作学生资料卡（结构体）
struct Student {
    string name; // 名字
    int y, m, d; // 年、月、日
    int idx;     // 🎫 第几个来的（排队号）
};

// 🧑‍⚖️ 自定义多级裁判：告诉电脑两个学生谁排前面
bool cmp(const Student &a, const Student &b) {
    // 1. 如果年份不一样，出生早的（数字小）排前面！
    if (a.y != b.y) return a.y < b.y;  
    
    // 2. 年份打平了，看月份。月份小的排前面！
    if (a.m != b.m) return a.m < b.m;  
    
    // 3. 月份打平了，看日子。日子小的排前面！
    if (a.d != b.d) return a.d < b.d;  
    
    // 4. 终极对决：同年同月同日生，排队号大的（后来报到的）排前面！
    return a.idx > b.idx; 
}

int main() {
    // ⚡ 加速读入
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;

    vector<Student> stu(n); // 准备 n 张资料卡
    
    // 📝 挨个发资料卡填表
    for (int i = 0; i < n; ++i) {
        cin >> stu[i].name >> stu[i].y >> stu[i].m >> stu[i].d;
        stu[i].idx = i; // 🎫 记录他们是第几个来的（发号码牌）
    }

    // 🪄 念动排序咒语，按照 cmp 裁判的规则排队！
    sort(stu.begin(), stu.end(), cmp);

    // 📢 大声念出排好队的名字！
    for (const auto &s : stu) {
        cout << s.name << '\n';
    }

    return 0;
}

🐍 Python 魔法秘籍 (知识点)

  • 📦 天然的多级比较器 Tuple (元组)： Python 有个超棒的特性！如果把几个数字打包成 (年, 月, 日) 的元组，排序时它会自动先比第一个，如果打平再比第二个，再打平比第三个... 完全不用我们自己写 if...else！
  • 🪄 逆向的负号魔法 -i： Python 默认都是“从小到大”排的。但如果遇到同生日，我们想要后来的人 (序号大的) 排前面怎么办？在序号前面加个负号 -i！本来 2 比 1 大，变成 -2 和 -1 后，-2 就比 -1 小了，就会被自动排到前面去！太聪明了！

💻 Python 终极法术 (代码详解)

import sys # 召唤高级工具箱！

def main():
    # 🌪️ 用超级吸尘器一口气吸入所有文字，并按行切开！
    data = sys.stdin.read().splitlines()
    if not data:
        return

    n = int(data[0].strip()) # 同学的人数
    students = [] # 准备一个空列表装资料

    # 📝 一行一行填资料，从第 1 行读到第 n 行
    for i in range(1, n + 1):
        parts = data[i].split() # 把这一行按空格切开
        name = parts[0]
        y = int(parts[1])
        m = int(parts[2])
        d = int(parts[3])
        
        # 📦 制作排序专用的神奇小包裹！
        # 顺序是：年, 月, 日, -入营序号, 名字
        # 加了负号的 -i 魔法，让后来的人排序时数字变得更小，从而排到前面！
        students.append((y, m, d, -i, name))

    # 🪄 召唤默认排序精灵！它会按小包裹里东西的顺序自动闯关比较！
    students.sort()

    # 📢 大声念出排好队的名字！包裹里第 5 个东西（下标为4）就是名字
    for student in students:
        print(student[4])

if __name__ == "__main__":
    main() # 🚀 启动魔法！

🌟 重点复习小贴士

  • 💡 不能只存年龄！ 如果你把所有的年月日加减乘除算成一个固定的数字（比如天数）来比较也是可以的，但是直接比较 年 -> 月 -> 日 的方式最简单，最不会算错！
  • 💡 如果不写 idx 会怎么样？ 如果同年同月同日，一般的排序算法可能会把原来谁在前面的依然放在前面（这叫稳定排序），或者随便乱放（这叫不稳定排序）。题目强制要求“后输入先输出”，所以我们必须发号码牌 idx 来作为最后一关的评判标准哦！
