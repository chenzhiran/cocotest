📜 任务清单 (题目描述)

在冰川深处，你必须把 1, 2, 3, 4, 5, 6, 7, 8, 9 这九个数字每个只用一次，分成三队，每队组合成一个三位数。

封印开启法则：

  • 这三个三位数的大小比例，必须恰好等于守护者给你的 A : B : C。
  • 你需要找出所有可能的组合，并且按照第一个三位数从小到大排列输出。
  • 如果无论如何都拼不出来，就大喊一声：No!!!

🧠 魔法道具箱 (核心思路：放大镜倍数法)

如果我们要去胡乱拼凑 1 到 9 的数字组合，那可是有 362,880 种可能！计算机也会累得喘粗气的。

但我们有一个超强的数学秘籍：既然这三个数的比例是 A : B : C，那么它们一定可以写成 A * k，B * k，C * k 的形式！这里的 k 就像是一个神奇的“放大镜倍数”。

✨ 施法步骤：

  1. 寻找放大极限： 三个数里最大的一定不能超过 987（因为 9, 8, 7 是我们能组成的最大且不含重复数字的三位数）。所以，我们可以算出放大镜的极限 k_max = 987 / 最大的比例。
  2. 逐个试验倍数： 我们让放大镜倍数 k 从 1 开始慢慢变大，算出现在的三个数字 a, b, c。
  3. 字符串验身法： 把算出的 a, b, c 像串糖葫芦一样连成一长串字符串。如果长度正好是 9，并且把它按从小到大排序后，正好等于 "123456789"，那就说明我们成功破译了一组密码！

📘 C++ 魔法秘籍 (知识点)

  • 🛡️ 边界守护 if (A == 0)： 题目中的 A 可能是 0，但我们无法用 1~9 拼出 0（也没有 0 这个数字给你用），所以遇到 0 直接宣告无解并结束程序！
  • 🔗 字符串拼接术 to_string： 把数字变成字符的超级法术！例如 to_string(192) 就会变成字符串 "192"，然后可以用加号 + 把它们连起来。
  • ✨ 自动排序咒 sort()： 召唤 #include <algorithm> 后，使用 sort(s.begin(), s.end()) 可以瞬间让字符串里的字符从小到大排好队！

💻 C++ 终极法术 (代码详解)

#include <iostream>
#include <string>
#include <algorithm>
using namespace std;

int main() {
    int A, B, C;
    cin >> A >> B >> C;
    
    // 🚨 特判：如果 A 为 0，无法用 1-9 组成三位数，直接失败
    if (A == 0) {
        cout << "No!!!" << endl;
        return 0;
    }
    
    // 🔍 寻找最大的比例数字
    int maxVal = max({A, B, C});
    
    // 📏 计算最大的放大镜倍数 (保证相乘后不超过 987)
    int k_max = 987 / maxVal;          
    
    bool found = false; // 准备一个小红旗，记录有没有找到答案
    
    // 🔄 开始用放大镜逐一测试倍数 k
    for (int k = 1; k <= k_max; ++k) {
        int a = A * k;
        int b = B * k;
        int c = C * k;
        
        // 检查放大后的三个数字是不是都乖乖待在三位数的范围里
        if (a >= 100 && a <= 999 && b >= 100 && b <= 999 && c >= 100 && c <= 999) {
            // 🪄 把三个数字变成字符串，然后串成一串糖葫芦
            string s = to_string(a) + to_string(b) + to_string(c);
            
            // 如果中间混入了前导0之类的奇怪东西导致长度不对，直接跳过
            if (s.length() != 9) continue;   
            
            // 🧹 对这个大字符串里的所有字符从小到大排序
            sort(s.begin(), s.end());
            
            // 🎯 见证奇迹的时刻：看看它是不是正好等于 "123456789"
            if (s == "123456789") {
                cout << a << " " << b << " " << c << endl;
                found = true; // 举起小红旗，我们找到啦！
            }
        }
    }
    
    // 如果小红旗一直没举起来，说明找遍了都没有，那就大喊 No!!!
    if (!found) {
        cout << "No!!!" << endl;
    }
    
    return 0;
}

🌟 C++ 重点复习小贴士

  • 💡 为什么不用穷举 x, y, z？ 如果你用三层循环去猜 x, y, z，哪怕缩小了范围，速度也远不及我们这种只猜“放大镜倍数”的办法。利用数学关系 A * k 直接生成数字，是编程里最聪明的做法！
  • 💡 排序的艺术： 判断九个字符是不是正好是 1 到 9，最笨的办法是写 9 个 if，最聪明的办法就是给它们排个序，看结果是不是 "123456789"！

🐍 Python 魔法秘籍 (知识点)

  • 🧮 整除法术 //： 在 Python 中算极限倍数时，记得用 987 // max_val。这是向下取整，保证倍数是一个干干净净的整数，没有讨厌的小数点尾巴！
  • 🧩 排序组合技 sorted(s)： 面对一个字符串 s，直接使用 sorted(s) 就能把它拆成一个排好序的列表。然后跟标准答案 list('123456789') 相比对，简单极了！
  • 📏 优雅的范围判断： 在 Python 里，判断一个数是不是三位数，可以直接写 100 <= a <= 999，连 and 都省了，是不是特别像数学公式？

💻 Python 终极法术 (代码详解)

def main():
    # 🗺️ 读取契约上的三个比例数字
    A, B, C = map(int, input().split())
    
    # 🚨 特判：如果 A 为 0，冰雪大门直接拒绝访问
    if A == 0:
        print("No!!!")
        return
        
    # 🔍 寻找最大的比例数字
    max_val = max(A, B, C)
    
    # 📏 计算最大的放大镜倍数 (注意用 // 保证结果是整数)
    k_max = 987 // max_val          
    
    found = False # 记录是否找到成功组合的旗帜
    
    # 🔄 让放大镜倍数 k 从 1 慢慢变大，一直试到 k_max
    for k in range(1, k_max + 1):
        a = A * k
        b = B * k
        c = C * k
        
        # 检查这三个兄弟是不是都乖乖符合三位数的标准
        if 100 <= a <= 999 and 100 <= b <= 999 and 100 <= c <= 999:
            # 🪄 把它们变身成字符串，紧紧相连
            s = str(a) + str(b) + str(c)
            
            # 检查总长度是不是9，并且排序后看看是不是等于 1 到 9
            if len(s) == 9 and sorted(s) == list('123456789'):
                print(a, b, c)
                found = True # 举起旗帜！成功啦！
                
    # 走到最后，如果旗帜还是没举起来，只能承认失败咯
    if not found:
        print("No!!!")

if __name__ == "__main__":
    main()

🌟 Python 重点复习小贴士

  • 💡 range 的小陷阱： 记得 for k in range(1, k_max + 1) 里的 + 1 哦！因为 Python 的 range 是包含开头但不包含结尾的，如果忘记加 1，就会漏掉最后一个极限倍数。
  • 💡 列表比对法： sorted(s) 返回的是一个一个字符分开的列表（比如 ['1', '2', '3'...]），所以等号右边我们要用 list('123456789') 把标答也变成列表，这样才能完美匹配。
