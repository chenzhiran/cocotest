冰川祭坛上的图腾有一种自我复制的魔法！只要掌握了“递归”的法则，哪怕是再庞大的雪峰，也能用最简单的积木一次次拼凑出来。

📜 任务清单 (题目描述)

孔明想要刻出一个大小为 n 的“分形图腾”。它的规律是：把一个基础的小三角形，按规律拼成更大的三角形，如此反复。

刻画要求：

  • 只使用字符 /、\、_ 和空格组成图案。
  • 一个大小为 n 的图腾，它的高度是 2^n 行，宽度是 2^(n+1) 列。
📥 魔法输入阵：
(刻画大小 n=2 的图腾)
2
📤 奇迹输出阵：
(输出由 3 个小积木拼成的雪峰)
   /\
  /__\
 /\  /\
/__\/__\

🧠 魔法道具箱 (核心图解：递归之美)

绘制复杂的字符画，最好的办法不是逐行输出，而是“先铺好一张大画布，然后在指定位置盖章”。这个“章”就是我们的递归函数 draw(r, c, s)，意思是：在左上角坐标为 (r, c) 的地方，画一个高度为 s 的大雪峰！

👇 基础积木 (n=1, 高度 s=2, 宽度 4) 👇
/\ /__\
👇 分形进化 (n=2, 拆分为 3 个高度为 s/2 的模块) 👇
/\ /__\ /\ /\ /__\/__\
如果大雪峰左上角是 (r, c)，高度是 s，那么它的一半高度 t = s/2：
⛰️ 顶部积木： 画在 (r, c + t)
⛰️ 左下积木： 画在 (r + t, c)
⛰️ 右下积木： 画在 (r + t, c + s)

✨ 递归的两个核心条件：

  1. 停止条件 (Base Case)： 当雪峰的高度缩小到 s == 2 时，不能再拆了！这时候直接在画布上把 /\ 和 /__\ 这个固定形状“画”上去。
  2. 拆分动作 (Recursive Step)： 如果 s > 2，就把任务分成 3 份，分别召唤 3 个尺寸减半的小雪峰（如上方坐标公式所示）。

📘 C++ 魔法秘籍 (知识点)

  • ⚡ 位运算算乘方： 1 << n 就是 2 的 n 次方，这比调用 pow(2, n) 更快，且直接返回整数类型，非常适合用来计算高度 h 和宽度 w。
  • 🎨 字符画大排版： vector<string> grid(h, string(w, ' ')) 这句魔法瞬间创建了一块满是空格的大画布！把二维字符数组当作画布，用坐标直接覆盖字符，比一行一行拼接要简单一万倍。
  • ✂️ 去除行末尾迹： find_last_not_of(' ') 可以精准找到这一行最后一个“非空格”字符的位置。配合 substr 就能干净利落地切掉右边多余的空气。

💻 C++ 终极法术 (代码详解)

#include <iostream>
#include <vector>
#include <string>
using namespace std;

// 🎨 魔法画笔：在(r,c)点画一个高度为s的雪峰
void draw(int r, int c, int s, vector<string>& grid) {
    if (s == 2) {
        // 🛑 遇到最小积木啦！直接一笔一划写进画布
        // 注意：C++里反斜杠需要用 '\\' 来表示转义
        grid[r][c+1] = '/';
        grid[r][c+2] = '\\';
        grid[r+1][c] = '/';
        grid[r+1][c+1] = '_';
        grid[r+1][c+2] = '_';
        grid[r+1][c+3] = '\\';
        return;
    }
    
    int t = s / 2; // t 是一半的高度
    
    // 🌀 施展递归分身术：召唤3个小一点的雪峰
    draw(r, c + t, t, grid);      // ⛰️ 顶部
    draw(r + t, c, t, grid);      // ⛰️ 左下
    draw(r + t, c + s, t, grid);  // ⛰️ 右下
}

int main() {
    int n;
    cin >> n;
    
    int h = 1 << n;       // 高度 = 2^n
    int w = 1 << (n + 1); // 宽度 = 2^(n+1)
    
    // 铺设一张装满空格的巨大画布
    vector<string> grid(h, string(w, ' '));
    
    // 挥动魔法画笔，从顶点 (0,0) 开始画
    draw(0, 0, h, grid);
    
    // 🖨️ 打印出美丽的图腾
    for (int i = 0; i < h; ++i) {
        // ✂️ 精准定位最后一个不是空格的字符，把后面的空白切掉
        int pos = grid[i].find_last_not_of(' ');
        if (pos != string::npos) {
            cout << grid[i].substr(0, pos + 1) << endl;
        } else {
            cout << endl; 
        }
    }
    return 0;
}

🐍 Python 魔法秘籍 (知识点)

  • 📐 二维列表生成术： [[' '] * w for _ in range(h)] 是 Python 生成独立二维数组的标准咒语！千万别写成 [[' '] * w] * h，否则修改一行会把所有行都改掉！
  • ✂️ 超级修剪工具 rstrip()： 打印时要求行末不要带多余的空格，Python 只需要在字符串后面加上 .rstrip()，它就会自动把右边所有的空白字符“咔嚓”剪干净，优雅至极！
  • 🧩 转义字符的秘密： 字符串里想打印一个真正的 \，必须连写两个 \\，因为第一个反斜杠会被系统当成“转义提示”。

💻 Python 终极法术 (代码详解)

# 🎨 魔法画笔：在(r,c)点画一个高度为s的雪峰
def draw(r, c, s, grid):
    if s == 2:
        # 🛑 遇到最小积木啦！直接写进画布二维列表里
        grid[r][c+1] = '/'
        grid[r][c+2] = '\\'
        grid[r+1][c] = '/'
        grid[r+1][c+1] = '_'
        grid[r+1][c+2] = '_'
        grid[r+1][c+3] = '\\'
        return
        
    t = s // 2  # t 是一半的高度 (用整除)
    
    # 🌀 施展递归分身术：召唤3个小一点的雪峰
    draw(r, c + t, t, grid)      # ⛰️ 顶部
    draw(r + t, c, t, grid)      # ⛰️ 左下
    draw(r + t, c + s, t, grid)  # ⛰️ 右下

def main():
    n = int(input())
    
    h = 1 << n         # 高度 = 2^n
    w = 1 << (n + 1)   # 宽度 = 2^(n+1)
    
    # 铺设一张装满空格的巨大画布
    grid = [[' '] * w for _ in range(h)]
    
    # 从顶点 (0,0) 开始挥动画笔
    draw(0, 0, h, grid)
    
    # 🖨️ 打印输出
    for row in grid:
        # ''.join(row) 把字符列表拼成一个长字符串
        # .rstrip() 瞬间剃掉右侧所有多余空格
        print(''.join(row).rstrip())

if __name__ == "__main__":
    main()

💡 技巧和重点总结 (划重点啦)

  • 🌟 画布思维 (Canvas Pattern)： 绝大多数二维字符图案题，最忌讳就是用一重重的 for 循环一行行硬拼（那样找规律能把人逼疯）。学会先开一个巨大的二维数组当“画布”，然后用 绝对坐标 去涂色，这是一种降维打击的思维！
  • 🌟 找准基准点： 坐标转换是这题的难点。我们统一把一个图腾的 左上角 定为 (r, c)，然后再推算三个子图腾的左上角坐标。画个草图，宽度是高度的 2 倍，偏移量自然就清晰了！
  • 🌟 分形与递归的血缘关系： 分形（Fractal）的本质就是“局部和整体相似”。只要在代码里写出整体怎么被切分成局部的公式，递归的魔法就会自动帮你生成震撼的万花筒效果。
