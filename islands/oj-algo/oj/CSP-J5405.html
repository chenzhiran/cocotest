📜 任务清单 (题目描述)

森林里有编号从 1 到 n 的树叶。你必须从中恰好挑出 r 片。

石门开启的规则非常严格：

  • 不看顺序只看组合： 挑出树叶 1 和 2，跟挑出树叶 2 和 1 是一样的，算作同一种组合。
  • 从小到大念： 每种组合里的数字必须从小到大排好。而且所有组合之间，也要按照“字典序”从小到大一行行念出来。
  • 占位魔法： 念出来的每一个数字，必须刚好占 3 个字符的宽度（并且要靠右对齐）。错一个空格石门都不会开！
📥 魔法输入阵：
(5片树叶，挑出3片)
5 3
📤 奇迹输出阵：
(每个数字占3格，注意开头的空格哦)
1 2 3 1 2 4 1 2 5 1 3 4 ... (后面省略)

🧠 魔法道具箱 (核心思路：回溯法 / 深度优先搜索)

要找齐所有的组合，最厉害的法术叫做“回溯搜索 (DFS)”。它就像是在迷宫里探险：选了一条路走到底，把发现的组合记下来，然后退后一步，换一条路继续找！

为了保证不出现重复的组合（比如选了 1、2，又选了 2、1），我们定下一个规矩：下一次选的树叶编号，必须比这一次选的大！

就这样不断前进、后退、再前进，天然就会按“字典序”找出所有组合！

📐 宽度魔法解析：

题目要求每个数字占 3 格宽度。如果数字是 1，它前面就要补两个空格；如果数字是 10，它前面补一个空格。像这样：

1     10

📘 C++ 魔法秘籍 (知识点)

  • 🎒 魔法篮子 vector： 使用 comb.push_back(i) 把树叶放进篮子，探索完毕后，用 comb.pop_back() 把最后一片树叶拿出来，这就是“回溯”的关键动作。
  • 📏 占位神器 setw(3)： 包含头文件 <iomanip> 后，在输出数字前加上 setw(3)。它就像一个模具，强行给下一个输出的内容撑出 3 个格子的宽度，而且自带靠右对齐的功能，简直是为这道题量身定做的！

💻 C++ 终极法术 (代码详解)

#include <iostream>
#include <iomanip> // 召唤宽度魔法 setw 需要的工具箱！
#include <vector>
using namespace std;

int n, r;
vector<int> comb; // 存放我们挑中树叶的篮子

// 🌲 DFS 探险法术
// start: 这一次能从几号树叶开始挑
// depth: 篮子里现在有几片树叶了
void dfs(int start, int depth) {
    // 🎯 如果篮子里的树叶数量达到了目标 r
    if (depth == r) {
        // 把篮子里的树叶按要求倒出来给大家看！
        for (int i = 0; i < r; ++i) {
            // 📏 setw(3) 保证输出这个数字时，连数字带空格刚好占 3 格
            cout << setw(3) << comb[i];
        }
        cout << endl; // 这一行念完啦，换行！
        return; // 结束这一轮，准备后退一步
    }
    
    // 🚶 从 start 号树叶开始往后看，一直看到 n 号
    for (int i = start; i <= n; ++i) {
        comb.push_back(i);        // 放入篮子！
        
        // 继续深入探险！下一步只能从 i+1 号树叶开始挑了哦，防止选到重复的组合
        dfs(i + 1, depth + 1); 
        
        comb.pop_back();          // 回溯！把刚刚放进去的树叶拿出来，换下一片试试
    }
}

int main() {
    cin >> n >> r;
    dfs(1, 0); // 探险开始：从 1 号树叶开始挑，当前篮子里有 0 片树叶
    return 0;
}

🌟 C++ 重点复习小贴士

  • 💡 为什么输出时不用加空格？ 初学者很容易写成 cout << setw(3) << comb[i] << " ";。其实 setw 会自动用空格填充不足的宽度。如果你再额外加空格，总宽度就会超过 3，导致石门报错哦！

🐍 Python 魔法秘籍 (知识点)

  • 🎒 魔法列表操作： Python 的列表也是一个天然的篮子。用 path.append(i) 把树叶放进去，用 path.pop() 把最后一片弹出来，完成回溯。
  • 📏 格式化字符串 f"{num:3d}"： 这是 Python 最优雅的格式化魔法！冒号后面的 3d 表示这个整数（digit）要强行占据 3 个字符的宽度，并且默认靠右对齐。
  • 🚫 取消换行 end=""： print 魔法默认念完就换行。我们在一行里念同一组的几个数字时，要加上 end=""，让它们在同一行挨着站好。

💻 Python 终极法术 (代码详解)

def dfs(start, path):
    # 🎯 如果我们手里的树叶组合长度刚好等于目标 r
    if len(path) == r:
        # 按顺序大声念出组合里的每一片树叶
        for num in path:
            # 📏 {num:3d}：保证这个数字恰好占据 3 格宽度！
            # end=""：念完这个数字不要换行，接着念下一个！
            print(f"{num:3d}", end="")
        
        print() # 这组念完啦，输出一个空的换行符，开启下一组
        return  # 结束这层的探险，回去换别的树叶
        
    # 🚶 从 start 号树叶开始，一直看到 n 号树叶
    # 注意 Python 的 range 不包含结尾，所以要写 n + 1
    for i in range(start, n + 1):
        path.append(i)      # 把树叶收进兜里！
        
        dfs(i + 1, path)    # 继续深入！下一次只能从 i+1 开始挑
        
        path.pop()          # 回溯！把刚放进去的树叶拿出来，换下一片

# 🗺️ 读取魔法指令
n, r = map(int, input().split())

# 探险开始！从第 1 片树叶开始看，初始兜里是空的 []
dfs(1, [])

🌟 Python 重点复习小贴士

  • 💡 为什么不能直接 print(path)？ 因为直接打印列表会长这样：。这不仅有逗号和中括号，而且宽度完全不符合题目要求。必须老老实实写个 for 循环配合格式化输出。
  • 💡 偷懒的进阶魔法： 其实如果你不想手写 DFS，Python 有一个神器可以直接生成组合：itertools.combinations(range(1, n+1), r)。有兴趣的小探险家可以去查查它是怎么配合格式化一起使用的哦！不过自己手写 DFS 是练好内功的基础！
