📜 任务清单 (题目描述)

小老鼠从坐标 (0, 0) 出发。地图上有 n 块奶酪，每块奶酪都有自己的坐标。老鼠要把它们全部吃掉。

路线要求：

  • 奶酪最多有 15 块（1 ≤ n ≤ 15）。
  • 你需要找出一条把所有奶酪吃光且总路程最短的路线。
  • 两点之间的距离就是平面直线距离：√((x1−x2)² + (y1−y2)²)。
  • 最后要告诉它这条路的总长度，四舍五入保留 2 位小数。
📥 魔法输入阵：
(4 块奶酪，正好在正方形的四个角)
4
1 1
1 -1
-1 1
-1 -1
📤 奇迹输出阵：
(最短路线长度，保留 2 位小数)
7.41

🧠 魔法道具箱 (核心思路：状态压缩 DP)

如果用普通的排列组合去猜所有的顺序，15 块奶酪会有 15!（大约 1.3 万亿）种走法，老鼠还没走完电脑就先冒烟了！这时候我们需要一种叫 状态压缩动态规划 (Bitmask DP) 的超级魔法。

✨ 把状态压缩进二进制数字里

我们怎么记录“哪些奶酪已经被吃掉了”呢？用一个二进制数字！

比如有 4 块奶酪（编号 0、1、2、3），如果 0 号和 2 号被吃了，1 号和 3 号没吃，我们可以写成二进制 0101。在十进制里，这个数字就是 5。

👇 状态表示魔法：二进制 0101 (十进制 5) 👇
🧀
3号(未吃)
0
🧀
2号(已吃)
1
🧀
1号(未吃)
0
🧀
0号(已吃)
1
新状态 = 原状态 | (1 << 准备吃的奶酪编号)

✨ 状态转移：老鼠的思考过程

我们准备一本魔法大字典：dp[mask][i]，表示“在已经吃掉的奶酪集合是 mask，并且老鼠最后停在第 i 块奶酪时，走过的最短距离”。

  1. 预知距离： 提前算好起点 (0,0) 到每块奶酪的距离，以及任意两块奶酪之间的距离。
  2. 第一步初始化： 从起点直奔第 i 块奶酪，状态就变成了 1 << i，距离就是起点到它的距离。
  3. 继续探索： 对于当前的任何一个状态，如果老鼠站在第 i 块奶酪上，它环顾四周，找到一个还没吃过的奶酪 j，跑过去！新状态就变成了 mask | (1 << j)，距离增加了 dist[i][j]。打个擂台保留最小值！

📘 C++ 魔法秘籍 (知识点)

  • 🧮 极大的数字 1e20： 求最小值时，一开始要把所有的距离都设置为“正无穷大”。在 C++ 里我们可以用 1e20 来表示一个极其巨大的实数。
  • 🎯 位运算魔法：
    1 << i：把 1 向左推 i 位，表示第 i 块奶酪。
    mask & (1 << i)：用来检查 mask 这个状态里，第 i 块奶酪是不是被吃过了。
    mask | (1 << j)：把第 j 块奶酪吃掉，加入到 mask 这个状态里！
  • 📏 保留小数法术： 包含 <iomanip> 后，用 cout << fixed << setprecision(2) 就可以强制让输出的小数保留 2 位。

💻 C++ 终极法术 (代码详解)

#include <iostream>
#include <iomanip>
#include <vector>
#include <cmath>
#include <algorithm>
using namespace std;

const double INF = 1e20; // 定义正无穷大

int main() {
    int n;
    cin >> n;
    vector<double> x(n), y(n);
    for (int i = 0; i < n; ++i) {
        cin >> x[i] >> y[i];
    }

    // 📏 计算起点 (0,0) 到每块奶酪的距离 d0
    vector<double> d0(n);
    for (int i = 0; i < n; ++i) {
        d0[i] = sqrt(x[i]*x[i] + y[i]*y[i]);
    }

    // 📏 计算任意两块奶酪点之间的距离 dist
    vector<vector<double>> dist(n, vector<double>(n));
    for (int i = 0; i < n; ++i) {
        for (int j = 0; j < n; ++j) {
            double dx = x[i] - x[j];
            double dy = y[i] - y[j];
            dist[i][j] = sqrt(dx*dx + dy*dy);
        }
    }

    int total = 1 << n; // 总共有 2^n 种状态
    // dp 数组：大小为 total 行，每行 n 列，初始全是 INF
    vector<vector<double>> dp(total, vector<double>(n, INF));

    // 🏁 初始化：老鼠从起点直接冲向某一块奶酪 i
    for (int i = 0; i < n; ++i) {
        dp[1 << i][i] = d0[i];
    }

    // 🏃 开始状态转移 (遍历所有可能的状态)
    for (int mask = 1; mask < total; ++mask) {
        for (int i = 0; i < n; ++i) {
            // 检查：当前状态 mask 里，老鼠必须得吃了第 i 块奶酪才合理
            if (!(mask & (1 << i))) continue; 
            
            if (dp[mask][i] >= INF) continue; // 如果这个状态还没走到过，跳过
            
            // 老鼠站在奶酪 i 上，环顾四周，寻找下一块没吃过的奶酪 j
            for (int j = 0; j < n; ++j) {
                if (mask & (1 << j)) continue; // j 已经被吃过了，跳过
                
                int nextMask = mask | (1 << j); // 加上 j 之后的新状态！
                double newDist = dp[mask][i] + dist[i][j]; // 计算新距离
                
                // 打擂台保留最短距离
                if (newDist < dp[nextMask][j]) {
                    dp[nextMask][j] = newDist;
                }
            }
        }
    }

    // 🏆 最终答案：所有奶酪都被吃光了 (也就是二进制全是1，对应数值是 2^n - 1)
    double ans = INF;
    int full = (1 << n) - 1;
    for (int i = 0; i < n; ++i) {
        // 看看最后停在哪块奶酪上距离最短
        ans = min(ans, dp[full][i]); 
    }

    // 📢 保留 2 位小数输出
    cout << fixed << setprecision(2) << ans << endl;
    return 0;
}

🐍 Python 魔法秘籍 (知识点)

  • 📐 直线距离神器 math.hypot(x, y)： Python 最擅长算数学啦！要算平面上直角三角形的斜边（也就是两点间直线距离），直接调用 math.hypot(x差值, y差值)，比自己写平方再开根号要省事得多！
  • 📝 f-string 格式化输出： 使用 print(f"{ans:.2f}") 就能把浮点数 ans 完美地四舍五入并保留两位小数输出。
  • 🛠️ 位运算小技巧： 判断第 i 位是不是 1，除了用 mask & (1 << i)，还可以写成 (mask >> i) & 1。把数字往右推 i 位，看最末尾是不是 1，这样更加直观易懂！

💻 Python 终极法术 (代码详解)

import math

def main():
    n = int(input())
    points = []
    for _ in range(n):
        x, y = map(float, input().split())
        points.append((x, y))

    # 📏 计算起点 (0,0) 到各块奶酪的距离
    # 使用 math.hypot 瞬间算出直线距离
    d0 = [math.hypot(x, y) for x, y in points]

    # 📏 计算任意两块奶酪之间的距离
    dist = [[0.0] * n for _ in range(n)]
    for i in range(n):
        for j in range(n):
            dx = points[i][0] - points[j][0]
            dy = points[i][1] - points[j][1]
            dist[i][j] = math.hypot(dx, dy)

    INF = 1e20 # 正无穷大
    total = 1 << n # 2 的 n 次方个状态
    
    # dp[mask][i] 的二维数组，初始化为无穷大
    dp = [[INF] * n for _ in range(total)]

    # 🏁 初始化：老鼠从起点直接冲到奶酪 i，此时状态就是 1<
    for i in range(n):
        dp[1 << i][i] = d0[i]

    # 🏃 DP 状态转移核心部分
    for mask in range(total):
        for i in range(n):
            # 检查当前 mask 状态里，老鼠是不是真的吃了奶酪 i
            if not (mask >> i) & 1:
                continue
            # 如果这步还没走通过，跳过
            if dp[mask][i] >= INF:
                continue
                
            # 老鼠在奶酪 i 上，寻找下一块没吃的奶酪 j
            for j in range(n):
                # 如果 j 已经被吃过了，跳过
                if (mask >> j) & 1:
                    continue
                    
                # 用 | 运算符把 j 加入已吃状态
                next_mask = mask | (1 << j) 
                new_dist = dp[mask][i] + dist[i][j]
                
                # 打擂台，只要距离更短就更新
                if new_dist < dp[next_mask][j]:
                    dp[next_mask][j] = new_dist

    # 🏆 最终状态：所有的 1 都填满了！也就是 (1<
    full = (1 << n) - 1
    # 从最后停在任何一点的结果里挑出最小的一个
    ans = min(dp[full])

    # 📢 格式化保留两位小数输出
    print(f"{ans:.2f}")

if __name__ == "__main__":
    main()

🌟 Python 重点复习小贴士

  • 💡 一维找最小值的快捷法术： 在 C++ 里我们需要写一个 for 循环找最后停在各个点上的最小值，但在 Python 中，因为 dp[full] 本身就是一个包含了到达所有终点距离的列表，我们直接用 min(dp[full]) 就能瞬间拿到全局最短距离，太方便了！
