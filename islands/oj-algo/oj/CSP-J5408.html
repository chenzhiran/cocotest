📜 任务清单 (题目描述)

在雪山之巅的雪峰议会，有 n 位守护者（编号从 1 到 n）。今天是大选日，雪民们投下了 m 张会发光的“雪票”。

你的终极任务： 把这茫茫多（最多 2,000,000 张！）的雪票，按照上面的编号 从小到大 排好队。数字整齐闪耀，冰门就会“咔哒”一声打开哦！🗝️

(5个候选人，10张选票)
5 10
(乱糟糟的10张雪票)

📤 奇迹输出阵 (输出样例)：
(从小到大铺好的雪序之路！)
1 2 2 2 2 2 2 2 5 5

🧠 魔法道具箱 (核心思路：桶排序/计数排序)

因为雪票的数量超级多（200万张！），如果用普通的排队魔法（比如冒泡排序），电脑会累趴下的！🥵

但是我们发现了一个秘密：守护者的编号很小（最大才999）！ 所以我们可以使用超级魔法——“魔法分类桶 (计数排序)”！🪣

✨ 施法步骤：

  1. 准备空桶： 守护者有 n 个，我们就准备 n 个魔法桶，每个桶上写着守护者的编号（从 1 到 n）。一开始里面都是空的（0）。
  2. 扔进雪票： 拿起一张雪票，看到上面写着几号，就往几号桶里扔进一颗发光星星 🌟。
  3. 依次倒出： 从 1 号桶走到 n 号桶，桶里有几颗星星，我们就把这个编号喊几次！排队完成！

📘 C++ 魔法秘籍 (知识点)

  • 🎒 超级魔法口袋 vector： 以前我们用普通的数组 a[1000]，现在我们可以用 vector<int> cnt(n+1, 0)，这就像是一个能自动帮你把里面全都变成 0 的超级口袋！
  • ⚡ 光速念咒 ios::sync_with_stdio(false);： 当要读入 200万 个数字时，普通的 cin 就像乌龟爬。加上这两句神秘咒语，你的 cin 就会踩上风火轮，瞬间读完所有数据！

💻 C++ 终极法术 (代码详解)

#include <iostream>
#include <vector>
using namespace std;

int main() {
    // ⚡ 念动加速咒语，让读入速度飞起来！(必背哦)
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n, m;
    cin >> n >> m; // 🖥️ 电脑问：有几个候选人？几张雪票？

    // 🎒 准备 n+1 个带编号的魔法桶，一开始里面都是0！
    // 为什么是 n+1 呢？因为编号是从 1 到 n，我们需要第 n 号桶！
    vector<int> cnt(n + 1, 0);

    // 🌪️ 开启循环，把 m 张雪票一张张读进来
    for (int i = 0; i < m; ++i) {
        int a;
        cin >> a;     // 读到一张雪票上面的编号 a
        ++cnt[a];     // 往 a 号桶里扔进一颗星星 (计数+1)
    }

    // 🚶‍♂️ 准备倒出桶里的星星啦！
    bool first = true; // 这是一个小旗子，标记是不是第一个输出的数字

    for (int i = 1; i <= n; ++i) {     // 从 1号桶 走到 n号桶
        for (int j = 0; j < cnt[i]; ++j) { // 桶里有几颗星，就输出几次
            if (!first) {
                cout << ' '; // 如果不是第一个数字，就在前面加个空格
            }
            cout << i;       // 喊出这个数字！
            first = false;   // 小旗子倒下，后面的数字都不是第一个啦
        }
    }
    cout << '\n'; // ✨ 华丽地换个行，冰门开启！

    return 0; // 任务圆满完成！
}

🌟 C++ 重点复习小贴士

  • 💡 为什么不用 sort？ 因为 `m` 实在太大了，普通排序会超时。当数字的范围很小（比如这里的 n 最大才 999），计数排序（桶排序） 永远是你的神兵利器！
  • 💡 注意空格控制： 题目要求数字之间有空格，但最后一个数字后面最好不要有多余的空格，所以我们用了一个小布尔旗帜 `first` 来完美解决！

🐍 Python 魔法秘籍 (知识点)

  • 🌪️ 超级吸尘器 sys.stdin.buffer.read()： 面对多达 200万 个数字，如果用普通的 `input()` 一行行读会很慢。这个咒语能一口气把所有的字都吸到程序里！
  • 🎒 瞬间造桶 [0] * (n + 1)： 在 Python 里，想造出 1000 个里面装着 0 的小桶，只要用乘法咒语 `*` 就可以一秒生成！
  • 🔗 神奇胶水 ' '.join()： 把一堆文字用空格像串糖葫芦一样串起来，一起输出，速度飞快！

💻 Python 终极法术 (代码详解)

import sys # 召唤高级工具箱！

def main():
    # 🌪️ 开启超级吸尘器，一口气吸入所有的数字，并按空格或回车切成小块！
    data = sys.stdin.buffer.read().split()
    if not data:
        return

    # 📦 前面两个包裹分别是 n(候选人数) 和 m(选票数)
    n = int(data[0])
    m = int(data[1])

    # 🎫 剩下的全部都是雪票上的编号啦，把它们都变成整数装进列表
    nums = list(map(int, data[2 : 2+m]))

    # 🎒 准备好 n+1 个魔法桶，里面一开始都是 0 颗星星！
    # 下标从 1 开始对应守护者编号，所以长度要 n+1 哦
    cnt = [0] * (n + 1)

    # 🏃‍♂️ 跑过每一张雪票，扔进对应的桶里
    for a in nums:
        cnt[a] += 1 # 编号为 a 的桶里，星星数量 +1

    # 📝 准备一张白纸，用来写最后排好队的数字
    out = []

    # 🚶‍♂️ 从 1 号桶依次走到 n 号桶
    for i in range(1, n + 1):
        if cnt[i]: # 如果桶里有星星 (数量不为0)
            # 桶里有几个，就把这个数字变成字符串，复制几次贴到白纸上！
            out.extend([str(i)] * cnt[i])

    # 🪄 用空格作为胶水，把白纸上的数字串联起来，一次性大声喊出来！
    sys.stdout.write(' '.join(out))

if __name__ == "__main__":
    main() # 🚀 启动魔法！

🌟 Python 重点复习小贴士

  • 💡 大象装冰箱分几步？ 读入速度是这题的生死线！切记使用 `sys.stdin.buffer.read().split()`。
  • 💡 为什么不用 `out.append` 而是 `out.extend` 呀？ `append` 是一次塞一个东西进去，`extend` 可以把一大堆东西（一个列表）直接倒进主列表里。因为我们要把同个编号重复好多次，`[str(i)] * cnt[i]` 生成了一个小列表，用 `extend` 倒进去效率最高！
