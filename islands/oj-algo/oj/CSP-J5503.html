📜 任务清单 (题目描述)

你需要修补一堵长为 N、宽为 2 的“冰墙”（即 2 × N 的网格）。工匠给了你两种可以任意旋转的魔法砖：

  • 直砖： 占 2 个小格的“长条冰”。
  • L 砖： 占 3 个小格的“拐弯冰”。

拼装要求： 砖块不能重叠，也不能留空。你需要计算出一共有多少种拼满整面墙的方法。由于拼法可能极其庞大，大门护卫只需要看答案的 最后 4 位数字（不足 4 位不用补零）。

📥 魔法输入阵：
(冰墙的长度 N = 13)
13
📤 奇迹输出阵：
(一共有 13465 种，最后 4 位是 3465)
3465

🧠 魔法道具箱 (核心图解：状态与滚动变量)

这道题的核心是找出铺砖规律的递推公式！首先，我们来认识一下手中的魔法砖块：

👇 我们拥有的两种魔法冰砖 👇
直砖 (竖放)
直砖 (横放)
L 砖 (可旋转)

✨ 超好用的黄金递推公式

通过仔细推导直砖和L砖的拼合方式，数学向导给出了一个完美的公式。如果设铺满长度为 n 的墙有 f(n) 种方法，那么对于 n ≥ 3，它满足：

$$f(n) = 2 \cdot f(n-1) + f(n-3)$$

✨ O(1) 空间的“滚动”秘术

要算 f(n)，我们根本不需要把前面的所有结果都存下来，我们只需要它前面的 3 个兄弟：f(n-3), f(n-2), 和 f(n-1)。就像接力跑一样，每次算出新结果，旧的变量就往前“滚动”移交接力棒！

🏃 滚动变量大挪移 (假设算出了新的一项 cur) 🏃
旧 a
丢弃
→
新 a (原b)
b
新 b (原c)
c
新 c (原cur)
cur

📘 C++ 魔法秘籍 (知识点)

  • 🛡️ 随时取模防溢出： 因为我们只需要最后四位数字，所以在每做一次加法后，都要立刻执行 % MOD。这样数字永远不会超过 10000，连普通的 int 都装得下，非常轻巧！
  • 🔄 三个变量的“左移”： 核心的滚动逻辑 a = b; b = c; c = cur; 就是把状态往旧的方向推，给下一次循环的计算腾出空间。这样我们的空间复杂度完美降到了 O(1)。

💻 C++ 终极法术 (代码详解)

#include <iostream>
using namespace std;

const int MOD = 10000; // 只需要保留最后四位

int main() {
    int N;
    cin >> N;

    // 🛑 应对边界魔法：长度太短的时候，直接给出答案
    if (N == 0) {
        cout << 1 << endl;
        return 0;
    }
    if (N == 1) {
        cout << 1 << endl;
        return 0;
    }
    if (N == 2) {
        cout << 2 << endl;
        return 0;
    }

    // ✨ 初始化滚动接力棒
    int a = 1; // 代表 f(0) 或者在循环里的 f(n-3)
    int b = 1; // 代表 f(1) 或者在循环里的 f(n-2)
    int c = 2; // 代表 f(2) 或者在循环里的 f(n-1)
    int cur;   // 用于存放当前计算出的新 f(n)
    
    // 🏃 开始飞速铺砖，直接迭代到目标 N
    for (int i = 3; i <= N; ++i) {
        // 核心公式：f(n) = 2 * f(n-1) + f(n-3)，并随时取模保留末4位
        cur = (2 * c + a) % MOD;
        
        // 🔄 滚动变量大挪移 (把状态传给下一个人)
        a = b;
        b = c;
        c = cur;
    }
    
    // 🏆 循环结束时，c 就变成了最终我们要的 f(N)
    cout << c << endl;
    return 0;
}

🐍 Python 魔法秘籍 (知识点)

  • ✨ 多变量同步赋值术： Python 最擅长的就是一行代码交换或更新多个变量。写成 a, b, c = b, c, cur 可以瞬间完成状态的“滚动”，代码看起来既清爽又极具魔法感！
  • 🎯 不需要手动补零： 题目明确要求“如果答案本身不到 4 位，就直接输出它本身”，所以我们取模 % 10000 后直接 print(c) 即可，完全不需要格式化填充 0。

💻 Python 终极法术 (代码详解)

MOD = 10000  # 我们的截断魔法器：只留最后 4 位

def main():
    N = int(input())
    
    # 🛑 处理超短冰墙的特殊情况
    if N == 0:
        print(1)
        return
    if N == 1:
        print(1)
        return
    if N == 2:
        print(2)
        return

    # ✨ 初始化三只“小精灵”：分别对应 f(n-3), f(n-2), f(n-1)
    a, b, c = 1, 1, 2  
    
    # 🏃 核心循环，从长度 3 一直推到 N
    for _ in range(3, N + 1):
        # 套用公式算新项，并切掉多余的高位
        cur = (2 * c + a) % MOD
        
        # 🔄 Python 特有的同步赋值，一秒完成接力棒滚动！
        a, b, c = b, c, cur
        
    # 🏆 输出最终成果
    print(c)

if __name__ == "__main__":
    main()

💡 技巧和重点总结 (划重点啦)

  • 🌟 滚动数组的空间奇迹： 当一个递推公式只依赖于前几项（例如前 3 项）时，我们就没有必要开一个长度达 N 的超级大数组 dp 了。用几个变量来回倒腾，不仅节省内存，运行速度还会变快！
  • 🌟 特判防错底线： 在进入 for i = 3 的循环前，一定要用 if 拦截掉 N=0, 1, 2 的情况，否则你的逻辑还没开始转动就会发生错误。
  • 🌟 题意阅读满分： “输出最后 4 位，若不足则直接输出”——这说明不要画蛇添足用类似 %04d 去给不够位数的数字补零，直接输出数字变量本身就完全符合要求了！
