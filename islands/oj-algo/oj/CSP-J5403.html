📜 任务清单 (题目描述)

地上散落着 n 颗发光石。法阵的规则很严格：每次你必须从中恰好选出 k 颗（不分先后顺序）。

你的目标： 把选出的 k 颗石头的能量相加，如果总和是一个素数，就会点亮一颗星光。请算出共有多少种不同的选法能点亮星光！

💡 素数小知识： 素数就是大于 1 且只能被 1 和它自己整除的数（比如 2, 3, 5, 7, 11...）。注意，1 不是素数哦！

📥 魔法输入阵 (输入样例)：
(4颗石头，挑3颗)
4 3
(4颗石头的能量值)
3 7 12 19
📤 奇迹输出阵 (输出样例)：
(能点亮法阵的选法数量)
1

🧠 魔法道具箱 (核心思路：枚举所有组合 + 素数判定)

这是一个经典的“组合与检验”问题。我们分为两步来施展魔法：

  1. 生成所有组合： 从 n 颗石头里挑 k 颗，由于不计顺序，我们要把所有可能的组合都找出来。
  2. 检验素数： 对每一种组合进行求和，然后用“素数检验法术”看看和是不是素数。是的话，计数器就加 1！

📘 C++ 魔法秘籍 (知识点)

  • 🌲 深度优先搜索 (DFS)： 要实现“从 n 个里选 k 个”，最强大的法术是递归（DFS）。为了不重复挑选，我们每次都从上一次挑选的石头的下一个开始挑（利用 start 变量）。
  • ⚡ 素数判定加速： 判定一个数 num 是不是素数，如果用一个循环从 2 一直跑到 num-1 会太慢。我们只需要跑到 sqrt(num)（开平方根）就足够啦！这能极大地提升魔法效率！

💻 C++ 终极法术 (代码详解)

#include <iostream>
#include <vector>
#include <cmath> // 召唤数学工具箱，用来求平方根 sqrt
using namespace std;

// 🛡️ 检验素数的法术函数
bool isPrime(int num) {
    if (num < 2) return false; // 1 和负数都不是素数
    if (num == 2) return true; // 2 是唯一的偶素数
    if (num % 2 == 0) return false; // 其他偶数直接淘汰！
    
    // 跑到平方根就足够了，大大提升速度
    int limit = sqrt(num); 
    for (int i = 3; i <= limit; i += 2) { // 每次加2，只检查奇数
        if (num % i == 0) return false;
    }
    return true;
}

int n, k;
vector<int> nums;
int ans = 0; // 记录点亮的星光数量

// 🌲 DFS 魔法分支寻找所有组合
// start: 这一步能从哪个石头开始选
// count: 你的篮子里已经挑了几颗石头
// sum: 篮子里所有石头的能量总和
void dfs(int start, int count, int sum) {
    // 🎯 如果已经挑满了 k 颗石头，停下来检查能量总和
    if (count == k) {
        if (isPrime(sum)) ans++; // 如果是素数，星光加一！
        return; // 篮子满了，不能再挑了，退回上一步
    }
    
    // 从 start 开始，尝试把后面的石头放入篮子
    for (int i = start; i < n; ++i) {
        // 魔法递归：进入下一层！
        // 注意：下一个只能从 i+1 开始选，防止重复选同一颗，也防止组合重复！
        dfs(i + 1, count + 1, sum + nums[i]);
    }
}

int main() {
    cin >> n >> k;
    nums.resize(n);
    for (int i = 0; i < n; ++i) {
        cin >> nums[i];
    }
    
    // 从第 0 颗石头开始看，篮子里有 0 颗，总能量是 0
    dfs(0, 0, 0); 
    
    cout << ans << endl; // 宣布胜利！
    return 0;
}

🌟 C++ 重点复习小贴士

  • 💡 DFS 组合树的剪枝： for (int i = start; i < n; ++i) 这是组合枚举的灵魂！通过限制每一次只能从上一个选中的元素之后开始选，我们天然避免了把 (3,7) 和 (7,3) 当成两种选法，不仅省去了去重的麻烦，还大幅加快了速度。
  • 💡 边界保护： 能量的最大值可能比较大，但是由于 k < 20 且单个值最大 5,000,000，相加后大约是一亿（1e8），在 int 的承受范围内，所以本题使用普通的 int 是完全安全的。

🐍 Python 魔法秘籍 (知识点)

  • 🧰 超级魔法盒 itertools.combinations： Python 最令人羡慕的就是它的标准库。面对组合问题，我们不需要手写复杂的递归树，只需念出 itertools.combinations(nums, k)，它就会瞬间把所有 n 选 k 的组合全部生成好端到你面前！
  • ⚡ 素数函数 math.sqrt()： 同样为了提高速度，素数判断里只需要循环到开平方根的整数部分。记得把结果转成整数 int(math.sqrt(num))，才能放进 range 里当边界！

💻 Python 终极法术 (代码详解)

import math
import itertools # 召唤高级迭代工具箱，准备使用组合魔法！

# 🛡️ 检验素数的法术函数
def is_prime(num):
    if num < 2:
        return False
    if num == 2:
        return True
    if num % 2 == 0: # 偶数直接淘汰
        return False
        
    # 计算上限：只需检查到开平方根的地方
    limit = int(math.sqrt(num))
    
    # 从 3 开始，步长为 2 (只检查 3, 5, 7...)
    for i in range(3, limit + 1, 2):
        if num % i == 0:
            return False
    return True

def main():
    # 🗺️ 读取法阵要求：共有 n 颗，必须选 k 颗
    n, k = map(int, input().split())
    
    # 读取所有发光石的能量值
    nums = list(map(int, input().split()))
    
    count = 0 # 星光计数器
    
    # 🪄 见证魔法！itertools.combinations 直接生成所有的组合
    # comb 会是一个元组，比如 (3, 7, 19)
    for comb in itertools.combinations(nums, k):
        # 把元组里的数字加起来，并判断是不是素数
        if is_prime(sum(comb)):
            count += 1 # 是素数，星光加一！
            
    # 📢 输出最终点亮的星光数量
    print(count)

if __name__ == "__main__":
    main()

🌟 Python 重点复习小贴士

  • 💡 为什么不用 itertools.permutations？ permutations 是排列，会把选出来的石头区分顺序（比如 (3,7) 和 (7,3) 算两次），而题目里只关心挑了哪些石头，不看顺序，所以必须用 combinations（组合）！
  • 💡 Python 高级技巧（选学）： 如果组合非常多，导致经常重复计算同一个和的素数检验，可以在 is_prime 函数上方加上一句 @functools.lru_cache(None)。这是一个魔法备忘录，算过一次的数字就会自动记住结果，速度能快到飞起！不过在这道题的数据量下，不加也能轻松通关哦。
