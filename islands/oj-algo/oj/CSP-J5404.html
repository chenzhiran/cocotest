火星人正在用手指排队形来表示超大数字。你能帮他们算出前进 M 步后的新队形吗？
🚀 点击这里，开启魔法传送门（前往做题）！

📜 任务清单 (题目描述)

火星人把从 1 到 N 的数字排成一个序列。把所有可能的序列按照从小到大的“字典序”排好排名。

你的终极任务： 科学家给了一个很小的数字 M。你要把火星人现在的队伍当成一个“排名编号”，然后让这个编号前进 M 步，最后输出前进后的新队伍顺序！

📥 魔法输入阵 (输入样例)：
(5根手指，前进3步)
5
3
(当前队形)
1 2 3 4 5
📤 奇迹输出阵 (输出样例)：
(前进3步后的新队形)
1 2 4 5 3

🧠 魔法道具箱 (核心思路：字典序的下一个排列)

如果我们要老老实实把所有排列排出来，N 最大到 10000，计算机算到明天也算不完！好在 M 非常小（最大才 100），我们只需要连续做 M 次“寻找下一个排列”的法术就可以了。

✨ “下一个排列”的三步法术：

  1. 寻找转折点 (轴)： 从队伍的最右边往左找，找到第一对左边比右边小的数字 a[i] < a[i+1]。这个 a[i] 就是我们需要替换的“轴”。
  2. 寻找替换者并交换： 再次从队伍的最右边往左找，找到第一个比刚刚的轴 a[i] 大的数字 a[j]，把这两个人互换位置。
  3. 尾部倒序： 因为刚才换了一个稍大的数字到前面，为了让这支队伍尽可能“小”（即刚好是下一个排名），我们需要把轴后面的所有数字全部反转（首尾对调）。
👇 魔法演示：以 1 2 3 5 4 为例，寻找它的下一个队形 👇
当前：
1
2
3
5
4
轴是 3 (因为 3 < 5)。右边第一个比 3 大的是 4，交换它们！
⬇️
交换后：
1
2
4
5
3
现在轴的后面是 5 和 3，把它们倒过来 (反转) 变成 3 和 5！
⬇️
最终结果：
1
2
4
3
5
就这样成功前进了一步！连续做 M 次就行啦！🎉

📘 C++ 魔法秘籍 (知识点)

  • 🪄 现成的魔法书 next_permutation： C++ 最让人喜爱的地方就是它自带了这套三步法术！只要召唤了 #include <algorithm>，就可以直接使用 next_permutation(开始位置, 结束位置)，它会自动把数组变成下一个队伍！
  • 🔄 M 次循环： 使用一个简单的 for (int i = 0; i < M; ++i)，让上面这句咒语连续触发 M 次。
  • 🧹 完美的空格控制： 输出要求数字之间只有 1 个空格。我们可以用一个绝妙的判断：if (i > 0) cout << " "; 然后输出数字，这样行末就不会有多余空格啦！

💻 C++ 终极法术 (代码详解)

#include <iostream>
#include <algorithm> // 召唤自带魔法 next_permutation！
#include <vector>
using namespace std;

int main() {
    int N, M;
    cin >> N >> M;
    
    // 🗺️ 读取当前的火星人队伍
    vector<int> fingers(N);
    for (int i = 0; i < N; ++i) {
        cin >> fingers[i];
    }
    
    // 🔄 连续施展 M 次下一个排列的法术
    for (int i = 0; i < M; ++i) {
        next_permutation(fingers.begin(), fingers.end());
    }
    
    // 📢 输出结果，注意空格控制
    for (int i = 0; i < N; ++i) {
        if (i > 0) cout << " "; // 不是第一个数字就在前面加空格
        cout << fingers[i];
    }
    cout << endl;
    
    return 0;
}

🌟 C++ 重点复习小贴士

  • 💡 时间复杂度极低： 虽然 N 有 10000 那么大，但是 next_permutation 在最坏情况下的单次操作也仅仅是跑一趟数组而已。加上 M 最大只有 100，所以总共最多运行一百万次级别，计算机 0.01 秒就能算完！

🐍 Python 魔法秘籍 (知识点)

  • 🛠️ 亲手打造轮子： Python 虽然没有直接修改原数组的 next_permutation，但我们可以自己把那“三步法术”写出来。这就叫做算法的魅力！
  • 🔀 极简交换法 a, b = b, a： 在 Python 里想要交换两个变量，不需要借助第三个临时杯子，直接用逗号隔开赋值，就能瞬间完成乾坤大挪移。
  • 🔗 输出胶水 ' '.join()： 面对一个全是数字的列表，先用 map(str, fingers) 把它们全都变成字符串类型，然后用空格胶水粘起来，输出格式直接满分！

💻 Python 终极法术 (代码详解)

def next_permutation(arr):
    # 🪄 魔法第一步：从右往左，寻找第一个下降的点 (轴)
    i = len(arr) - 2
    while i >= 0 and arr[i] >= arr[i + 1]:
        i -= 1
        
    # 如果 i 跑到最前面都没找到，说明这已经是最大的降序排列了
    if i < 0:
        return False
        
    # 🪄 魔法第二步：再从右往左，寻找第一个比轴大的数字
    j = len(arr) - 1
    while arr[j] <= arr[i]:
        j -= 1
        
    # 交换它们！Python 特色极简交换
    arr[i], arr[j] = arr[j], arr[i]
    
    # 🪄 魔法第三步：把轴后面的所有数字倒反天罡 (反转)
    left, right = i + 1, len(arr) - 1
    while left < right:
        arr[left], arr[right] = arr[right], arr[left]
        left += 1
        right -= 1
        
    return True

def main():
    # 🗺️ 读取 N 和 M
    N = int(input())
    M = int(input())
    
    # 读入队伍并变成整数列表
    fingers = list(map(int, input().split()))
    
    # 🔄 用下划线表示不用关心的循环变量，连续施法 M 次
    for _ in range(M):
        next_permutation(fingers)
        
    # 📢 用空格胶水把整个队伍粘合输出
    print(' '.join(map(str, fingers)))

if __name__ == "__main__":
    main()

🌟 Python 重点复习小贴士

  • 💡 为什么不用自带的排列法？ Python 的 itertools.permutations 是生成所有排列，没办法原地修改现在的数组，如果我们用它查排名再往后数，内存会直接爆炸。所以学会这套原地的三步法术是非常有价值的！
  • 💡 双指针反转： 在最后反转区间的步骤中，我们使用了一个在左 left，一个在右 right 的双指针，一边向中间走一边交换，这就是最经典的列表原地反转法。
