📜 任务清单 (题目描述)

在金色沙漠里，竖立着 n 根代表候选人的“星冕石柱”。每根石柱上都刻着一串超长超长的发光数字，那是他们的票数。

你的任务： 找出票数最大的那位候选人，并大声宣布他的 编号 (从1开始数) 和他的 票数！

📤 奇迹输出阵：
(第4个人的票数最长，是他赢了！)

🧠 魔法道具箱 (核心思路：超长数字的“字符串”比较法)

⚠️ 警报！ 这里的票数可能长达 100 位！普通的 int 甚至 long long 魔法口袋都会被撑爆！所以我们必须用 字符串 (文字) 的形式来存储和比较它们。

比较两个用文字写的超级大数字，我们有 两条黄金法则：

法则 1：比长度！
98765
VS
1022356
哪怕前面是9，但右边有 7 位数，左边只有 5 位数。
长度越长，数字绝对越大！
法则 2：长度一样，从左往右比！
98765
VS
87954
都是 5 位数，我们就比第一位。
'9' 比 '8' 大，所以左边赢！这就是字典序比较！

📘 C++ 魔法秘籍 (知识点)

  • 📦 无限长口袋 string： 只要带上 #include <string>，无论多长的数字，哪怕 1000 位，它都能像装字母一样轻松装下！
  • 📏 自带卷尺 .size()： 想要知道数字有多长？直接用 a.size() 就能量出字符串的长度啦。
  • 🔤 字典序对决 a > b： 在 C++ 中，如果用 > 来比较两个字符串，它会自动像查字典一样，从左到右一个字符一个字符地比对哦！

💻 C++ 终极法术 (代码详解)

#include <iostream>
#include <string>    // 📦 召唤无限长口袋 string
#include <algorithm>
using namespace std;

int main() {
    // ⚡ 加速咒语，让读入飞起来
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n; // 🖥️ 电脑问：有多少个候选人呀？

    string best;      // 👑 用来记录目前看到的最大票数
    int bestIdx = 1;  // 👑 用来记录最大票数是谁的编号（从1开始）

    // 🥇 先把第 1 个人的票数记作“目前的冠军”
    cin >> best;      

    // 🏃‍♂️ 从第 2 个人开始，一个个过来挑战冠军！
    for (int i = 2; i <= n; ++i) {
        string cur;
        cin >> cur;   // 读入挑战者的票数
        
        bool greater = false; // 记录挑战者有没有赢

        // ⚖️ 开始黄金法则比对！
        if (cur.size() != best.size()) {
            // 法则 1：长度不一样，比谁更长！
            greater = cur.size() > best.size();
        } else {
            // 法则 2：长度一样，直接用 > 进行字典序比较！
            greater = cur > best; 
        }
        
        // 如果挑战者赢了，交出星冕！
        if (greater) {
            best = cur;   // 冠军票数更新
            bestIdx = i;  // 冠军编号更新
        }
    }

    // 🎉 比赛结束，大声宣布星冕得主的编号和票数！
    cout << bestIdx << '\n' << best << '\n';
    
    return 0;
}

🐍 Python 魔法秘籍 (知识点)

  • 📦 天生神力： 其实 Python 里的整数 (int) 是可以无限大的！但为了不弄丢数字前面的“前导零”（虽然这题说没前导零，但养成好习惯很重要），我们还是用 字符串 (文字) 来保存票数。
  • 📏 量尺魔法 len()： 只要念出 len(文本)，就能立刻知道这段文字有几个字符啦。
  • 📝 去掉尾巴 strip()： 用吸尘器吸进来的文字，最后可能会带一个隐形的“换行符”。用 strip() 能把它洗得干干净净。

💻 Python 终极法术 (代码详解)

import sys # 召唤高级工具箱！

def main():
    # 🌪️ 用超级吸尘器一口气吸入所有文字，并按行切开！
    data = sys.stdin.read().splitlines()
    if not data:
        return

    # 第一行是候选人的数量 n，记得洗干净并变成整数
    n = int(data[0].strip())
    
    # 🗳️ 把下面 n 行的票数都洗干净，收进 votes 列表里
    votes = [line.strip() for line in data[1:1+n]]

    # 👑 先让第 1 个人（下标为 0）当“目前的冠军”
    best = votes[0]
    best_idx = 1  # 编号是从 1 开始的哦！

    # 🏃‍♂️ 从第 2 个人（下标 1）开始，挨个挑战冠军！
    for i in range(1, n):
        cur = votes[i] # 挑战者的票数
        
        # ⚖️ 开始黄金法则比对！
        if len(cur) != len(best):
            # 法则 1：长度不一样，比谁更长！
            if len(cur) > len(best):
                best = cur
                best_idx = i + 1 # 因为 i 是从0开始的下标，编号要 +1
        else:
            # 法则 2：长度一样，直接用 > 比较文字 (字典序)
            if cur > best:
                best = cur
                best_idx = i + 1
                
    # 🎉 比赛结束，大声宣布星冕得主的编号和票数！
    # f"{...}" 是一种超级格式化魔法，能把变量直接填进字符串里哦
    sys.stdout.write(f"{best_idx}\n{best}\n")

if __name__ == "__main__":
    main() # 🚀 启动魔法！

🌟 重点复习小贴士

  • 💡 千万不能偷懒直接 > 哦！ 如果你比较两个字符串 "9" 和 "100"，电脑按从左到右比，会觉得 '9' 大于 '1'，从而判定 "9" 更大！这就闹笑话啦！所以必须先比长度！
  • 💡 下标和编号的区别： 在数组或列表里，大家都是从 0 开始排队的。但题目要的编号是从 1 开始的。所以 Python 代码里记录冠军时，记得要写 i + 1。
