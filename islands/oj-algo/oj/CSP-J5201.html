🏓 乒乓球记分大师

跟华华一起学习如何记录精彩的比赛吧！

🌟 题目描述

华华是个乒乓球迷！他记录了自己比赛中每一球的胜负情况。W 代表华华赢了一球，L 代表对手赢了一球。所有的记录都连在一起，直到看到 E 才结束。

我们要帮华华按照11分制计算每一局的比分。

🧠 解题思路分析

想象你就是赛场边的裁判，手里拿着记分牌：

1. 收集信息 📥
把所有读到的 'W' 和 'L' 存进小本本里，直到遇到 'E'。
2. 开始记分 ✍️
一个一个看记录：遇到 W 华华加1分，遇到 L 对手加1分。
3. 谁赢了？ 🏆
满足两个条件就结束一局：
1. 有人到了 11分
2. 两人分差 ≥ 2分

💡 温馨提示： 如果比赛结束时（遇到 E），当前这一局还没打完，也要把最后的比分记下来哦！

🔍 样例详细拆解

🌈 样例 1：热身赛

输入： WWWWWWWWWWWWWWWWWWWWWWLWE

  • 第一局： 华华连赢11球，比分 11:0。达成条件（≥11且差≥2），清零！
  • 第二局： 华华又连赢11球，比分 11:0。达成条件，清零！
  • 第三局： 剩下的记录是 W 和 L，遇到 E 结束，比分 1:1。

🌈 样例 2：跨行记录

输入：
WWWWWWWWWWW
LWELWLL...

  • 第一行有11个W，直接输出 11:0。
  • 第二行在 E 之前有一个 L 和一个 W，记录为 1:1。E 之后的记录我们就不看啦！

🌈 样例 3：激烈的拉锯战

分析： 当双方打到 10:10 平手时，必须要拉开 2 分差距。比如 12:10 或 13:11 才会结束这一局。样例 3 中出现了许多 11:0 和 0:11，最后的 0:2 是因为还没打完就遇到了 E。

💻 魔法代码实现

点击下方按钮，切换你喜欢的编程语言：

import sys

def main():
    # 1. 像吸尘器一样读入所有数据
    records = []
    for line in sys.stdin:
        for ch in line.strip():
            if ch == 'E': # 看到E就停止
                break
            if ch == 'W' or ch == 'L':
                records.append(ch)

    hua_score = 0
    opp_score = 0

    # 2. 模拟比赛过程
    for ch in records:
        if ch == 'W':
            hua_score += 1
        else:
            opp_score += 1

        # 3. 检查这局结束了吗？(11分制且分差≥2)
        if (hua_score >= 11 or opp_score >= 11) and abs(hua_score - opp_score) >= 2:
            print(f"{hua_score}:{opp_score}")
            hua_score, opp_score = 0, 0

    # 4. 最后没打完的也要报幕哦！
    if hua_score != 0 or opp_score != 0:
        print(f"{hua_score}:{opp_score}")

if __name__ == "__main__":
    main()
📝 Python 代码小抄：
  • sys.stdin：帮我们读取多行输入。
  • abs(a - b)：计算两个数差了多少，不分正负。
  • records.append(ch)：把这一球的结果放进列表存起来。
#include <bits/stdc++.h>
using namespace std;

// 准备一个大口袋装所有的记录
char s[100000]; 

int main() {
    char temp;
    int idx = 0;
    
    // 1. 读入数据直到遇到 'E'
    while (cin >> temp && temp != 'E') {
        if (temp == 'W' || temp == 'L') {
            s[idx++] = temp;
        }
    }

    int w = 0, l = 0;
    
    // 2. 遍历口袋里的记录
    for (int i = 0; i < idx; i++) {
        if (s[i] == 'W') w++;
        else l++;
        
        // 3. 判断是否满足获胜条件
        if ((w >= 11 || l >= 11) && abs(w - l) >= 2) {
            cout << w << ":" << l << endl;
            w = 0; l = 0; // 记得清零哦！
        }
    }
    
    // 4. 处理最后一局
    if (w != 0 || l != 0) {
        cout << w << ":" << l << endl;
    }
    
    return 0;
}
📝 C++ 代码小抄：
  • char s[100000]：开辟一个足够大的空间存放字符。
  • abs(w - l)：同样是计算绝对值（差值）。
  • cin >> temp：跳过空格和换行，只抓取有用的字符。
