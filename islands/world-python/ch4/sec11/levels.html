LEVEL 20

分裂箭 (Splitting Arrows)

战场清道夫

这是一场大规模战争！我们将击败 77 个食人魔。
如果我们把尸体都留在地上，电脑内存会“爆炸”（变卡）。
所以，我们使用 defeat 事件作为触发器，立即销毁尸体。

常规情况 (Normal)

敌人死亡 -> 播放死亡动画 -> 留下尸体 -> 占用内存。

清道夫模式 (Cleanup)

onDefeat 触发 -> 计数+1 -> 立即调用 unit.destroy()
结果：敌人瞬间消失，不留痕迹，游戏运行如丝般顺滑。

Clean Memory
