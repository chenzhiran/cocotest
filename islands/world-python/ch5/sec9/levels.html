强壮的沙牦牛 (Strong Yak)

战术思路分析：相对躲避

沙漠里的牦牛太强壮了，不能硬碰硬。当它靠近时，我们必须**相对于当前位置**向右移动 10 米来躲避。

关键：获取并修改 X 坐标

1. 获取当前 X：`x = hero.pos.x`
2. 计算新 X：`x = x + 10` (或简写为 `x += 10`)
3. 移动到新坐标：`hero.moveXY(x, y)` (Y 保持不变)

Current X + 10

Move to safety!

读取当前 Y 坐标
5

6

7
# 如果到牦牛的距离小于10：
8
if hero.distanceTo(yak) < 10:
如果牦牛太近
9
# 向右移动，在英雄位置的x坐标值基础上加10。

11
hero.moveXY(x, y)
移动到新的 X 坐标

绿洲 (The Oasis)

战术思路分析：动态平移

英雄默认要向右边的绿洲前进 (`x + 10`)。但如果牦牛太近，必须反向移动 (`x - 10`) 来躲避。

关键：If/Else 切换方向

- `if 牦牛靠近`: `x -= 10` (向左)

**注意：** 移动动作 `hero.moveXY(x, y)` 放在 `if` 和 `else` 的代码块内执行。

十字路口 (The Crossroads)

战术思路分析：四向定位

食人魔从东南西北四个方向进攻。英雄必须通过比较自身和敌人的坐标来确定它们的来向，并在正确的 X 标记处放置陷阱。

关键：判断方向

  • 左边来：`enemy.pos.x < hero.pos.x`
  • 右边来：`enemy.pos.x > hero.pos.x`
  • 下方来：`enemy.pos.y < hero.pos.y`
  • 上方来：`enemy.pos.y > hero.pos.y`
**注意：** 使用 `elif`（else if）确保只有满足第一个匹配的方向时才放置陷阱。

IF X ELIF X>hero.X: Trap Right

第 53 关

截断 (Interception)

战术思路分析：中点保护

你是保护农民的守卫，必须站在农民和敌人之间最完美的中点位置。这需要使用**求中点公式**。

关键公式：中点坐标

X 坐标：`X = (A.x + B.x) / 2`
Y 坐标：`Y = (A.y + B.y) / 2`

X = (Friend.X + Enemy.X) / 2
Y = (Friend.Y + Enemy.Y) / 2

Python 程序解法

1
while True:
2
enemy = hero.findNearestEnemy()
找到最近的敌人
3
friend = hero.findNearestFriend()
找到最近的农民
4
# 通过将friend.pos.x和enemy.pos.x相加来计算x
5
x = (friend.pos.x + enemy.pos.x) / 2
计算 X 轴中点
6
# 现在对y做同样的事情
7
y = (friend.pos.y + enemy.pos.y) / 2
计算 Y 轴中点
8
# 移动到计算得到的X和Y坐标。
9
hero.moveXY(x, y)
移动到中点位置

战术思路分析：上下拦截

牦牛随机从上方或下方来袭。你需要比较 `yak.pos.y` 和 `hero.pos.y`，来决定在牦牛的上方还是下方建造围栏，引导它们走向水坑。

Y 轴比较逻辑

- `if yak.pos.y > hero.pos.y`: 牦牛在**上方** → 围栏建在 `yak.pos.y - 10` (牦牛的**下方**)
- `else`: 牦牛在**下方** → 围栏建在 `yak.pos.y + 10` (牦牛的**上方**)

Yak.Y > Hero.Y ?

Build Y = Yak.Y - 10

第 55 关

概念挑战. 友好的雷区 (Friendly Mines)

战术思路分析：Y轴避险

食人魔被困在上下方的雷区中，你可以利用它们的位置来引导自己安全通过。英雄必须一直向右走，但需要根据敌人的垂直位置上下微调。

相对 Y 轴移动

- `if enemy.pos.y > hero.pos.y`: 敌人在**上面** → 英雄向上微调 (`y + 4`)
- `else`: 敌人在**下面** → 英雄向下微调 (`y - 4`)
**关键：** X 轴的移动 (`x + 12`) 必须在每次循环的最后执行。

IF Enemy.Y > Hero.Y: Y += 4
ELSE: Y -= 4
Move X += 12

Python 程序解法

1
while True:
2
enemy = hero.findNearestEnemy()
寻找最近的敌人
3
if enemy:
如果敌人存在
4
if enemy.pos.y > hero.pos.y:
判断：敌人是否在英雄**上方**
5
hero.moveXY(hero.pos.x, hero.pos.y + 4)
向上微调 4 米
6
else:
否则 (敌人在下方或同高)
7
hero.moveXY(hero.pos.x, hero.pos.y - 4)
向下微调 4 米
8
hero.moveXY(hero.pos.x + 12, hero.pos.y)
向右前进 12 米 (每次循环都会执行)
