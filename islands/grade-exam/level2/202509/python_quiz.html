第 4 题
想取出正整数 N 从右往左数的第 M 位数字（例如 1234 的第 2 位是 3），横线处填什么？
N = int(input("请输入一个正整数:"))
M = int(input("请输入从右到左取第几位数:"))
div = 10 ** (M - 1)
digit = _____________
print(digit)
  • A. N % div // 10
  • B. N // div // 10
  • C. N % div % 10
  • D. N // div % 10
查看答案
✅ 正确答案：D

🎓 魔法解析：
假设 N=1234, M=2。我们要取 3。
1. div = 10^(2-1) = 10。
2. N // div 就是 1234 // 10 = 123 (把右边不需要的切掉)。
3. 然后对 10 取余数 123 % 10 = 3 (拿到最后一位)。
所以公式是：先整除，再取余。选 D！

第 6 题
有一种编号规则 "XX-Y"。

XX 从 00 变到 11，超过 11 就回到 00。
如果输入 N，如何计算 part1 (XX) 和 part2 (Y)？ 
N = int(input())
# 这里的 N 是总序号，从0开始
part1 = (N // 10) % 12  # 计算XX
part2 = N % 10          # 计算Y
# 题目选项问的是填空代码

(注：原题考察取余和整除的逻辑)

第 8 题
嵌套循环大考验！cnt 最后是多少？
cnt = 0
for i in range(-10, 10):
    for j in range(i):
        cnt += 1
print(cnt)
  • A. 145
  • B. 125
  • C. 55
  • D. 45
查看答案
✅ 正确答案：D

🎓 魔法解析：
1. 外层循环 i 从 -10 到 9。
2. 内层循环 range(i)：如果 i 是负数或 0，range 是空的，不执行！
3. 所以只有 i = 1, 2, ..., 9 的时候，内层循环才会跑。
4. 当 i=1 跑1次，i=2 跑2次... 直到 i=9 跑9次。
5. 总数 = 1 + 2 + ... + 9 = 45。

第 9 题
这个复杂的循环会输出什么？
for i in range(1, 12):
    if i % 2 == 0:
        continue
    for j in range(i):
        if i * j % 2 == 1: # 假设题目意图是乘积为奇数
            break
    else:
        print(i, j)
  • A. 9 9
  • B. 1 1 (或类似少量输出)
  • C. 0
  • D. 没有输出
查看答案
✅ 正确答案：B (参考答案)

🎓 魔法解析：
这道题代码逻辑比较绕。
1. 外层循环跳过所有偶数，只看奇数 i (1, 3, 5...)。
2. 内层循环 j 从 0 开始。
3. 如果 i * j 是奇数，就 break。因为 i 是奇数，只要 j 变成奇数 (也就是 j=1)，乘积就是奇数，循环被打断。
4. 如果循环被打断，else 就不会执行。
5. 只有当循环完整跑完没被打断，才会执行 else。
6. 唯一可能的情况是 range(i) 只有 0 的时候（即 i=1）。当 i=1，j=0，乘积是0（偶数），不break，循环结束，执行 else，打印。
所以只有 i=1 时会有输出。

第 10 题
关于求最大公约数的辗转相除法代码，哪个说法正确？
a = int(input())
b = int(input())
while b != 0:
    remainder = a % b
    a = b
    b = remainder
print(a)
  • A. b不能为0，否则报错
  • B. a必须小于b
  • C. a和b必须是正整数，否则%报错
  • D. 如果初始b为0，输出就是原始的a
查看答案
✅ 正确答案：D

🎓 魔法解析：
这是一个经典的数学魔法：欧几里得算法。
如果一开始 b 就是 0，while b != 0 根本不会执行，直接打印 a。这是完全正确的逻辑（任何数和0的最大公约数是它自己）。
A错，while条件就是保护。B错，a比b大也没关系，第一轮会自动交换。

编程题 1：优美的数字

题目描述：
如果一个正整数的所有数位都相同，它就是"优美的数字"。
例如：6 (只有6), 99 (都是9)。但 123 就不优美。
请你算出，从 1 到 n 之间，有多少个优美的数字？

输入： 一个整数 n
输出： 优美数字的数量

查看参考代码与解析

🎓 思路小黑板：
1. 我们可以遍历 1 到 n 的每一个数字。
2. 对于每个数字，检查它的每一位是不是都相等。
3. 比如 num % 10 拿到个位，然后不断 // 10 检查其他位。
4. 如果全部通过，计数器 + 1。

# 优美数的数量
n = int(input()) # 不大于n的优美数
ans = 0 # 保存优美数的数量
for i in range(1, n + 1):
    num = i % 10 # 个位数，优美数肯定与个位数全相同
    Flag = True # 默认是优美数
    while i > 0:
        if num != i % 10:
            Flag = False # 一旦遇到与个位数不等，不是优美数
            break # 一旦遇到与个位数不等，终止循环
        i //= 10
    if Flag == True: # 如果Flag没有被改变为False，则为优美数
        ans += 1
print(ans)
编程题 2：绘制菱形

题目描述：
输入一个奇数 n，画出一个 n 行 n 列的空心菱形。
用 # 画边，用 . 填充背景。
例如 n=5：

..#..
.#.#.
#...#
.#.#.
..#..
查看参考代码与解析

🎓 思路小黑板：
这是一个找规律的题目。
中心点坐标是 k = (n-1) // 2。
对于任意一个点 (i, j)，它到中心的"曼哈顿距离"如果等于 k，那就是边界！
公式：abs(i - k) + abs(j - k) == k

# 菱形字符图，n必须为奇数，n是层数
n = int(input())
k = (n - 1) // 2 # 中间#的位置
# 外层i控制层数，内层j控制字符位置
for i in range(n):
    for j in range(n):
        if j + i == k: # 菱形左上边
            print('#', end='')
        elif j - i == k: # 菱形右上边
            print('#', end='')
        elif i - j == k: # 菱形左下边
            print('#', end='')
        elif i + j == 3 * k: # 菱形右下边
            print('#', end='')
        else:
            print('.', end='')
    print()
