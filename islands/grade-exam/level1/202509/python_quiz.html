第 4 题
数学魔法时间！执行下面的代码，输出会是什么？

a = 13
b = 5
print(a / b, a // b, a % b)
  • A. 2.6 2 3
  • B. 2.6 2 3.0
  • C. 2 2 3
  • D. 2 3 4
✨ 查看答案 ✨
✅ 正确答案：A

🎓 魔法解析：
我们要分清三种除法武器：
1. / 是普通除法：13 ÷ 5 = 2.6。
2. // 是地板除（整除）：13 里面有几个完整的 5？答案是 2。
3. % 是取余数：分完后还剩下多少？13 - 10 = 3。
所以结果是 2.6, 2, 3。

第 5 题
算术优先级大挑战！表达式 3 * 3 ** 3 // 3 的值是多少？
  • A. 243
  • B. 81
  • C. 27
  • D. 9
✨ 查看答案 ✨
✅ 正确答案：C

🎓 魔法解析：
Python 算术也是要排队的，幂运算（**，就是几次方） 优先级最高！
1. 先算 3 ** 3：3 × 3 × 3 = 27。
2. 算式变成了 3 * 27 // 3。
3. 乘法和整除同级，从左往右算：3 * 27 = 81。
4. 最后 81 // 3 = 27。
答案就是 27 啦！

编程题 1：商店折扣

题目描述：
商店有两种优惠方案：
1. 满减：购物满 x 元，减 y 元。
2. 打折：直接打 n 折（例如 n=8 就是打8折，价格乘以 0.8）。
小明买了 p 元的东西，请帮他算算，用哪种方案最省钱？输出最少需要支付的金额（保留2位小数）。

输入变量： x, y, n, p

✨ 查看参考代码与解析 ✨

🎓 思路小黑板：
我们需要分别算出两个价格，然后比一比谁更小！
1. 算方案一：如果总价 p 超过 x，价格变成 p - y；否则还是 p。
2. 算方案二：价格变成 p * n / 10。
3. 比较：用 if 判断哪个小，或者直接输出较小的那个。


# 参考代码
x = int(input("满多少(x): "))
y = int(input("减多少(y): "))
n = int(input("打几折(n): "))
p = int(input("总价(p): "))

# 方案1：满减
price1 = p
if p >= x:
    price1 = p - y

# 方案2：打折
price2 = p * n / 10

# 比较并输出较小的，保留2位小数
if price1 < price2:
    print(f"{price1:.2f}")
else:
    print(f"{price2:.2f}")
编程题 2：金字塔

题目描述：
金字塔有 n 层。从下往上数：
第 1 层有 n * n 块石头，
第 2 层有 (n-1) * (n-1) 块石头，
...
最顶层有 1 * 1 块石头。
请问搭建这个金字塔总共需要多少块石头？

✨ 查看参考代码与解析 ✨

🎓 思路小黑板：
这其实是一个数学规律题！
我们需要计算：1^2 + 2^2 + 3^2 + ... + n^2 的和。
我们可以用一个 for 循环，从 1 数到 n，每次把当前层数的平方加到存钱罐（变量）里！


n = int(input("请输入层数: "))
total = 0  # 这是一个存石头的篮子

for i in range(1, n + 1):
    # i * i 就是这一层的石头数量
    total = total + (i * i)

print(total)

比如 n=3，就是 1 * 1 + 2 * 2 + 3 * 3 = 1 + 4 + 9 = 14 块石头。
