2. ⚡ 核心资源：法力值 (Mana)

法力值是战场上唯一的“货币”，用于建造新塔和升级已有塔。

获取方式

  • 每秒自动恢复： 稳定的基本收入。
  • 击败怪物奖励： 击杀怪物获得额外法力，怪物越强，奖励越多。
⚠️ 法力上限： 最多持有 100 点。溢出部分将作废，接近上限时请务必立即使用！
# 查看当前法力值 myMana = hero.mana
// 查看当前法力值 let myMana = hero.mana;
// 查看当前法力值 int myMana = hero.mana;

3. 🏗️ 领地与建造规则

阵营领地

  • 🟥 红色格子： 红队专属建造区。
  • 🟦 蓝色格子： 蓝队专属建造区。
  • 🟪 紫色格子： 公共区域，双方均可建造。

升级与替换 (Upgrade & Replace)

  • 升级： 在已有塔位建造同类型塔 → 塔等级提升，威力更强。
  • 替换： 在已有塔位建造不同类型塔 → 替换为新塔，且保留之前的等级（法力不浪费）。
🔄 蓝队镜像提示： 系统会自动翻转蓝队的 X 坐标。无论你在哪一队，只需按照 红队视角 编写一套代码即可。
# 判断是否可建造 hero.canBuildAt(x, y) # 方式一：用点位名 ("A"~"I") hero.build("archer", "A") # 方式二：用坐标 (x, y) hero.build("mage", 15, 40)
// 判断是否可建造 hero.canBuildAt(x, y); // 方式一：用点位名 ("A"~"I") hero.build("archer", "A"); // 方式二：用坐标 (x, y) hero.build("mage", 15, 40);

4. 🏰 防御塔总览 (Towers)

通过组合不同的防御塔，构筑完美的钢铁火力网。所有防御塔的建造和升级费用均为 25 Mana。

中文名称 类型代码 (极重要) 特点描述
🏹 弓箭塔 "archer" 射速最快，基石火力
🔮 法师塔 "mage" 天空克星，射程最远
💥 加农炮 "cannon" 范围爆炸，不能对空
❄️ 寒霜塔 "frost" 范围减速，辅助核心

属性参数对比

  • 🏹 弓箭塔： 冷却 0.5s (最快)，伤害 13，范围 24。可对空。
  • 🔮 法师塔： 冷却 1.0s，范围 30 (最远)。对飞行怪 4 倍伤害 (18)。可对空。
  • 💥 加农炮： 冷却 1.5s，伤害 13，爆炸半径 10。不可对空。
  • ❄️ 寒霜塔： 冷却 2.0s，减速系数 0.5 (速度减半)，减速持续 2s。可对空。
📈 伤害成长公式： ROUND(BASE_DAMAGE * (1 + LOG(1.55, LEVEL)))

5. 🌀 特殊技能 (Special Abilities)

技能完全不消耗法力值，是独立冷却的免费王牌。释放通用施法时间均为 0.4s (英雄会被锁定)。

中文名称 类型代码 效果说明
🌀 拉力 "pull" 向内吸聚。强度3000，持续3秒。与加农炮绝配。
🌬️ 推力 "push" 持续向外推开。强度-3000，持续3秒。
💣 爆炸 "burst" 推力的33倍瞬间爆发！保命神技，冷却极短。
🔮 传送 "teleport" 将怪物向对手水晶送回 60 格，并附带晕眩。

经典连招推荐

  1. 天罗地网： 使用 "pull" 吸聚 + 加农炮范围轰炸。
  2. 潮汐连环： 使用 "pull" 吸聚后立刻接 "push"，让怪物在火力网中来回。
  3. 乾坤挪移： 针对高血量精英怪，使用 "teleport" 直接让其重新走一遍。

6. 👾 怪物情报与对手信息

特殊怪物警惕

  • 飞行怪物： 免疫加农炮。法师塔对其有 4 倍伤害，是对空主力。
  • 远程怪物： 远距离拆家，必须优先处理！
