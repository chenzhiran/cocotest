攻击 Brack。
12

再次攻击。
13
hero.moveRight(2);
大跨步逃生！
14
return 0;
15
}

hero.say("Achoo");
大声说出密码！
5
hero.moveUp();
门开了。
6
hero.moveUp();
逃离。
7
return 0;
8
}

No.13

第13关 焰中舞动

关卡思路：无限循环

你需要永远在火焰中跳舞：左、右、左、右……
手动写代码太累了，我们需要循环！

While(true) 循环

while(true) { ... } 就像一个“永动机”。
放在花括号 {} 里面的代码，会一遍又一遍地执行，永远不会停止。

solution.cpp
1
int main() {
2
while(true) {
循环开始。
3
hero.moveRight();
花括号内：向右移动。
4
// 在这里给循环里加 moveLeft 命令。
提示：不要忘了反向。
5
hero.moveLeft();
花括号内：向左移动。
6
}
循环结束，回到第2行。
7
return 0;
8
}
No.14

第14关 闹鬼的迷宫

关卡思路

这个迷宫的路径是重复的：
右 -> 右 -> 上 -> 上 -> (重复)

我们只需要写一次这个组合，然后把它放进 while(true) 循环里。
英雄就会自动重复这个路径，直到走出迷宫！

solution.cpp
1
int main() {
2
while(true) {
开启无限循环。
3
hero.moveRight();
第1步：向右。
4
hero.moveRight();
第2步：向右。
5
hero.moveUp();
第3步：向上。
6
hero.moveUp();
第4步：向上。
7
}
循环结束，重复。
8
return 0;
9
}

1. 向右

把这一套动作放进循环，英雄就能不断重复这套动作穿过迷宫。

R
4

D
5

R(2)
6
hero.moveUp();
U
7
}
8
return 0;
9
}

int main() {

保持光环移动。
3
hero.moveUp();
向上驱散。
4
hero.moveDown();
向下驱散。
5
hero.moveRight(2);
趁机推进2格。
6
}
7
return 0;
8
}
