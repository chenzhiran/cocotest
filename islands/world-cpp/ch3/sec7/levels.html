Function Parameters: 传递数据的魔法

def func(target):
形参 (Parameter):
就像一个空盒子（比如叫 target）。
在函数内部，我们只操作这个盒子。
func(enemy)
实参 (Argument):
这是实际放进盒子里的东西。
你可以把 enemy1、enemy2 轮流放进去。

万能的 Target

我们定义一个函数，它的括号里有一个词：target。 这只是一个代号！

函数定义：
def check(target): attack(target)

调用 1:
check(topEnemy) -> target 变成了 topEnemy

调用 2:
check(bottomEnemy) -> target 变成了 bottomEnemy

参数中的逻辑

在函数内部，target 就是你的操作对象。 你可以对它做任何事：判断、攻击、或者忽略。
