逻辑判断：如果...就...

IF 语句 (条件判断)
if enemy:
如果敌人存在，就攻击。
如果敌人不存在，就跳过。
安全检查
防止英雄对着空气乱打！
先检查 findNearestEnemy() 是否找到了东西。

No.7-9

巡逻兵克星 & 野餐毁灭者 & 鹰眼

循环中的判断

现在我们要把 if 放到 while True 循环里。
这叫**嵌套 (Nesting)**。

英雄会一直重复做这几件事：
1. 找敌人。
2. 如果有 -> 打。
3. 如果没 -> 没事发生，继续找。

注意缩进：
if 要在 while 里缩进。
attack 要在 if 里**再**缩进一次！
solution.py
1
while True:
无限循环。
2
enemy = hero.findNearestEnemy()
在循环里持续侦察。
3
if enemy:
如果找到了...
4
hero.attack(enemy)
双重缩进：攻击！
如果没敌人，什么都不做，直接下一轮循环。
