变身巨人的逻辑

想一想《超级马里奥》，当他吃了蘑菇会发生什么？变大！变大后就能撞碎砖块。 这一关我们就要实现这个机制。

状态变化 (Collect)

吃到蘑菇 -> unit.scale = 2 (体型变大两倍)。
同时设置一个倒计时，时间到了变回原形。

碰撞检测 (Collide)

当英雄碰到东西时：
1. 对方是墙(Fence)吗？
2. 我现在是巨人(scale == 2)吗？
如果两个条件都满足，那就把墙撞碎(destroy)！

冻结标签 (Frozen Tag)

冰冻人游戏规则

这是一个经典的捉迷藏游戏！你是“鬼”，弓箭手是“人”。
我们用 behavior 属性来标记它们的状态。

奔跑状态 (Scampers)

弓箭手到处乱跑。如果这时被你(Player)碰到 -> 变成冰冻状态。
behavior = "Defends" (站着不动)

冰冻状态 (Defends)

弓箭手站着不动。如果这时被其他弓箭手(Archer)碰到 -> 解冻！
behavior = "Scampers" (又开始跑了)

Frozen Count: 0
